## 1️⃣ 1차 전지 vs 2차 전지

### 1차 전지 (Primary Battery)
- **일회용**, 방전되면 끝 (재충전 불가)
- 알카라인 건전지, 리튬 코인전지 등
- 구조 단순, 저렴

### 2차 전지 (Secondary Battery / Rechargeable Battery)
- **재충전 가능** — 수백~수천 회 사용
- 납축전지 (자동차 시동용) → Ni-Cd → Ni-MH → **Li-ion (현재 주류)**
- **"리튬이온 배터리(LIB)"** = 대표적 2차전지

### 왜 리튬인가?
- 원소주기율표상 **가장 가벼운 금속** (원자량 6.94)
- 전기화학 전위차 최대 → **에너지 밀도 ↑**
- 그래서 스마트폰·노트북·EV 전부 리튬이온

### 주요 2차전지 비교
| 종류 | 에너지 밀도 (Wh/kg) | 수명 (cycles) | 용도 |
|---|---|---|---|
| 납축 (Lead-Acid) | 30~50 | ~500 | 자동차 시동, UPS |
| Ni-Cd | 45~80 | ~1,000 | (환경 규제로 퇴출) |
| Ni-MH | 60~120 | ~500 | 하이브리드차 (구세대) |
| **Li-ion** | **150~300** | **500~3,000** | **EV·스마트폰·ESS (주류)** |
| 전고체 (Solid-State) | **400+** | **1,000+** | 미래 |

## 2️⃣ 전지의 구성 — 4대 소재 + 팩

리튬이온 배터리 **1 셀**은 다음 4개 소재로 구성됩니다:

```
[양극 (Cathode)]  ←──  [분리막 (Separator)]  ──→  [음극 (Anode)]
     │                        │                         │
   NCM/LFP                 분리막                    흑연/실리콘
     │                        │                         │
     └────────────  전해액 (Electrolyte)  ─────────────┘
                   LiPF6 + 유기용매
```

### 각 구성요소의 역할
| 요소 | 영어 | 역할 | 무게 비중 |
|---|---|---|---|
| **양극** | Cathode | 리튬이온을 **저장/방출** (전지 용량·전압 결정) | ~35% |
| **음극** | Anode | 충전 시 리튬이온을 **수용** | ~20% |
| **분리막** | Separator | 양극·음극 접촉 방지 (안전), 이온 통과 허용 | ~5% |
| **전해액** | Electrolyte | 리튬이온 **이동 통로** | ~15% |
| 기타 (동박·알루미늄박·케이스·탭) | - | 집전체·외장 | ~25% |

### 셀 → 모듈 → 팩
- **셀 (Cell)**: 최소 단위. 1개 파우치/캔
- **모듈 (Module)**: 셀 수십 개 묶음
- **팩 (Pack)**: 모듈 수 개 + BMS + 냉각 시스템
- EV 1대 = 팩 1개 = 모듈 10~20개 = 셀 수백~수천 개

## 3️⃣ 작동 원리 — 충·방전

### 🔋 방전 (Discharge, 전기 사용)
1. 음극에서 **리튬이온 (Li⁺)** 이 빠져나옴
2. 전해액을 통해 양극으로 이동
3. 동시에 **전자 (e⁻)** 가 외부 회로를 따라 이동 → **전기 흐름**
4. 음극이 비워지고 양극이 채워짐

### ⚡ 충전 (Charge, 전기 공급)
- 반대 과정: 양극에서 리튬이온이 음극으로 돌아감

### 📌 비유로 이해하기
- **양극 = 거대한 차고**, **음극 = 주차장**, **리튬이온 = 자동차**
- 충전: 차들이 차고 → 주차장으로 이동 (전기 에너지 저장)
- 방전: 차들이 주차장 → 차고로 복귀 (전기 에너지 방출)

### 왜 수명(cycles)이 정해져 있나?
- 충·방전 반복 시 **양극·음극 구조가 조금씩 손상**
- 리튬이 **전해액과 부반응** → 용량 감소
- 고온·고충전 시 **리튬 덴드라이트** 생성 → 단락 위험
- 일반 EV 배터리: 1,500~3,000 cycles (약 10~15년)

## 4️⃣ 주요 화학 체계 — **NCM vs LFP 대전**

양극재의 화학 조성이 배터리의 성격을 결정합니다.

### 🔋 NCM (Nickel-Cobalt-Manganese) — 삼원계
- **Ni·Co·Mn 3종 금속** 혼합
- 에너지 밀도 높음 (250~300 Wh/kg) → **장거리 EV**
- 코발트 비쌈·분쟁광물·니켈 화재 위험
- **한국 3사 주력** (LG엔솔·삼성SDI·SK온)

#### NCM 세부
- NCM 523 (Ni 50%, Co 20%, Mn 30%)
- NCM 622 (Ni 60%, Co 20%, Mn 20%)
- NCM 811 (Ni 80%, Co 10%, Mn 10%) — **하이니켈**
- NCM 9 ½ ½ (Ni 90%+) — 최신
- → Ni 비중 높을수록 에너지 밀도 ↑ but 안정성 ↓

### 🔋 NCA (Nickel-Cobalt-Aluminum)
- Al이 Mn 대체 → 고출력, 고밀도
- **Tesla Model S/X (Panasonic NCA)**, 삼성SDI 일부

### 🔋 LFP (Lithium Iron Phosphate) — **중국 주도 재부상**
- **Fe(철)·P(인) 기반** — 니켈·코발트 미사용
- 장점: **저렴, 안전 (열폭주 낮음), 수명 길다** (3,000~10,000 cycles)
- 단점: 에너지 밀도 낮음 (160~180 Wh/kg) → 같은 주행거리에 **더 무거움**
- **BYD Blade, Tesla Model 3 SR, CATL** 주력
- 2022년까지 NCM:LFP = 50:50 → **2024년 LFP 63~65%로 역전**

### 🔋 LMFP (Lithium Manganese Iron Phosphate)
- LFP에 망간 추가 → 에너지 밀도 210 Wh/kg까지 상승
- Gotion·Dynanonic 주도
- NCM-LFP 중간 영역

### 🔋 LCO (Lithium Cobalt Oxide)
- 최초의 상용 리튬이온 (소니 1991)
- 현재는 스마트폰·노트북용
- EV에는 사용 안 함 (코발트 100%)

### 📊 화학 체계 비교 (2026 기준)

| 항목 | NCM 811 | LFP | LMFP | 전고체 (목표) |
|---|---|---|---|---|
| 에너지 밀도 (Wh/kg) | 250~280 | 160~180 | 180~210 | 400+ |
| 수명 (cycles) | 1,500~2,500 | 3,000~10,000 | 2,000~4,000 | 2,000+ |
| 안전성 | 중 | **높음** | 중~높음 | 매우 높음 |
| 가격 ($/kWh, 2025) | ~$128 | **$81** | ~$95 | 실험실 |
| 주요 사용처 | 프리미엄 EV, 플래그십 | 보급형 EV, ESS | 중가 EV | 미래 |

## 5️⃣ 셀 형태 — 각형·원통형·파우치

### 🔲 각형 (Prismatic)
- 사각 금속 캔
- 공간 효율 좋음, 조립 쉬움
- **CATL·BYD·삼성SDI 주력**
- 현재 가장 많이 쓰이는 형태 (EV 70%+)

### ⭕ 원통형 (Cylindrical)
- 캔 원통 (AA 건전지 확대판)
- **규격**:
  - 18650 (직경 18mm × 높이 65mm) — 노트북, 초기 Tesla
  - 21700 — Tesla Model 3/Y (Panasonic)
  - **4680 (46mm × 80mm)** — Tesla 차세대, LG엔솔·삼성SDI 추격
- 대량 생산 유리, Tesla가 선도
- **LG엔솔·삼성SDI 4680 양산** 중

### 📦 파우치 (Pouch)
- 알루미늄 필름 백
- 가볍고 얇음, 자유 형태 가능
- 에너지 밀도 높음
- 단점: 외부 충격 약함, 부풀음(swelling)
- **LG엔솔·SK온 주력**

### 셀 형태별 점유율 (2024 글로벌)
| 형태 | 비중 | 대표 |
|---|---|---|
| 각형 | ~70% | CATL, BYD, 삼성SDI |
| 원통형 | ~20% | Tesla, Panasonic, LG엔솔 |
| 파우치 | ~10% | LG엔솔, SK온, 현대-기아 |

## 6️⃣ 핵심 성능 지표

### 에너지 밀도 (Energy Density)
- **단위 무게/부피당 저장 가능한 에너지**
- Gravimetric (중량 기준): **Wh/kg** — EV 주행거리 직결
- Volumetric (부피 기준): **Wh/L** — 공간 제약 중요 시

### 출력 밀도 (Power Density)
- **단위 시간당 방출할 수 있는 전력**
- 단위: W/kg
- EV 가속·급속 충전에 중요

### C-rate (충·방전 속도)
- **1C** = 용량을 1시간 내 완전 충·방전
- **2C** = 30분, **0.5C** = 2시간
- EV 급속충전 목표: **3C+** (20분 내 80%)

### 수명 (Cycle Life)
- **80% 용량까지 유지되는 충·방전 횟수**
- NCM: 1,500~2,500, LFP: 3,000~10,000

### 안전성 (Safety)
- **열폭주 (Thermal Runaway)** 저항성
- NCM 150°C → 폭주, LFP 270°C+ → 폭주

### 가격 ($/kWh)
- 팩 기준 가격. BNEF 조사.
- 2010: $1,500 → **2025: $108** → 2030E: $80

## 1️⃣1️⃣ 4680·전고체·Na-ion — 미래 기술

### 4680 원통형 셀
- Tesla 2020년 발표, 46mm × 80mm
- 기존 21700 대비 **에너지 5배, 전력 6배, 주행거리 16%↑**
- **Tab-less 구조** (용접 단순화, 제조비 절감)
- Tesla 직접 생산 + Panasonic·LG엔솔·삼성SDI 공급
- 4680 탭리스 = **전극 시트 전체가 전류 집전체 역할** → 저항·발열 감소

### 전고체 (Solid-State Battery)
- **액체 전해액 → 고체 전해질**
- **장점**:
  - 에너지 밀도 400+ Wh/kg (기존 2배)
  - 화재 위험 최소 (전해액 증발 無)
  - 수명 2,000+ cycles
- **단점**:
  - 양산 난이도 극high
  - 가격 (2026 현재 NCM 대비 5~10배)
- **타임라인**:
  - 삼성SDI: 수원 파일럿 2023, **2027 소량 → 2030 양산**
  - 도요타·이데미츠: 2027~2028 BEV 탑재 (1,000km)
  - QuantumScape (VW Powerco), Solid Power (BMW/현대)
- 3대 기술:
  - **황화물계** (가장 우수, 삼성SDI·도요타)
  - 산화물계 (QuantumScape)
  - 폴리머계 (Solid Power)

### 나트륨이온 (Na-ion)
- 리튬 대신 **나트륨 (소금)** 사용
- **장점**: 원료 풍부 (바닷물), 저렴, 저온 성능 우수
- **단점**: 에너지 밀도 낮음 (120~160 Wh/kg)
- **CATL Freevoy** (2023~), **BYD Seagull**
- **용도**: 저가 EV·ESS·엔트리급 스쿠터

### Mn-rich (LMR)
- Ni 50% + Mn 40% + Co 5%
- **GM-LG엔솔 2028년 양산** 발표
- 원가 $6,000/차 절감 목표 (vs 기존 NCM)

<a id="12"></a>
## 1️⃣2️⃣ EV vs ESS — 요구사항 차이

같은 리튬이온이지만 **사용처에 따라 요구 성능이 다릅니다**.

| 축 | EV | ESS (Energy Storage System) |
|---|---|---|
| 에너지 밀도 | **중요** (주행거리) | 덜 중요 |
| 출력 밀도 | 중요 (가속) | 덜 중요 |
| 수명 | 10~15년 | **20년+** |
| 가격 | 중요 | **매우 중요** |
| 크기·무게 | 제약 有 | 제약 거의 無 |
| 최적 화학 | **NCM·NCA** (하이니켈) | **LFP** (가격·수명) |
| 셀 형태 | 각형·원통·파우치 | 각형·원통 |

→ **ESS는 LFP가 압도적 우위**, EV는 프리미엄 NCM vs 보급형 LFP 양립.

### ESS 주요 시장
- **유틸리티 스케일** (수GW): ERCOT, CAISO 등
- **하이퍼스케일러 BESS**: AI DC 전력 안정화
- 가정용 (Tesla Powerwall, LG엔솔 RESU)

<a id="13"></a>
## 1️⃣3️⃣ 배터리 팩·BMS 개념

### 배터리 팩 구성
```
[셀 × 수백~수천]
   ↓
[모듈 × 10~20]
   ↓
[배터리 팩]
   ├─ BMS (Battery Management System)
   ├─ 냉각 시스템 (공랭·수랭·액침)
   ├─ 보호 회로
   └─ 케이스
```

### BMS (Battery Management System)
- 팩의 **두뇌** — 소프트웨어·하드웨어 결합
- 셀별 전압·온도 모니터링
- 충·방전 제어
- **SOC** (State of Charge) 표시
- **SOH** (State of Health) 추적
- 열폭주 예방
- 한국: 삼성SDI·LG엔솔 내재화, 파워로직스 (053690) 등

### 냉각 시스템
- 공랭: 단순, 저효율
- **수랭 (Liquid Cooling)**: EV 주류
- **액침 (Immersion Cooling)**: 차세대 (성능·안전성↑)
