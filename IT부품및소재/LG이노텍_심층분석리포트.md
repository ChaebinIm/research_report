### 2.3 카메라 모듈이란?

> 🔍 **용어 박스 #1 — 카메라 모듈 (Camera Module)**
> 스마트폰 후면에 박힌 동그란 검은 부품 = 카메라 모듈. 안에 **렌즈 + CIS 센서 + ASIC + 액추에이터 + 모듈 기판** 통합.
>
> 비유: **카메라 = 사람 눈**. 렌즈 = 각막 + 수정체, CIS 센서 = 망막, ASIC = 시신경.

### 2.4 카메라 모듈 구조 (단위 부품)

| 부품 | 역할 | 공급사 |
|---|---|---|
| **렌즈** | 빛 모음 | Largan(대만) + Sunny Optical(중국) |
| **CIS 센서** | 빛 → 디지털 변환 | **Sony 50% + 삼성전자 30%** |
| **ASIC** | 이미지 처리 | Apple 자체 |
| **VCM (액추에이터)** | 초점 조절 | TDK + Mitsumi(일본) |
| **OIS (손떨림 보정)** | 흔들림 보정 | Mitsumi + Alps(일본) |
| **모듈 기판** | 통합 | **LG이노텍·삼성전기·Sunny** |
| **조립·테스트** | 완성품 | **LG이노텍·삼성전기·Sunny** |

→ **LG이노텍의 핵심 역량 = "여러 부품을 정밀 조립해서 카메라 모듈로 완성"**.

## 3. 사업부 #2: 패키지솔루션 — FC-BGA 반도체 기판 (8%)

### 3.1 패키지솔루션이란?

**패키지솔루션** = 반도체 칩과 메인보드를 연결하는 **반도체 패키지 기판** 사업. LG이노텍의 차세대 성장 동력.

### 3.2 반도체 기판이란?

> 🔍 **용어 박스 #2 — 반도체 기판 (Substrate)**
> 반도체 칩(GPU·CPU)과 컴퓨터 메인보드 사이의 **중간 다리 역할** 부품. 칩 신호를 메인보드로 전달.
>
> 비유:
> - **메인보드** = 도로
> - **반도체 칩** = 자동차
> - **반도체 기판** = 자동차와 도로 사이의 타이어 + 서스펜션

### 3.3 FC-BGA란 무엇인가?

**FC-BGA (Flip Chip Ball Grid Array)** = 고성능 반도체 패키지 기판.

#### 일반 기판 vs FC-BGA
| 항목 | 일반 PCB | FC-BGA |
|---|---|---|
| 용도 | TV·가전 | **AI GPU·CPU·서버** |
| 층수 | 4~8층 | **20~30층** |
| 가격 | $5~10/장 | **$300~1,000/장** ⭐ |
| 마진 | 5% | **15~25%** |
| 신뢰성 | 일반 | 매우 높음 |

→ **AI 시대 핵심 부품**.

### 3.4 LG이노텍 FC-BGA 사업 진입 역사

#### 2021: 신사업 진입 결정
- LG이노텍이 FC-BGA 시장 진출 발표
- 2026년까지 매출 1조 목표

#### 2022-06: 구미 4공장 인수
- LG전자로부터 구미 4공장 인수
- FC-BGA 전용 라인 구축
- 약 8,000억 투자

#### 2024-말: 북미 빅테크 공급 시작
- NVIDIA·Intel·AMD에 FC-BGA 공급
- **글로벌 AI Capex 폭발 직접 수혜**

#### 2026: 매출 1.3조 목표
- 글로벌 빅테크 매출 가속
- **2026 영업이익 2,000억+ 가능**
