# 🛡️ 글로벌 방위산업 2025~2030 전망 + 한국 방산 빅4 심층 비교
**작성일: 2026-04-24 | Agent Team 취합본**

> **대상**: 2022년 러-우 전쟁 이후 시작된 글로벌 재무장 사이클의 구조적 장기 성장 국면. 2025년 NATO 헤이그 정상회의 **"GDP 5% 국방비"** 목표 합의로 2035년까지 수요 천장 상향. 한국 방산 빅4 (한화에어로스페이스·KAI·LIG넥스원·현대로템) 합산 매출 40.45조·영업이익 4.63조 사상 최대.

---

## 📌 Executive Summary

| 항목 | 현황 |
|---|---|
| 글로벌 국방비 (2024, SIPRI) | **$2.72조 (+9.4% YoY)** — 냉전 종식 후 최대 증가율 |
| 2030 글로벌 국방비 (추정) | **~$3.3조** |
| NATO 헤이그 선언 (2025.6) | **2035년까지 GDP 5%** (코어 3.5% + 인프라 1.5%) |
| 현재 3.5% 도달국 | 라트비아·리투아니아·폴란드 3개국 → 구조적 catch-up 수요 |
| 한국 방산 수출 2025 | **$15.4B (23.2조, +60.4% YoY)** — 2020~24 세계 10위 (M/S 2.2%) |
| 한국 빅4 FY25 매출 합산 | **40.45조** |
| 한국 빅4 FY25 영업이익 | **4.63조** |
| 한국 빅4 수주잔고 합산 | **100조+** |

**핵심 결론**: 방산 사이클은 구조적 장기 상승. **가격·납기·현지화 3박자**로 한국 빅4는 유럽·중동의 substitute tier 선점. 빅4 backlog 기반 실적은 일회성 아닌 **구조적 이익**.

**선호 순위**: 🥇 **현대로템** (폴란드 K2 2차 본격 인식, op leverage) → 🥈 **LIG넥스원** (중동 실전 레퍼런스) → 🥉 **한화에어로스페이스** (한화오션 통합 시너지) → KAI (KF-21 수출 가시화 전까지 laggard)

## Part III. 한국 방산의 구조적 경쟁력

| 요인 | 실증 |
|---|---|
| **가격** | K2 ~$9~10M vs Leopard 2A8 ~$15~20M. K9 ~$3~4M vs PzH2000 ~$17M |
| **납기** | 폴란드 K2 1차: 계약 2022.8 → 초도 인도 2022.12 (**4개월**). 독일 Leopard는 18~36개월 |
| **현지 생산·기술이전** | K2PL 폴란드 현지화, M-SAM 사우디 현지화 조건 → Rheinmetall·Lockheed 대비 유연 |
| **한미동맹 라이선스** | F-404/F-414 엔진, PAC-3, F-35 OEM → FMS 우회 없이 확장 |
| **지정학적 가용성** | 한국은 우크라이나 직접 무기 공급 불가 → 폴란드-독일-미국 경유 간접 공급 체인 구축 |

→ **2025 한국 방산수출 $15.4B (+60.4%)**, 2023~2024 침체 후 강한 반등. SIPRI 기준 2020~24 **세계 10위 무기수출국 (M/S 2.2%)**.

## Part IV. 리스크

1. **수주 사이클 변동성**: 메가딜 1건이 실적 좌우. 2025 폴란드 K2 2차 없었다면 빅4 합산 순익 -20%
2. **美 FMS·ITAR**: KF-21 엔진, PAC-3, Aegis 승인 필수. 트럼프 2기 동맹 재조정 변수
3. **지정학 반전**: "반사이익" 구조 → 휴전·종전 시 수요 둔화
4. **신흥국 외환·재정**: 폴란드 국방비의 37%가 차입. 이집트·이라크·필리핀 디폴트 리스크
5. **ESG·수출통제**: 중동 무기수출 EU·UN 반발, K2-Leopard 쟁송 재점화
6. **Hanwha-KAI 지배구조**: 2026.3 지분 4.99% 재취득 → 적대적 vs 전략적 제휴 해석 분열

## 📊 투자 시사점 — 선호 순위

### 🥇 **현대로템** (선호 1위)
- **Pros**: 폴란드 K2 2차 본격 매출 인식, 루마니아·이라크 파이프라인, **첫 1조 돌파 후 operating leverage 여력**, 철도 파생 성장
- **Cons**: 자동차 계열 지배구조, Abrams/Leopard 경합
- **Catalyst**: 루마니아 K2 수주 확정, 2026~27 폴란드 인도 가속

### 🥈 **LIG넥스원** (선호 2위)
- **Pros**: 천궁-II 중동 확장 + **UAE 실전 레퍼런스**, 수출 비중 21%, LAMD 주관사
- **Cons**: Block-III 기술 인증, Patriot/THAAD 경합
- **Catalyst**: 이라크 $2.6B 천궁-II, 추가 중동 국가 채택

### 🥉 **한화에어로스페이스** (선호 3위)
- **Pros**: 한화오션 통합 시너지 (land-air-sea 플랫폼), 미국 DIB 진출, 천무 폴란드 3차 5.6조
- **Cons**: 단기 EPS 변동성, 한화오션 조선 사이클
- **Catalyst**: 미 해군 MRO 확장, Boeing F-15 추가 수주

### **KAI** (선호 4위, Relative laggard)
- **Pros**: KF-21 120대 양산 확정, FA-50 수출 지속, 한화 지분 취득 M&A 모멘텀
- **Cons**: KF-21 수출 가시성 미흡, 공공 최대주주
- **Catalyst**: KF-21 인도네시아·UAE 수출 확정, 엔진 국산화 진척
