## 1. 밸류체인이란?

밸류체인(가치사슬)이란, **하나의 제품이 만들어져서 고객에게 전달되기까지의 모든 단계**를 말해요.

양자컴퓨터의 밸류체인을 간단히 그리면 이래요:

```
[소재/부품]  →  [극저온 냉각]  →  [제어 장비]  →  [양자 프로세서]  →  [오류 보정]  →  [소프트웨어]  →  [응용 서비스]
   |              |              |              |              |            |              |
 탄탈럼,       Bluefors,      Oxford        IBM,         Riverlane,    Qiskit,       금융,
 실리콘,      Oxford Inst.   Ionics,       Google,      Iceberg      PennyLane,    제약,
 조셉슨 접합   ULVAC         Monarch       IonQ                     Amazon Braket  물류
```

비유하자면, 자동차의 밸류체인이 "철강 → 엔진 → 타이어 → 조립 → 소프트웨어 → 딜러"인 것처럼, 양자컴퓨터도 여러 단계를 거쳐서 만들어져요. 각 단계마다 전문 회사들이 있어요.

## 2. 하드웨어 — 양자컴퓨터를 만드는 데 필요한 것들

### 2-1. 극저온 냉각 시스템 — 우주보다 차갑게!

#### 왜 필요할까?

초전도 큐비트(IBM, Google 방식)는 **영하 273.14도**에서 작동해요. 이건 우주의 온도(영하 270도)보다 더 차가워요! 이렇게 극단적으로 차갑게 만들어야 원자의 움직임이 거의 멈추고, 양자 상태가 안정적으로 유지돼요.

이 극저온을 만드는 장비가 **희석 냉동기(Dilution Refrigerator)**예요.

#### 희석 냉동기란?

쉽게 말하면, **극도로 차가운 냉장고**예요. 하지만 일반 냉장고와는 비교도 안 될 만큼 복잡하고 비싸요.

작동 원리를 간단히 설명하면: 헬륨-3와 헬륨-4라는 두 종류의 헬륨 가스를 섞었다가 분리하는 과정에서 열을 흡수하는 현상을 이용해요. 마치 소금이 얼음에 뿌려지면 온도가 더 내려가는 것과 비슷한 원리예요! 이 과정을 여러 단계로 반복하면서 15mK(영하 273.135도)까지 냉각해요.

장비 하나의 가격은 **수십억 원**이에요!

#### 냉각 장비를 만드는 회사들

| 회사 | 나라 | 시장 점유율 | 특징 |
|---|---|---|---|
| **Bluefors** | 핀란드 | 약 45% | **세계 1위!** IBM의 양자 시스템에 냉각기 공급. 1,000큐비트 이상 지원 가능 |
| **Oxford Instruments** | 영국 | 약 25% | 2위. 대형 모듈형 냉동기 설치 |
| **ULVAC** | 일본 | 성장 중 | IBM과 협력, 2026년 배치 목표의 차세대 냉동기 개발 |
| **Leiden Cryogenics** | 네덜란드 | - | 전통 강자, 연구기관 중심 |

### 2-2. 양자 프로세서 소재/부품 — 큐비트를 만드는 재료

양자 프로세서를 만드는 데 필요한 재료와 부품을 큐비트 종류별로 알아봐요.

#### 초전도 큐비트용 소재

초전도 큐비트는 특수한 금속과 기판으로 만들어요.

| 소재 | 역할 | 설명 |
|---|---|---|
| **알루미늄 (Al)** | 큐비트 회로 | 가장 기본적인 초전도 금속 |
| **니오븀 (Nb)** | 큐비트 회로 | 알루미늄보다 더 안정적 |
| **탄탈럼 (Ta)** | 큐비트 회로 | **최신 트렌드!** Princeton 대학이 사용, 코히런스 시간 3배 향상 |
| **실리콘 기판** | 큐비트를 올려놓는 판 | 기존 사파이어에서 고품질 실리콘으로 전환 중 |
| **인듐 (In)** | 칩 연결 | 플립칩 본딩(칩을 뒤집어 연결)에 사용 |

**조셉슨 접합(Josephson Junction)**이라는 부품이 특히 중요해요. 이건 두 초전도체 사이에 아주 얇은(수 나노미터) 절연층을 끼운 것으로, 초전도 큐비트의 **핵심 부품**이에요.

비유: 조셉슨 접합은 양자컴퓨터에서 **트랜지스터(스위치)와 같은 역할**이에요. 보통 반도체에서 트랜지스터가 0과 1을 만들듯, 조셉슨 접합이 양자 0과 1(큐비트)을 만들어요.

**코히런스 시간의 발전:**
- 업계 표준: 약 70마이크로초 (0.00007초)
- Princeton 최신 성과 (2025.11): **1밀리초 (0.001초) 이상!** — 업계 표준의 약 15배!

0.001초가 짧아 보이지만, 양자 연산 한 번에 약 0.00000001초(10나노초)밖에 안 걸리니까, 코히런스 시간 안에 **10만 번** 이상 연산할 수 있어요!

#### 이온 트랩용 소재

- **이트륨(Y), 바륨(Ba)** 이온
- 진공 챔버, 전극 어레이
- 정밀 레이저, 마이크로파 장비

#### 광자 큐비트용 소재

- **박막 리튬 니오베이트(TFLN)** 칩
- 단일 광자 검출기
- 도파관, 빔스플리터

### 2-3. 제어 전자장비 — 큐비트를 조종하는 장비

양자 프로세서를 만들었다고 끝이 아니에요. 큐비트를 실제로 **조종**하려면 정밀한 제어 장비가 필요해요. 이 영역이 사실 **확장성의 핵심 병목**이에요.

#### 마이크로파 제어 (초전도 큐비트용)

초전도 큐비트는 **마이크로파 펄스**(전자레인지에서 나오는 것과 비슷한 전파)로 조종해요. 아주 정밀한 주파수와 타이밍으로 마이크로파를 쏘면, 큐비트의 상태가 변해요.

비유: 기타 줄을 특정 주파수로 튕기면 특정 소리가 나듯이, 큐비트에 특정 주파수의 마이크로파를 쏘면 특정 양자 연산이 수행돼요.

**주목할 기술 발전:**
- **Oxford Ionics** (IonQ가 인수): 레이저 대신 **칩 내 전자신호**로 이온 트랩 큐비트를 제어하는 EQCC 기술 개발. 커다란 레이저 장비를 없앨 수 있어서 확장성에 큰 도움!
- 새로운 마이크로칩 소자: 기존 대비 **마이크로파 전력 80배 절감**

#### 레이저 제어 (이온 트랩/중성 원자용)

이온 트랩, 중성 원자 방식은 **레이저**로 큐비트를 조종해요.

문제는 큐비트 하나하나에 레이저를 정밀하게 쏘아야 해서, 큐비트 수가 늘어나면 레이저 장비도 기하급수적으로 복잡해진다는 거예요.

**Monarch Quantum** (2026.01 출범): 레이저, 변조기, 제어 전자장치를 하나의 모듈로 패키징해서 "플러그 앤 플레이"처럼 쓸 수 있게 만드는 회사예요.

#### 왜 제어 장비가 병목일까?

양자 오류 보정을 실시간으로 하려면:
- 매초 **수백만 개**의 오류 신호를 처리해야 해요
- **1마이크로초(백만 분의 1초)** 이내에 피드백 보정을 수행해야 해요
- 현재 문제는 큐비트 자체가 아니라, **이 신호를 처리하는 기존(고전적) 전자장치의 속도**가 따라가지 못하는 거예요!

### 2-4. 양자 오류 보정 (QEC) — 양자컴퓨팅의 최대 도전

앞의 배경지식 파일에서도 설명했지만, 밸류체인 관점에서 더 자세히 봐볼게요.

#### 주요 코드 방식의 전환

**표면 코드(Surface Code)**에서 **qLDPC 코드**로 전환하는 것이 업계 최대 트렌드예요.

- **표면 코드**: 지금까지 가장 많이 연구된 방식. 하지만 안정적 큐비트 1개를 만드는 데 물리적 큐비트가 **수천~수만 개** 필요해서 비효율적
- **qLDPC 코드**: IBM이 밀고 있는 새 방식. 물리적 큐비트 오버헤드를 **90% 감소**시킬 수 있어요!

비유: 표면 코드가 "100명의 경비원이 1명의 VIP를 보호"하는 거라면, qLDPC는 "10명의 더 똑똑한 경비원이 같은 수준으로 보호"하는 거예요.

#### QEC 관련 주요 성과들

| 시기 | 성과 | 의미 |
|---|---|---|
| 2024.12 | 구글 Willow, 오류 임계값 이하 달성 | 큐비트 늘리면 오류가 줄어드는 것을 최초 증명! |
| 2025~26 | IBM, qLDPC 코드로 전환 | 필요 큐비트 90% 감소 가능 |
| 2026 | IBM Kookaburra, qLDPC 메모리 + 논리 프로세서 | 최초의 실용적 오류 보정 모듈 |
| 2027 | Microsoft + Atom Computing, Magne | 50 논리 큐비트 (1,200 물리 큐비트) |
| 2029 | IBM Starling | 200 논리 큐비트, 1억 게이트 |

#### 인력 부족이 심각!

전 세계 QEC 전문가는 약 **600~700명** 밖에 안 돼요. 2030년까지 **5,000~16,000명**이 필요할 것으로 예상되는데, 인력 양성이 업계의 큰 과제예요.

#### QEC 전문 기업

- **Riverlane** (영국): QEC 소프트웨어 전문 기업. 디코딩 하드웨어도 개발
- **Iceberg Quantum**: RSA-2048 해독에 필요한 물리 큐비트를 100만 개 이상에서 **10만 개 이하**로 줄일 수 있다고 주장

## 3. 소프트웨어 — 양자컴퓨터를 쓰는 도구들

### 3-1. 양자 프로그래밍 프레임워크

양자컴퓨터에서 계산을 하려면 **양자 프로그래밍 도구**가 필요해요. 보통 컴퓨터에서 Python, Java 같은 프로그래밍 언어를 쓰듯, 양자컴퓨터에도 전용 도구가 있어요.

현재 양자 프로그래밍은 거의 대부분 **Python** 기반이에요 (채용 공고의 65~70%).

| 프레임워크 | 만든 회사 | 쉽게 말하면 | 특징 |
|---|---|---|---|
| **Qiskit** | IBM | 양자컴퓨팅의 "공용어" | 가장 많이 쓰임. 무료. IBM 양자컴퓨터에 직접 연결 |
| **Cirq** | Google | 구글 전용 도구 | 하드웨어를 세밀하게 제어 가능 |
| **PennyLane** | Xanadu | 양자 AI 전문 | 여러 하드웨어와 호환. 양자 머신러닝에 특화 |
| **Q#** | Microsoft | 마이크로소프트 전용 | Azure 클라우드와 통합 |
| **Braket SDK** | AWS | 아마존 전용 | 여러 종류 양자컴퓨터를 한 곳에서 접근 |

비유: Qiskit이 "안드로이드"라면, Cirq은 "iOS", PennyLane은 "크로스 플랫폼"인 셈이에요. 각각 장단점이 있지만, Qiskit이 현재 가장 넓은 생태계를 가지고 있어요.

### 3-2. 양자 클라우드 서비스

양자컴퓨터를 직접 사는 건 너무 비싸니까 (수십~수백억 원!), 대부분의 사용자는 **클라우드**로 접근해요. 인터넷으로 양자컴퓨터를 원격 사용하는 거예요.

| 서비스 | 제공사 | 특징 | 쉽게 비유하면 |
|---|---|---|---|
| **IBM Quantum** | IBM | 자체 양자컴퓨터만 제공. 가장 성숙 | "애플 스토어" (자체 제품만) |
| **Amazon Braket** | AWS | 여러 회사 양자컴퓨터를 한 곳에서 | "아마존 마켓플레이스" (여러 브랜드) |
| **Azure Quantum** | Microsoft | 양자+AI+일반 컴퓨팅 통합 | "원스톱 솔루션" |
| **Google Quantum AI** | Google | 자체 양자컴퓨터 + 연구 | "구글 연구소" |

비유: 양자 클라우드는 **PC방**과 비슷해요. 비싼 게이밍 PC를 직접 사지 않고, PC방에서 시간당 요금을 내고 쓰는 것처럼, 비싼 양자컴퓨터를 직접 사지 않고 클라우드에서 사용료를 내고 쓰는 거예요.

## 4. 양자 통신과 보안 — 도청 불가능한 통신

이 분야는 양자컴퓨팅과는 다르지만, 밀접하게 연결돼 있어요. 양자컴퓨터가 기존 암호를 깨뜨릴 수 있으니, 양자 기술로 보안도 강화해야 하는 거죠.

### 4-1. QKD (양자키분배) — 도청 불가능한 열쇠 전달

#### QKD란?

두 사람이 비밀 대화를 하려면 **암호 열쇠(key)**를 안전하게 나눠 가져야 해요. QKD는 이 열쇠를 **양자 역학의 원리**로 전달해서, 누가 중간에 엿보면 **즉시 감지**할 수 있는 기술이에요.

비유: 마법 편지를 상상해보세요. 이 편지는 정해진 사람만 읽을 수 있고, 다른 사람이 몰래 열어보면 편지의 글씨가 **자동으로 바뀌어서** 도청 사실을 즉시 알 수 있어요. 양자 역학에서는 관측 자체가 상태를 바꾸기 때문에, 이게 실제로 가능해요!

#### QKD 시장 규모

- 2024년: 4.5억 달러
- 2030년 전망: **24.9억 달러** (CAGR 33.5%)
- QKD 서비스 이용 기업: 2025년 약 1,000개 → 2030년 3,000개

### 4-2. PQC (양자내성암호) — 양자컴퓨터도 못 푸는 암호

#### PQC란?

양자컴퓨터가 기존 암호(RSA 등)를 깰 수 있으니, 양자컴퓨터도 **못 푸는 새로운 수학 문제**를 기반으로 만든 암호예요. "격자(lattice)" 구조라는 매우 복잡한 수학 문제를 사용해요.

비유: 기존 암호가 "아주 큰 수의 곱셈을 거꾸로 푸는 것"이라면 (양자컴퓨터가 이걸 잘 풀 수 있음), PQC는 "다차원 공간에서 가장 가까운 점을 찾는 것"으로 바꾼 거예요 (양자컴퓨터도 잘 못 풀음).

#### NIST 표준화 현황

| 표준 | 이름 | 용도 | 상태 |
|---|---|---|---|
| **FIPS 203** | ML-KEM (Kyber) | 암호 열쇠 교환 | 2024.08 **확정** |
| **FIPS 204** | ML-DSA (Dilithium) | 디지털 서명 | 2024.08 **확정** |
| **FIPS 205** | SLH-DSA (SPHINCS+) | 디지털 서명 (백업) | 2024.08 **확정** |
| FIPS 206 | FN-DSA (FALCON) | 디지털 서명 | 2026~2027 확정 예상 |

**핵심 의무**: NIST는 양자에 취약한 기존 암호를 **2035년까지 완전 제거**하도록 규정했어요. 그래서 정부, 금융, 국방 분야에서 PQC 전환이 이미 시작됐어요!

#### PQC 시장 전망

- **2033년까지 226.8억 달러** (약 30조 원!)
- "지금 모아두고 나중에 해독" 위협 때문에, 데이터를 다루는 **모든 기업**이 결국 PQC로 전환해야 해요

#### PQC 관련 주요 회사

| 회사 | 나라 | 특징 |
|---|---|---|
| **SandboxAQ** | 미국 | Alphabet(구글 모회사)에서 분사. 10억 달러 이상 투자 유치 |
| **PQShield** | 영국 | PQC 하드웨어/소프트웨어 IP 제공. 6,300만 달러 투자 유치 |
| **IBM** | 미국 | NIST 격자 암호 알고리즘 개발 주도. z17 메인프레임에 PQC 하드웨어 내장 |
| **Post-Quantum** | 영국 | PQC 전문 솔루션 기업 |

#### PQC 업계 연합

**Post-Quantum Cryptography Coalition**: IBM, Microsoft, MITRE, PQShield, SandboxAQ 등이 공동 설립. 125개 이상 조직 참여.

## 5. 현실적 한계와 도전 과제

양자컴퓨팅이 장밋빛 전망만 있는 건 아니에요. 아직 넘어야 할 산이 많아요.

### 1) 큐비트 안정성

양자 상태는 **극도로 연약**해요.

- 초전도 큐비트의 코히런스 시간: 약 0.07밀리초 (업계 표준) ~ 1밀리초 (최신 기록)
- 주변 온도, 진동, 전자기파, 심지어 우주에서 오는 방사선까지 양자 상태를 방해해요
- 이웃한 큐비트끼리의 간섭(크로스토크)도 문제

비유: 비눗방울 위에 글씨를 쓰는 것과 비슷해요. 아주 살짝만 건드려도 비눗방울이 터지면서 글씨가 사라져요.

### 2) 오류율

| 항목 | 현재 수치 | 목표 수치 |
|---|---|---|
| 구글 Willow 오류율 | 0.143%/cycle | 0.0001% (실용 수준) |
| 물리큐비트/논리큐비트 비율 | 1,000~10,000:1 | 100:1 이하 (목표) |

아직 실용 수준과 **1,000배 이상** 차이가 나요.

### 3) 확장성

현재 최대 약 150큐비트인데, 실용적 문제를 풀려면 **수십만~수백만 큐비트**가 필요해요. 큐비트 수를 늘릴수록:
- 냉각이 더 어려워지고
- 제어 장비가 기하급수적으로 복잡해지고
- 큐비트 간 간섭이 늘어나요

### 4) 인력 부족

- QEC 전문가: 전 세계 **600~700명**
- 2030년까지 필요: **5,000~16,000명**
- 양자 소프트웨어 개발자도 극심한 부족

### 5) 상용화 타임라인

| 시기 | 예상 수준 |
|---|---|
| **2026** | 특정 문제에서 "양자 우위" 실증 (IBM 목표) |
| **2027~2029** | 50~200 논리 큐비트. 일부 실용적 문제 풀기 시작 |
| **2030년대** | 대규모 내결함성 양자컴퓨터(FTQC) 실현 |
| **2035~2040** | 본격적 상업적 가치 창출 |

**핵심 메시지**: 양자컴퓨팅은 **"혁명적이지만 아직 멀다"**는 점을 기억하세요. 투자 시 최소 10년 이상의 장기 시각이 필요해요.

## 6. 투자 포인트 요약

### 밸류체인에서 주목할 성장 영역 5가지

| 영역 | 왜 주목? | 시장 전망 |
|---|---|---|
| **극저온 냉각** | 양자컴퓨터 성장에 따라 필수. Bluefors/Oxford 과점 | 2027년부터 CAGR 27% |
| **오류 보정 (QEC)** | 양자컴퓨팅의 핵심 차별화 요소. 전문 인력 극심 부족 | 기술 성숙에 따라 폭발적 성장 |
| **PQC 전환** | 2035년 NIST 의무화. 모든 기업이 해야 함 | 2033년까지 226.8억 달러 |
| **제어 전자장비** | 확장성의 핵심 병목. 혁신 기회 존재 | 기술 혁신 주도 성장 |
| **한국 양자보안** | 통신 3사 + 정부 정책 주도 | CAGR 39.8%, 2030년 24.6조 원 |

### 밸류체인 위치별 투자 접근

```
상류(소재/부품)    → 아직 시장이 작고 전문 기업 적음. 장기적 관점
중류(하드웨어)     → Bluefors, ASML 같은 독점적 지위의 기업 주목
하류(SW/서비스)    → IBM Qiskit, 클라우드 서비스. 플랫폼 선점 중요
보안(PQC/QKD)     → 가장 빠른 상업화. 이미 현실적 수요 존재!
```

### 핵심 메시지

1. **양자컴퓨팅 자체**는 상업화까지 아직 5~10년 이상 걸려요
2. **양자 보안(PQC/QKD)**은 **지금 당장** 돈이 되는 분야예요 (NIST 의무화!)
3. **극저온 냉각**, **오류 보정**은 양자컴퓨팅이 성장할수록 필수적인 병목 해소 영역이에요
4. 한국에서는 **SK텔레콤**이 양자 생태계의 중심이에요
5. 투자는 **소규모, 분산, 장기** 전략이 안전해요
