## 1️⃣ 회사 개요

**(주)한국카본 (Hankuk Carbon Co., Ltd.)** — 1984년 설립된 탄소섬유·합성수지 기반 종합 복합소재 기업. **밀양 본사** (1988년 이전) + 1995년 **KOSPI 상장**(티커 017960).

| 항목 | 내용 |
|---|---|
| 설립 / 상장 | 1984년 / 1995년 KOSPI 상장 |
| 본사·생산 | 경남 밀양 본사·1·2공장(LNG 보냉재 + 프리프레그), 완주·예산(방산·항공) |
| 회장 | **조문수** (창업주 조용준 회장 장남, 1997년 대표 취임, 한국드론산업진흥협회장 겸임) |
| 3세 승계 | **조연호** 전략기획실장 — 2024-06+12 주식 증여로 **22.86% 최대주주 등극** |
| 주요주주 (2026) | **조연호 외 4인 34.95%**, **국민연금 10.31%**, 외국인 14.66% |
| 주요 자회사·관계 | **한국신소재**(2023.9 흡수합병, 글래스패브릭·프리프레그 수직통합), **HANKUK COMPOSITE US**(2026.4 신설, 300억 출자), **누보**(농자재, KOSDAQ 332290), KAT(IAI 합작 무인기) |

- **② 섬유강화복합재** — 카본/유리섬유 프리프레그(T700·T800급), 풍력 블레이드 소재
- **③ 방산·우주항공** — KF-21·FA-50 동체, 천무·천궁 발사관, 누리호 페어링·동체 스커트
- **④ AAM/UAM·드론(신성장)** — Vertical Aerospace 벨류체인 후보, IAI 합작 KAT 사단급 무인기

## 2️⃣ 제품·핵심 기술

### ① LNG 보냉재 — 독점적 기술장벽

- **GTT Mark III + NO96 멤브레인 시스템 라이선시** (국내 단 2개사: 한국카본·동성화인텍).
- 제품 구조: **1차 Insulation Box** (강화 PU폼 + 합판 상하판 + **Triplex 2차 방벽**(알루미늄 필름 + 글래스패브릭 샌드위치)) → -163℃ 극저온에서 BOR(Boil-Off Rate) 최소화.
- **Triplex 2차 방벽 글로벌 독점 공급** (1999년 세계 최초 국산화, 25년+ 양산 경험).
- 글로벌 보냉재 M/S **45~50%**, 한국 빅3(HD현대중공업·삼성중공업·한화오션) **전사 공급**.
- 1Q26 수주잔고 **USD 12.93억 ≈ 1.75조원** (2020년 대비 ×3).

### ② 섬유강화복합재(프리프레그·FRP)

- 국내 **최초(1984)** 탄소섬유 프리프레그 상용화.
- 카본 프리프레그: **T700급(산업·스포츠)** + **T800급(항공·방산)**.
- **열가소성(Thermoplastic) CFRP 자체 개발** — 재활용·열성형 가능, eVTOL·EV 배터리 케이스 타깃.
- 풍력 블레이드: 2017년 두산중공업 국책과제 완료, 서남해 해상풍력 실증단지 공급.
- 2023년 **한국신소재 흡수합병**으로 글래스패브릭·프리프레그 원소재 자급률 상승.

### ③ 방산·우주항공

| 분야 | 제품 | 고객 |
|---|---|---|
| 전투기 동체·구조재 | FA-50, KF-21 보라매 | **KAI** |
| 미사일 발사관 | 천무·천궁 원통형 복합재 (필라멘트 와인딩) | **한화에어로스페이스, LIG넥스원** |
| 우주발사체 | 누리호 페어링·동체 스커트 라미네이트 | **KARI** (4차 발사 2025.11 성공, 2027년까지 2회 추가) |
| 민항 | A350 부품 (Tier-2) | **에어버스** |

### ④ AAM/UAM (신성장)

- **Vertical Aerospace VX4** 협력 벨트(한화에어로-Vertical MOU 2024) 기반 복합재 공급 파이프라인.
- **이스라엘 IAI 합작 KAT** — 차기 사단급 UAV 동체 복합재 공급.
- C004B 난연 레진 시스템 — UAM 플로어용 프리프레그 상용화.

### 지식재산권·인증
- **GTT Mark III / NO96 라이선스** (핵심 진입장벽)
- **AS9100(KS Q 9100)** + **NADCAP** 항공우주 품질 인증
- 방위사업청 **방산업체 지정** + 방산 시설보안 인증

### 2025~2026 전략 포지셔닝
1. **LNG 슈퍼사이클 수혜 본격화** — 카타르 NFE Phase 2(151척 검토), Venture Global CP2(12척 임박), 한화오션 2.6조 LNGC 7척(2025.12) → **2027~2030 인도분 매출 레버리지**
2. **수직계열화 완성** — 한국신소재 합병 시너지로 OPM 회복(FY2023 0.3% → FY2025 14.4%)
3. **방산/우주 비중 확대** — KF-21 양산(~2032년 120대), 누리호 6호기까지 발사
4. **AAM/드론** — 2028년 UAM 상용화 1단계 대비 Vertical 벨류체인 편입 시도
5. **美 생산법인** — MASGA(美 조선업 재건) 정책 + 항공·방산 진출 거점
