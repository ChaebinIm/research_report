## 1. Executive Summary (3-5분 + 핵심 표)

### 1.1 한 줄 요약

> **"바다 밑 인터넷의 99%, 바다 위 풍력의 100%가 모두 한 종류의 자산 — 해저케이블 — 을 통과한다. 2026년 시점에서 이 자산은 글로벌 4-5개 그룹이 사실상 과점 중이며, 발주잔량은 케이블 회사 시가총액보다 크다."**

### 1.2 5분 핵심

- 해저케이블은 **두 종류**: ① 전력(해상풍력·국가간 송전) ② 통신(인터넷 백본).
- 전력 해저케이블 시장 **2024 약 $10B → 2030 $30B+**, 통신 **2024 $4B → 2030 $10B+** (BloombergNEF·Wood Mackenzie·Submarine Telecoms Forum 추정).
- 글로벌 점유율: 전력은 **Prysmian(~30%) + Nexans(~12%) + NKT + LS + 일본계**로 사실상 5강. 통신은 **SubCom(美) + ASN(노키아) + NEC(日) + HMN(中)**의 4강.
- 2026년 5월 현재 **Prysmian Transmission 잔고 약 €16B**, **Nexans Transmission 잔고 €8.1B**, **LS전선 그룹 전체 잔고 7.63조 원(2025년말 +22% YoY)**, **대한전선 잔고 3.25조 원 (역대 최고)**.
- 폭증 동인 4개: ① 글로벌 해상풍력 92GW(2025) → 236GW(2030) ② 빅테크 자체 케이블 발주 (Meta Project Waterworth $10B 등 2025-27 누적 $13B) ③ 국가간 인터커넥션 (Viking Link·SuedLink·North Sea Link) ④ 미·EU 그리드 갱신 + 사이버보안 강화.
- **한국 직접 노출 종목**: LS(006260, 모회사) — LS전선아시아(229640) — LS마린솔루션(060370) — 대한전선(001440). 비상장 LS전선 자체는 LS 지분으로 우회.
- **부설선(CLV) 부족**이 이 산업의 핵심 진입장벽 + 마진 보호 장치. 글로벌 Top 10 부설선이 **수년치 풀가동 상태**.
- 리스크 1순위: **트럼프 2기의 미국 해상풍력 동결**. 2026년 4월 기준 기존 5개 동부 해상풍력은 진행하되 신규 리스 일시 중단, $1B 보상금으로 일부 프로젝트 취소.

## 2. 해저케이블이 뭔가 — 5분 입문

### 2.1 한 줄 정의

**바다 밑에 까는 케이블.** 끝. 두 종류 = 전기를 보내는 것, 정보를 보내는 것.

### 2.2 두 종류, 두 시장

#### (가) 전력 해저케이블 (Power Submarine Cable)

- **용도**: ① 해상 풍력 발전기에서 만든 전기를 육지로 보내는 송전선 ② 나라와 나라 사이를 잇는 송전선 (예: 영국-노르웨이 North Sea Link 1.4GW) ③ 섬을 본토와 잇는 송전선 (예: 제주 ↔ 본토 HVDC).
- **방식**: 두 가지가 있음 — **HVAC**(고압 교류, ~150km 한계) vs **HVDC**(고압 직류, 100~수천 km 가능).
- **재질**: 구리(Cu) 도체 + XLPE 절연 + 강철 보호층 + 외부 보호층. 보통 한 가닥 무게가 km당 50~200톤.
- **가격**: 525kV HVDC 케이블 1km = **약 $5-8M** (50-80억 원). 100km 짜리 = 5,000-8,000억 원. 비싸다.

> **비유**: 전력 해저케이블 = **바다 밑 송수도관**. 두꺼울수록 많은 물(전기)을 멀리 보낼 수 있는데, 두꺼우면 무거워서 부설(설치)이 어렵다.

#### (나) 통신 해저케이블 (Telecom Submarine Cable)

- **용도**: 인터넷 백본의 99%. 대륙과 대륙 사이 데이터를 옮긴다. 위성? 1-2%도 안 된다.
- **방식**: 광섬유(Optical Fiber) 다발 + 외부 보호층. 광섬유 = 머리카락보다 가는 유리 가닥, 빛으로 데이터 전송.
- **거리**: 수천~수만 km. 중간에 **Repeater(중계기)** 또는 **EDFA(어븀 첨가 광증폭기)** 가 50-100km마다 들어감. 빛이 약해지지 않게 다시 증폭해주는 장치.
- **가격**: 1건당 **약 5,000억-1.5조 원**. 길이·지점 수에 따라 다름. Meta의 Project Waterworth는 **5만 km, $10B(약 13조 원)** 단일 건이다.

> **비유**: 통신 해저케이블 = **바다 밑 우편 도로**. 많은 차선(광섬유 쌍)을 까는 게 핵심. 한 가닥에 **20-30Tbps**가 흐른다. 24-fiber-pair 케이블이면 한 케이블에 약 **500-700Tbps** = **초당 700조 비트** = 4K 영화 1억 편 동시 전송.

### 2.3 4단 가치사슬

```
[1] 케이블 제조        [2] 매설/설치        [3] 부설선          [4] 유지보수
 - Prysmian             - Boskalis,         - CLV               - 24/7 모니터링
 - Nexans                 Subsea7            (Cable Lay         - 보호선
 - NKT, LS              - LS마린              Vessel)           - 보안 점검
 - 케이블 자체            - JDR               - 글로벌 ~30척      - 절단 시 긴급
   = 부피의 99%          - Saipem 일부          (만년 부족)         복구
   = 매출의 60%
```

각 단계가 별개의 시장이지만, **케이블 회사들이 점차 4단을 모두 통합**하는 중(Prysmian·Nexans·LS그룹 모두). 통합형이 마진이 더 좋다.

### 2.4 단위 — 모르면 본문이 안 읽힌다

| 단위 | 뜻 | 예시 |
|---|---|---|
| **km** | 케이블 길이 | Project Waterworth = 50,000 km |
| **GW** | 전력 송전 용량 (1 GW = 100만 kW) | North Sea Link = 1.4 GW (서울 시민 약 280만 명 동시 전기 사용량) |
| **Tbps** | 통신 용량 (Terabits per second) | 24-fiber-pair = 약 600 Tbps |
| **kV** | 전압 | 525kV HVDC = 현존 최고 상용 전압 |

### 2.4-1 케이블 한 가닥 안에는 무엇이 들어 있나 (해부도)

전력 해저케이블 한 가닥(525kV HVDC 기준)의 단면을 보면:

```
[중심] 도체(Cu)                ← 구리 1,500mm² (전기 흐르는 길)
        ↓
        반도전층 (Inner Semiconductor)   ← 전기장 균일화
        ↓
        절연층 (XLPE)                    ← 가교 폴리에틸렌, 두께 25-30mm
        ↓
        반도전층 (Outer Semiconductor)
        ↓
        납피복(Lead Sheath)              ← 수밀(海水 차단)
        ↓
        PE Jacket
        ↓
        강철 Armour Wire                 ← 외부 충격·앵커 보호
        ↓
        외피 Polypropylene Yarn
[외경] 약 13-15cm. 무게 km당 50-90톤.
```

> **포인트**: 케이블 1가닥 = 6-8개 층. 한 층이라도 결함 = 전체 폐기. 그래서 **품질·인증(IEC 62067 등)** 이 진입장벽. 신규 회사가 525kV 인증을 받는 데 **3-5년** 걸린다.

통신 해저케이블은 더 단순:

```
[중심] 광섬유 다발 (12-24-32 fiber pair)
        ↓
        스틸 튜브 + 젤 (수분 차단)
        ↓
        구리 도체 (중계기 전원 공급용)
        ↓
        절연 + 강철 Armour
[외경] 약 2-3cm 심해용, 5-10cm 천해용 (어선·앵커 보호 강화).
```

### 2.4-2 부설(Installation)이 더 비싸다는 충격적 사실

해저케이블 프로젝트 비용 분해 (525kV HVDC, 100km 기준):

| 항목 | 비용 | 비중 |
|---|---|---|
| 케이블 자체 | $500-800M | 50-60% |
| **부설 (CLV 운임)** | **$150-250M** | **15-20%** |
| 매설 (Burial) | $80-120M | 8-10% |
| 해저 조사·환경 평가 | $30-50M | 3-5% |
| 변환소(Converter Station, HVDC) | $200-400M | 20-25% |
| 보험·금융 비용 | $30-60M | 3-5% |

> **핵심 인사이트**: 부설+매설+조사가 **30%+** = 케이블 회사가 부설선까지 통합하면 마진이 더 좋아진다. 이게 Prysmian·Nexans·LS가 부설선을 직접 보유·신조하는 이유.

### 2.5 용어 박스 (첫 등장 약어 모두 풀어쓰기)

| 약어 | 풀이 | 본문 의미 |
|---|---|---|
| **HVDC** | High Voltage Direct Current = 고압 직류 송전 | 장거리·해저용. 손실 적음 |
| **HVAC** | High Voltage Alternating Current = 고압 교류 송전 | 단거리(<150km) 해저용 |

| **VSC** | Voltage Source Converter = 전압형 컨버터 | HVDC 변환방식, 양방향 가능 |
| **LCC** | Line-Commutated Converter = 전류형 컨버터 | 옛날 HVDC, 단방향 |
| **Repeater** | 중계기 | 통신 케이블 중간 신호 증폭기 |

| **Burial** | 매설 | 케이블을 해저 아래 1-3m 묻는 작업 |
| **CLV** | Cable Lay Vessel = 케이블 부설선 | 해저케이블 까는 전용 배 |
| **Tbps** | Terabits per second | 통신 용량 단위 |
| **SSAC** | Subsea Cable Action Coalition (美) | 미국 해저케이블 보안 연합 |
| **OPM** | Operating Profit Margin = 영업이익률 | |
| **CapEx** | Capital Expenditure = 자본적 지출 | 설비투자 |
| **EV/EBITDA** | 기업가치/세전이자감가전이익 | 자본집약 산업 표준 멀티플 |
| **P/Backlog** | 시가총액/수주잔고 | 잔고 가시성 큰 산업 핵심지표 |
| **OFW** | Offshore Wind = 해상풍력 | |
| **FOW** | Floating Offshore Wind = 부유식 해상풍력 | |
| **CFIUS** | 미국 외국인투자심의위 | 중국 케이블 견제 도구 |

> **숫자 환산**: $30B = 약 **40조 원**. 한국 GDP 2,200조의 **1.8%**. 삼성전자 매출 약 **15% 수준**의 시장이 매년 케이블만 깔린다.

#### (나) 발주잔량 (Backlog)

2026년 5월 현재 글로벌 전력 해저케이블 발주잔량 **약 $50-60B 추정** (Prysmian €16B + Nexans €8.1B + NKT €10B+ + LS 7.6조 원 그룹 + 대한전선 3.25조 원 + Sumitomo·Furukawa·Hellenic 등).

**잔고/매출 비율**: 글로벌 평균 **3-5배**. 즉 지금 발주잔고만으로 **3-5년치 매출이 확정**된 상태. 변압기 (잔고/매출 ~2-3년)보다 더 길다. 그래서 가시성이 매우 큼 = **P/Backlog가 가장 중요한 멀티플**.

#### (다) 4대 구동 동인

1. **해상풍력 어레이/송전 케이블** (60%): 2025년 92GW → 2030년 236GW. 1GW당 케이블 발주 약 $200-400M.
2. **국가간 인터커넥션** (15%): EU 그리드 통합 (Viking Link 1.4GW 영-덴, North Sea Link 1.4GW 영-노 완공, SuedLink 4GW 독일 남북 2028 완공 예정).
3. **섬-육지 연결** (10%): 한국 제주 ↔ 본토 HVDC 3·4호기, 일본 홋카이도-혼슈, 호주 Tasmania.
4. **미·EU 그리드 갱신** (15%): 노후 케이블 (1990년대 설치) 교체 사이클.

### 3.3 사이클 — 7-10년 + AI/풍력으로 가속

#### (가) 옛날 사이클

해저케이블은 본래 **7-10년 단위 사이클**이었다. 2002-09 닷컴 거품, 2010-17 중국 통신 인프라, 2018 풍력 투자 본격화, 2022~ 슈퍼사이클 진입.

#### (나) 지금 — 사이클이 깨졌다

- 풍력(전력) + AI(통신) **두 동력이 동시에 폭발** 중.
- 부설선과 고전압 케이블 라인 = **글로벌 풀가동**.
- Prysmian·Nexans·NKT 모두 2027-28년까지 capacity expansion 진행. 그래도 부족하다는 게 IEA의 진단.

> **결론**: 이번 사이클은 **2030년 전까지 평탄한 상승 + 약간의 변동** 정도로 보는 게 합리적. 단, 2027-28년 신규 capacity 가동 시점에 **단가 인하 압력**이 올 수 있음 (리스크 13장 참조).

### 3.5 매출-잔고-실적 3단계 시차 (Sales Recognition Cycle)

```
[T0] 입찰·발주 (Letter of Intent)
   ↓
[T0+3-6개월] 정식 계약 → 잔고 등재
   ↓
[T0+6-12개월] 케이블 제조 시작 (Long Lead Item)
   ↓
[T0+12-24개월] 부설선 동원 + 부설 → 매출 인식 (Percentage-of-Completion)
   ↓
[T0+24-36개월] 시운전·인도·잔금
```

> **투자 시사점**: 잔고가 오늘 +20% 늘어도 매출은 **12-18개월 뒤**에야 반영. 따라서 **잔고 추이가 주가 선행 지표**. P/Backlog가 EV/EBITDA보다 더 빠른 신호.

## 4. 전력 해저케이블 — 글로벌 종목 심층 (가장 길게)

### 4.1 Prysmian (PRY.IT, 이탈리아) — 세계 1위

#### (가) 회사 개요

- **국적**: 이탈리아 밀라노 본사. 1879년 Pirelli Cables 출발 → 2005년 분사 → 2018년 General Cable 인수 (北美 진출) → **2024년 Encore Wire 인수 ($4.5B)** 로 미국 점유율 폭증.
- **사업부**: ① Transmission (해저케이블·HVDC) ② Power Distribution ③ Industrial & Construction ④ Telecom.
- **상장**: Borsa Italiana (밀라노). 2026년 5월 시가총액 **약 €20-22B (~30조 원)**.

#### (나) 핵심 숫자

| 항목 | 2022 | 2023 | 2024 | 2025e | 2026e |
|---|---|---|---|---|---|
| 매출 (€B) | 16.0 | 15.4 | 17.0 | **18.5** | 20.5 |
| OPM (%) | 7.5 | 8.5 | 9.5 | **10.5** | 11.0 |
| Transmission 잔고 (€B) | 7 | 11 | 14 | **16** | 18+ |
| EPS (€) | 2.5 | 2.9 | 3.4 | 4.0 | 4.5 |

#### (다) 메가 프로젝트

- **SuedLink** (독일 남북 4GW HVDC): 케이블 공급 중. €1.5B 규모.
- **Viking Link** (영국-덴마크 1.4GW HVDC): 2024년 완공. 케이블 Prysmian + NKT 분담.
- **TenneT BorWin5/6** (독일 북해 풍력 → 본토 2GW): 525kV HVDC.
- **East Anglia Three** (영국 1.4GW 풍력): Array + Export Cable.
- **미국 Sunrise Wind·Empire Wind** (트럼프 보상 후 일부만 진행): Export 케이블.

#### (라) 강점·약점

**강점**:
- **525kV XLPE HVDC 케이블** 양산 가능한 회사 = 글로벌 3-4개뿐 (Prysmian·NKT·Nexans·Sumitomo).
- 미국·이탈리아·핀란드·UK·노르웨이 등 **글로벌 5대륙 공장** 보유.
- 부설선 **6척** (Leonardo da Vinci, Cable Enterprise 등).
- Encore Wire 인수로 **미국 매출 비중 35%+** = IRA 직접 수혜.

**약점**:
- Encore Wire 인수 직후 **레버리지 상승** (Net Debt/EBITDA ~2.5배).
- 노조 강한 이탈리아·프랑스 공장 = 임금 상승 압력.
- 트럼프 2기 미국 풍력 동결 = 단기 미국 매출 상단 캡.

#### (마) 멀티플 (2026.5 시점 추정)

- Forward P/E **~17x** (시장 평균 ~16x보다 살짝 위)
- EV/EBITDA **~10x**
- P/Backlog (시가총액/Transmission 잔고) **~1.4x**
- 배당수익률 ~1.5%

#### (바) 투자 포인트 한 줄

> **"잔고 €16B + 미국 35% + 525kV HVDC 양산 = 글로벌 1위의 모든 조건. 단점은 멀티플이 이미 1위를 반영했다는 것."**

### 4.3-1 NKT 추가 — Tennet 구조적 파트너십

NKT의 점유율을 정확히 이해하려면 **Tennet** (네덜란드·독일 송전운영자, TSO)을 알아야 한다.

- Tennet은 독일·네덜란드 북해 풍력의 송전을 책임지는 공기업.
- 2030년까지 **약 €100B (~135조 원) 송전 투자** 계획. 이 중 약 **€40B가 해저 HVDC 케이블**.
- 주요 패키지: BorWin5, BorWin6, DolWin4, LanWin1-4, NordOstLink 등 **2GW HVDC 시리즈**.
- NKT는 이 중 **5-6개 패키지를 사실상 단독 또는 우선 공급사**로 확보.
- 잔고 €10B의 **70% 이상이 Tennet 관련**.

> **리스크 측면**: 단일 고객 의존도. Tennet이 Prysmian·Nexans 분담을 늘릴 경우 NKT 잔고 위축. 그러나 525kV 양산 가능 회사가 Prysmian·NKT 둘뿐이라 단기 변경 어려움.

### 4.3-2 NKT의 부유식 풍력 동적 케이블 우위

- **부유식 풍력 (Floating Offshore Wind, FOW)** = 수심 60m 이상에서 풍력기를 부유체에 장착.
- 한국·일본·대만·노르웨이·미국 캘리포니아 등 **수심 깊은 해역의 유일한 옵션**.
- 2030년 시장 추정 **18GW 운영, 케이블 발주 $3B**.
- 동적 케이블(Dynamic Cable)은 파도에 따라 휘어졌다 펴지는 **"피로 강도"** 가 핵심.
- NKT가 노르웨이 Hywind Tampen (88MW, 2023 가동)에 공급 → 트랙 레코드 1위.
- LS전선·Prysmian이 추격 중이지만 2-3년 뒤.

### 4.4 Sumitomo Electric (5802.T, 일본)

- **종합 전선·케이블 회사**. 자동차 와이어하네스가 매출의 50%, 해저케이블·전력은 약 20%.
- **해저케이블**: 일본 1위, 글로벌 5위 추정. 525kV XLPE 양산 가능.
- **부설선**: 3척 (Subsea Viking 등).
- **시가총액**: 약 ¥2T (~18조 원).
- 멀티플 Fwd P/E **~10x**, EV/EBITDA **~7x** = **저평가 + 자동차 사업 디스카운트**.
- 투자 포인트: 자동차 사이클 회복 + 해저케이블 분리 IPO 가능성 = 옵션 가치.

### 4.5 Furukawa Electric (5801.T, 일본)

- 일본 2위. 광섬유 + 전력 케이블.
- 통신 해저케이블 분야가 더 강함 (NEC와 협력 다수).
- 시가총액 약 ¥350B (~3조 원).

### 4.6 JDR Cable Systems (영국, 비상장)

- TKH Group 자회사. **풍력 어레이 케이블 (Inter-Array Cable)** 의 강자.
- 부유식 풍력 동적 케이블도 NKT와 함께 양강.
- 영국 Hartlepool 공장 + Cambois 신공장 (2026 가동) = €130M 투자.

### 4.7 Hellenic Cables (그리스)

- **Cenergy Holdings** (CENER.AT) 자회사. 그리스+남유럽 강자.
- 매출 ~€1B, 잔고 €3B. **Crete-Attica HVDC** 등 지중해 프로젝트 다수.
- 멀티플 매력: P/E **~10x**.

### 4.7-1 Cenergy Holdings + Hellenic Cables 추가

- **Cenergy Holdings (CENER.AT)** = 그리스 Athens 상장. 시가총액 **약 €1.8B**.
- 자회사 = **Hellenic Cables (케이블) + Sovel·Sidenor (강관)**. 통합 = "케이블 + 강관 + 부속" 일체화.

### 6.2 LS마린솔루션 (060370) — 부설·유지보수 + CLV

#### (다) 옵션 가치

- 2027-28년 신규 대형 프로젝트 (SuedLink·Project Waterworth·미 동부 풍력 잔여) 동시 시행 = 부설선 단가 상승 가능성 큼.

#### (라) 리스크

- 2028년까지 **신조선 미가동** = 잔고 가시성 약함.
- 매출 변동성 큼 (단일 부설선 운임 변동).
- 시장이 신조 모멘텀을 이미 반영 중 = 멀티플 P/E **~30x+** 추정.

### 6.3 대한전선 (001440) — 호반 인수 후 재무 개선

#### (가) 개요

## 7. AI 데이터센터 — 통신 해저케이블 폭증 메커니즘

### 7.1 빅테크 자체 케이블 발주 폭증

#### 메커니즘

1. AI 학습 = GPU 클러스터 분산 = 대륙 간 데이터 동기화 트래픽 폭증.
2. 빅테크는 통신사에 의존하지 않고 **자체 케이블** 발주 = 비용·통제권 확보.
3. 2025-27 누적 **$13B 신규 투자** = 직전 3년 대비 **거의 2배**.

> **숫자 환산**: $13B = 약 17조 원. **삼성전자 분기 영업이익 1-2개**, **포스코 시총의 1/3**. 이걸 빅테크 4사가 3년 안에 케이블에 쓴다.

### 7.2 데이터센터 ↔ 데이터센터 인터커넥트

기존 통신 케이블 = 사람들이 쓰는 트래픽 (Netflix·SNS·Search).
지금 = **DC ↔ DC 트래픽**이 더 큼. 빅테크 자체 트래픽이 인터넷 백본 트래픽의 **70%+** 차지 (Cisco Annual Internet Report).

### 7.3 AI 모델 분산 학습 → 대륙간 트래픽 비대칭

- 미국 Texas → 유럽 GPU 클러스터 모델 동기화.
- Meta·Google이 24-fiber-pair 케이블 도입 = 한 케이블에 24쌍의 광섬유 = 600Tbps+.
- 옛날 12-fiber-pair → 지금 24-fiber-pair → 차세대 32-fiber-pair 검토 중.

### 7.4 통신 케이블 기본 스펙

- **수명**: 25년 설계.
- **용량**: 200-1,000Tbps (현세대).
- **중계기 (Repeater) / EDFA**: 50-80km마다.
- **단가**: 1건당 5,000억-1.5조 원 (Project Waterworth는 13조 원 단일).

## 8. 해상풍력 — 전력 해저케이블 폭증 메커니즘

### 8.1 글로벌 발주잔량

- **2025년말 운영중 92GW** → **2030년 236GW** (RenewableUK Pipeline Report 2026.3 기준).
- 즉 **5년 안에 144GW 신규 = 매년 약 30GW 신규**.
- 1GW당 케이블 발주 **약 $200-400M** = 매년 **$6-12B 케이블 시장**.

### 8.2 풍력 케이블 두 종류

#### (가) Inter-Array Cable (어레이 케이블, 풍력기 ↔ 변전소)

- 33-66kV.
- 거리 짧고 다발이 많음.
- **JDR Cable (TKH 자회사) + Hellenic + LS + Prysmian** 강세.

#### (나) Export Cable (해상 변전소 ↔ 육상 그리드)

- 220-525kV HVDC 또는 HVAC.
- 거리 50-200km.
- **Prysmian·NKT·Nexans·LS·Sumitomo** 5강.

### 8.3 부유식 풍력 (Floating Offshore Wind, FOW)

#### Dynamic Cable (동적 케이블)

- 부유식 풍력기는 파도에 따라 움직임 → 케이블도 함께 움직여야 함.
- 일반 케이블은 깨짐 → **동적 케이블** 별도 기술 필요.
- 글로벌 강자: **NKT + JDR**. Prysmian·LS도 진입 중.
- 시장: 2025년 ~$0.5B → 2030년 **~$3B**.

## 11. 밸류에이션 — 어떻게 평가하나 + 현재 위치 (필수)

### 11.1 핵심 평가 지표 (Primer)

해저케이블 산업은 **자본집약 + 잔고가 매출보다 길다 + 사이클 7-10년**의 특성을 가진다. 따라서 일반 제조업과 다른 멀티플이 필요하다.

#### (가) P/Backlog — 가장 중요

- **계산**: 시가총액 ÷ 수주잔고.
- **의미**: 잔고 1원당 시장이 매기는 가격. 낮을수록 저평가.
- **벤치마크**: 1.0배 = 잔고 = 시가총액. 1.5배 이상 = 프리미엄.
- **한계**: 잔고 마진이 다 다름 (저마진 잔고 vs 고마진 잔고 구분 필요).

#### (나) EV/EBITDA — 자본집약 산업 표준

- **계산**: (시가총액 + 순부채) ÷ EBITDA.
- 자본집약형 산업은 감가상각 큼 → P/E 왜곡 → EV/EBITDA가 더 정확.
- 벤치마크: 8x 이하 = 저평가, 12x 이상 = 프리미엄.

#### (다) Forward P/E — 잔고 안정 시 보조

- 잔고 가시성 큰 산업(=수주산업)은 forward P/E가 잘 맞는다.
- 한국 LS·대한전선·LS마린솔루션은 P/E도 봐야 함.

#### (라) DCF — 장기 잔고 + 옵션 가치

- 부설선 옵션·신공장 가동 등은 DCF로만 평가 가능.
- 단, WACC 가정에 민감 (자본 비용).

#### (마) Replacement Cost — 진입장벽 평가

- 525kV HVDC 라인 1기 = **$500M-1B** 투자 + **3-4년 가동 준비**.
- 부설선 1척 = **$200-300M** + **3년 건조**.
- 글로벌 풀가동 + 재현 비용 큼 = **진입장벽이 시가총액의 일부 정당화**.

### 11.6 P/Backlog 정밀 계산 (Prysmian 사례)

```
Prysmian 시가총액 = €21B (2026.5 추정)
Transmission 잔고 = €16B
P/Backlog (Transmission only) = €21B / €16B = 1.31x

Total 그룹 잔고 (Power Distribution + Industrial 포함) = ~€25B
P/Backlog (Total) = €21B / €25B = 0.84x
```

해석:
- **1.31x** = Transmission 잔고 1유로당 시장이 1.31유로 평가. 적정.
- **0.84x** = Total 잔고 기준 0.84유로 = 다른 사업부의 마진이 낮다는 디스카운트.

> **투자 활용**: 회사별로 "Transmission only P/Backlog"와 "Total P/Backlog"를 함께 보고, **Transmission 비중이 높을수록 멀티플 정당화**. NKT는 100% pure-play라 Transmission P/Backlog만 보면 됨.

### 11.7 OPM 개선 사이클 — 마진 확장 효과

```
Prysmian OPM 2022 7.5% → 2026e 11.0% = +3.5%p
매출 16B → 20.5B → 영업이익 1.2B → 2.25B = +88%
같은 기간 EPS 2.5 → 4.5 = +80%
```

마진 1%p 개선 = 영업이익 +14%. **수주 단가 인상 + 부설선 운임 + 캐파 가동률**이 마진 개선의 3대 동력.

### CASE 1. SuedLink (독일 남북 연결)

- **공급사**: **Prysmian (남부 구간) + NKT (북부 구간)** 분담.
- **금액**: 케이블만 €1.5B. 변환소·시공 합산 €10B+ 추정.
- **일정**: 2017 발주 → 2024-26 매설 → **2028년 가동**.
- **시사점**: 한 프로젝트가 케이블사 매출 **2-3년치**. 연속된 메가 프로젝트가 잔고를 €10B 넘게 채움.

## 17.7 자주 받는 질문(FAQ) — 한국 개인 투자자용

#### Q1. 해저케이블이랑 변압기·GIS 중 뭐가 더 좋은가?

A. **둘 다 슈퍼사이클**이지만 차이가 있다.
- **변압기**: 미국 그리드 갱신 + 데이터센터 = 폭증. HD현대일렉·효성중·LS ELECTRIC.
- **해저케이블**: 풍력 + AI 통신 + 인터커넥션 = 폭증. LS·대한전선·LS마린솔루션.
- **잔고 가시성**: 해저케이블이 더 길다 (3-5년치 vs 변압기 2-3년치).
- **결론**: 양쪽 분산. 변압기 60% + 해저 40% 정도가 균형.

#### Q2. 비상장 LS전선에 어떻게 노출하나?

A. 두 가지 방법.
- ① **LS Corp (006260)** = 모회사. LS전선 92.3% 보유. **간접 노출**.
- ② **LS전선 IPO** 분리 시 직접. 2027-28 가능성 거론되나 확정 없음.
- ③ LS전선아시아(229640)는 **해저 비중 작음** = 해저 우회로는 부적합.

#### Q3. 미국 트럼프 풍력 동결 = 한국 LS·대한전선에 좋은가 나쁜가?

A. **단기 나쁨, 중장기 중립**.
- 단기: 미국 5개 풍력 중 일부 취소 → LS·대한전선 잠재 매출 감소.
- 중장기: ① LS는 Chesterfield 공장 = **미국 내 생산 = 보편관세 회피**. ② 대한전선은 미국 노출 작음.
- 전체 글로벌 시장(EU·아·중동)이 더 크니 영향 제한적.

#### Q4. 통신 해저케이블에 한국 노출 종목 진짜 없나?

A. 거의 없다.
- LS전선이 광섬유·통신 케이블도 만들지만 매출 비중 < 5%.
- SubCom·ASN·NEC·HMN의 4강 과점이 워낙 강함.
- 우회 옵션: **NEC (6701.T) + 노키아 (NOK.HE) + Ciena (CIEN, 미국)**.

#### Q5. 부설선(CLV) 투자 = LS마린솔루션 한 가지인가?

A. 거의 그렇다.
- LS마린솔루션이 한국 유일.
- 해외: Boskalis (네덜란드, 비상장) + Subsea7 (SUBC, 노르웨이 상장) + Saipem (SPM.IT, 이탈리아).
- 해외 부설 전문은 **풍력 EPC 통합**으로 가는 중. 케이블 단독 부설사는 점차 통합됨.

#### Q6. ETF로 해저케이블 노출 가능한가?

A. **전용 ETF 없음**.
- iShares Global Clean Energy (ICLN) = 풍력 ETF. 케이블 비중 작음.
- First Trust NextG (NXTG) = 통신 인프라. 해저 노출 약함.
- **개별 종목 바스켓** 직접 구성이 가장 효율적.

#### Q7. 사이버보안 사건 = 케이블 회사에 좋은가 나쁜가?

A. **단기 변동성, 중장기 긍정**.
- 단기: 절단 사건 시 보험금·복구 비용으로 회사 단기 손실.
- 중장기: 케이블 다중화 + 보호 강화 = **신규 발주 증가**. 케이블 보호선 신시장 형성.
- 결론: 케이블 회사에는 **순증** 효과.

#### Q8. 구리 가격 상승 = 케이블 회사에 좋은가?

A. **단기 부정, 중장기 중립**.
- 단기: 매출원가 상승 → 마진 압축. 그러나 대부분 회사가 **금속 가격 슬라이드 조항**으로 헤지.
- 중장기: 단가 인상 패스스루 → 매출액 상승 (마진은 동일).
- 결론: 구리 가격은 **매출에 +**, **마진에는 중립**.
