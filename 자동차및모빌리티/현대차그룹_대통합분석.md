# 🚗 현대자동차그룹 (HMG) 대통합 심층 분석
**작성일: 2026-04-24 | Agent Team 취합본**

> **대상**: 현대차그룹 = **"Mobility + Robotics + Hydrogen" 3축 테크 복합체**. 완성차 세계 3위 + Boston Dynamics 휴머노이드 + Supernal UAM + HTWO 수소 + SMR 협력 + Genesis 럭셔리. 지금까지 쓴 **6~7개 리포트 (2차전지·휴머노이드·방산·SMR·AI인프라·밸류업)의 공통 접점**.
