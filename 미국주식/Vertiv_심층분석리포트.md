## 2. AI 데이터센터 인프라가 뭔지부터

### 2.1 데이터센터란?

**데이터센터** = 수천~수만 대의 컴퓨터(서버)가 모여 데이터를 처리·저장하는 대형 시설.

#### 일반 데이터센터 vs AI 데이터센터

| 항목 | 일반 데이터센터 | AI 데이터센터 |
|---|---|---|
| 전력 | 10~50 MW | **100 MW ~ 1 GW** ⭐ |
| 서버 | CPU 중심 | **GPU 중심 (NVIDIA H100·B200·Rubin)** |
| 발열 | 1.5 kW/서버 | **30~50 kW/랙 (10~30배)** |
| 냉각 | 공기 (Air) | **액체 (Liquid)** 필수 |
| 인프라 비용 | 일반 | **2~3배** |

비유:
- 일반 데이터센터 = **동네 사무실 빌딩**
- AI 데이터센터 = **공장 + 발전소 + 화학 플랜트** 종합

## 4. 액체 냉각 (Liquid Cooling) — AI GPU 시대의 핵심

### 4.1 왜 액체 냉각이 필수인가?

#### 공기 냉각의 한계
- 일반 서버: 1~2 kW/서버 → 공기 냉각 OK
- NVIDIA H100: 700W/GPU → 공기 한계
- **NVIDIA B200/GB200: 1,200W/GPU → 공기 불가능**
- NVIDIA Rubin: 1,500W+/GPU → 액체 냉각 필수

#### 액체 냉각의 원리
**물(또는 특수 액체)이 GPU에 직접 흘러 열을 빼앗아 가는 방식**.

비유:
- 공기 냉각 = **선풍기로 뜨거운 라면 식히기** (느림, 한계)
- **액체 냉각 = 얼음물에 라면 그릇 담그기** (빠름, 효율적)

### 4.2 액체 냉각의 3가지 방식

#### 방식 1: 직접 칩 냉각 (Direct-to-Chip)
- 차가운 물이 콜드 플레이트(Cold Plate) 통해 GPU에 직접 접촉
- 효율 80%+
- NVIDIA Rubin 표준 방식

#### 방식 2: 침지 냉각 (Immersion)
- 서버 전체를 특수 액체(절연유)에 담금
- 효율 90%+
- 가장 진보적 (다만 인프라 변경 필요)

#### 방식 3: 후방 열교환 (Rear Door)
- 서버 랙 뒤에 열교환기 설치
- 효율 50%
- 기존 데이터센터 보완용

### 4.3 액체 냉각 시장 폭발

| 연도 | 액체 냉각 시장 |
|---|---|
| 2023 | $1B |
| 2024 | $3B |
| 2025 | $7B |
| **2026E** | **$12B** |
| 2028E | $25B |
| 2030E | $40B |

→ **5년간 13배 성장 시장**. Vertiv가 30% 점유 시 매출 +$3.6B/년.

### 4.4 Vertiv의 액체 냉각 라인업

- **Liebert XDU 시리즈**: Direct-to-Chip 액체 냉각
- **Vertiv CoolPhase**: 침지 냉각
- **Liquid-cooled CDU**: 냉각 분배 장치
- **NVIDIA·CSP 공동 개발**: NVIDIA MGX 레퍼런스 아키텍처 채택

### 4.5 NVIDIA Rubin·Rubin Ultra 액체 냉각

NVIDIA가 2026~2027 출시할 **Rubin·Rubin Ultra GPU**는:
- 각 GPU 1,500W~2,000W 발열
- 랙당 **1 MW 전력 + 액체 냉각 필수**
- Vertiv가 NVIDIA 공식 파트너로 채택

→ **NVIDIA Rubin 양산 시 Vertiv 매출 직접 폭발**.
