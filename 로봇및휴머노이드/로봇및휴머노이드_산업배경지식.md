## 1️⃣ 로봇 산업 큰 그림 — 4대 분류

### 4가지 로봇 종류

| 분류 | 특징 | 주 용도 | 대표 기업 |
|---|---|---|---|
| **산업용 로봇** (Industrial) | 안전 펜스 안, 고정형, 무거움, 정밀 | 자동차·전자 조립 | FANUC, ABB, KUKA, Yaskawa |
| **협동로봇** (Cobot) | 사람과 같은 공간, 가볍고 안전 | 중소제조·검사 | Universal Robots, **두산로보틱스**, **레인보우로보틱스** RB |
| **서비스로봇** | 호텔·식당·청소·배송 | 상업·홈 | Bear Robotics (LG), Knightscope, iRobot |
| **휴머노이드** (Humanoid) | **2족 보행, 인간형, AI 기반** | 공장·물류·가정 | **Tesla Optimus, Figure, Boston Dynamics, Unitree** |

### 산업용 vs 협동로봇 차이
- **산업용**: 빠르고 강력하나 **사람과 함께 일할 수 없음** (안전 펜스 필수)
- **협동**: 토크·속도 제한 + 충돌 감지 → 펜스 불필요

### 휴머노이드가 왜 다른가?
- 기존 로봇은 **고정된 작업** (한 자리에서 같은 동작 반복)
- 휴머노이드는 **인간 환경에 적응** — 계단·문·도구를 인간처럼 사용
- **목표**: 공장 인력·물류 인력·가사 노동 대체

### 시장 규모 비교 (2024)
| 시장 | 규모 | CAGR |
|---|---|---|
| 산업용 로봇 | ~$25B | 10% |
| 협동로봇 | ~$2B | 25% |
| 서비스로봇 | ~$50B (광범위) | 15% |
| **휴머노이드** | **~$5B** | **25~35%** (Goldman: 2035 $38B / Morgan: 2050 $5T) |

## 2️⃣ 휴머노이드란 무엇인가

### 정의
- **인간형 (Human-shaped) 2족 보행 로봇**
- 머리·몸통·양팔·양다리 + AI 두뇌
- 키 1.5~1.8m, 무게 50~100kg

### 왜 "인간 모양"인가?
1. **인간 환경 그대로 사용** — 계단·문 손잡이·도구 별도 개조 불필요
2. **범용성** — 한 로봇이 여러 작업 (vs 산업용은 전용)
3. **인간 노동 대체** — 인구 감소·인건비 상승 대응

### 휴머노이드 첫 등장 / 주요 마일스톤
- **1986** Honda P1 (최초 양산 휴머노이드 시도)
- **2000** Honda **ASIMO** — 세계 최초 본격 휴머노이드
- **2013** Boston Dynamics **Atlas** (DARPA Robotics Challenge)
- **2022** Tesla **Optimus** 프로토타입 공개
- **2024** Boston Dynamics 전기 Atlas, Figure 02
- **2025** Figure BMW 양산 라인 투입, Tesla 사내 시범
- **2026** Boston Dynamics 양산 출하, Tesla Fremont 생산 개시 (목표)

### 산업·서비스·가정용 구분
| 용도 | 요구 사항 | 대표 |
|---|---|---|
| **산업용** | 정밀·반복·내구성 | Figure (BMW), Optimus (Tesla 공장) |
| **물류·창고** | 박스 들기·이동 | Agility Digit (GXO·Amazon) |
| **서비스** | 안내·청소 | Unitree, XPeng IRON |
| **가정용** | 안전·다정함 | 1X NEO, Tesla (장기) |

## 3️⃣ 로봇의 5대 구성 요소

```
[AI 두뇌 (Compute)]
       ↓
[센서 (Sensors)]  ← 환경 인식
       ↓
[제어 (Control)]
       ↓
[액추에이터 (Actuators)]  ← 모터 + 감속기
       ↓
[기구·구조 (Mechanics)]
       ↓
[배터리·전력]
```

### 5대 요소
| 구성 | 역할 | 비유 |
|---|---|---|
| **AI 두뇌** | 인지·판단·계획 | 뇌 |
| **센서** | 환경 인식 | 눈·귀·피부 |
| **액추에이터** | 움직임 생성 | 근육 |
| **감속기·기구** | 토크 증폭·구조 | 관절·뼈대 |
| **배터리·BMS** | 에너지 공급 | 심장 |

<a id="4"></a>
## 4️⃣ 액추에이터 — 로봇의 근육

### 액추에이터란?
- **전기 신호를 움직임으로 바꾸는 부품**
- 휴머노이드는 보통 **30~60개** 액추에이터 (관절마다 1개)
- **로봇 원가의 약 60%**를 차지

### 종류
| 종류 | 원리 | 특징 |
|---|---|---|
| **BLDC 모터** (브러시리스 DC) | 전자기 회전 | 가볍고 효율적, **휴머노이드 주류** |
| 서보 모터 | 위치·속도 정밀 제어 | 산업용 표준 |
| 유압 (Hydraulic) | 기름 압력 | 강력하나 무겁고 느림 (구 Atlas) |
| 공압 (Pneumatic) | 공기 압력 | 부드럽지만 정밀도 낮음 |
| **인공 근육** (HASEL 등) | 전기·열 자극 | 미래 (실험 단계) |

### Boston Dynamics Atlas의 변화
- **구 Atlas (2013~2024)**: 유압식 — 강력하지만 무겁고 시끄러움
- **신 Atlas (2024~)**: **전기식 BLDC** — 가볍고 조용·정밀

### 핵심 지표: **토크 밀도** (Torque Density, Nm/kg)
- 같은 무게에서 얼마나 큰 힘을 낼 수 있는가
- Boston Dynamics Atlas: **220 Nm/kg** (직구동)
- 일반 산업용: 50~100 Nm/kg
- 토크 밀도가 높을수록 **로봇 가벼움 + 페이로드 ↑**

### 휴머노이드 액추에이터 종류
- **Rotary (회전형)**: 어깨·팔꿈치
- **Linear (직선형)**: 무릎·발목 (실린더 형태)
- **Direct-drive**: 감속기 없이 모터 직접 (Atlas)

## 5️⃣ 감속기 — 모터의 힘을 키우는 기어

### 왜 감속기가 필요한가?
- 모터는 **빠르지만 힘이 약함**
- 감속기로 **속도 ↓ 토크 ↑** (10:1, 50:1, 100:1)
- 비유: 자전거 변속기 (무거울 때 낮은 기어 = 감속비 높음)

### 종류
| 종류 | 특징 | 용도 |
|---|---|---|
| **Harmonic** (하모닉, 조파) | 정밀, 백래시 적음, 비쌈 | **휴머노이드·산업용 정밀** |
| Cycloidal (사이클로이달) | 강력, 충격 흡수 | 대형 산업용 |
| Planetary (유성기어) | 단순, 저렴 | 협동로봇 |
| **RV** (Rotary Vector) | 중정밀 | 산업용 산업 |

### 글로벌 감속기 시장
| 기업 | 국가 | 강점 |
|---|---|---|
| **Harmonic Drive Systems** | 日 | **하모닉 감속기 글로벌 1위 (~70%)** |
| Nabtesco | 日 | RV 감속기 1위 |
| Sumitomo | 日 | Cycloidal |
| **SBB Tech** | 韓 | 하모닉 국산화 |
| **에스피지 (058610)** | 韓 | 정밀 감속기, **레인보우로보틱스 100% 채택** |
| Spinea | 슬로바키아 | TwinSpin |

### 휴머노이드 1대당 감속기
- **30~60개** (관절 수와 같음)
- 단가 $300~1,000/개
- → **휴머노이드 1대당 감속기만 $10K~30K**

### 한국 감속기 국산화 의의
- 일본 Harmonic Drive 의존도 70%+
- **현대차 Atlas에 SBB·에스피지 채택** (2024~)
- 가격 일본 대비 **30~50% 저렴**
- **2026 양산 본격화** = K-감속기 도약 원년

## 7️⃣ AI 모델 — 로봇의 뇌

### 전통 로봇 vs 현대 휴머노이드
- **전통**: "if-else" 규칙 + 산업용 PLC → 정해진 작업만
- **현대**: **VLA (Vision-Language-Action) 파운데이션 모델** → 범용 학습

### VLA란?
- **카메라 영상 (Vision) + 언어 명령 (Language) → 행동 (Action)** 생성
- 예: "테이블 위 빨간 컵 가져와"라고 말하면 → 카메라로 컵 찾고 → 팔 움직임 계획 → 잡고 → 가져옴
- 기존 산업 로봇은 좌표 입력 필수, VLA는 **자연어 OK**

### 주요 VLA 파운데이션 모델
| 모델 | 개발사 | 특징 |
|---|---|---|
| **GR00T N1 / N2** | **NVIDIA** | 휴머노이드 범용 (Figure·1X·Agility 사용) |
| **Helix** | Figure AI | Figure 02 전용, 빠른 추론 |
| **π0 / π0.6** | Physical Intelligence | $14B 밸류, OpenAI 출신 |
| **Gemini Robotics** | Google DeepMind | Gemini + 로봇 |
| **Skild Brain** | Skild AI | $14B 밸류, **omni-bodied** (다양한 로봇) |
| **Cosmos** | NVIDIA | 시뮬레이션·합성 데이터 |

### 학습 방법
1. **Imitation Learning**: 인간 데모 영상 → 로봇 따라하기
2. **Reinforcement Learning (RL)**: 시뮬레이션·실제에서 시행착오
3. **Synthetic Data**: NVIDIA Cosmos·Omniverse 디지털 트윈

### 로봇용 GPU
- **NVIDIA Jetson Thor** (2025) — Blackwell 기반, **800 TFLOPS FP8**
- 휴머노이드 1대 = Jetson Thor 1~2개
- Qualcomm Robotics RB6도 일부

### Test-Time Computing
- 추론 시 **여러 시뮬레이션 → 최적 행동 선택**
- AI 칩 수요 폭증 동인 (NVIDIA Rubin·Vera 수혜)

## 8️⃣ 배터리·열관리·전력

### 배터리 요구사항
| 지표 | 목표 |
|---|---|
| 작동 시간 | 8~12시간 (1일 노동) |
| 무게 | 휴머노이드 50kg 중 **5~10kg** |
| 충전 시간 | 1~2시간 (급속) or 자동 도킹 |
| 안전성 | 화재 위험 최소 |
| 용량 | 1~3 kWh |

### 전고체 배터리의 매력
- **에너지 밀도 2배** (Li-ion 200 Wh/kg → 400+)
- **화재 위험 최소**
- **삼성SDI** 2027 휴머노이드용 파우치형 전고체 첫 공개 (InterBattery 2026)
- **2032 휴머노이드 배터리 시장 91조** (인사이트코리아)

### 열관리
- 액추에이터 ×30개가 **각각 발열**
- 산업 로봇은 정지 가능, 휴머노이드는 연속 동작
- **방열판 + 냉각 팬** + 일부 액침 냉각 검토
- 2026 현재 **30분 연속 작동 시 발열 한계** = 양산 제약

### 충전 인프라
- 자동 도킹 (가정·창고)
- 핸드 페이로드 → 전력선 직접 연결 (산업 라인)

## 9️⃣ 자유도 (DoF) 개념

### DoF (Degrees of Freedom)
- **독립적으로 움직일 수 있는 관절 수**
- 사람: **244 DoF** (손가락 포함)
- 휴머노이드 (전체): **30~60 DoF**
- 사람 손 한쪽: **27 DoF** (휴머노이드 손은 11~22)

### 주요 휴머노이드 DoF 비교

| 로봇 | 전체 DoF | 손 (양손) | 발 |
|---|---|---|---|
| **Tesla Optimus Gen 3** | ~52 | 22 (×2) | - |
| **Figure 03** | ~36 | 16 (×2) | - |
| **Boston Dynamics Atlas (전기)** | **56** | 16 (×2) | 6 (×2) |
| **1X NEO Gamma** | ~40 | 16 (×2) | - |
| **XPeng IRON** | 60+ | **22 (×2)** | - |
| **Unitree G1** | ~23 | 7 (×2, 단순) | - |

### DoF가 많을수록
- 더 다양한 동작 가능
- but 액추에이터·감속기·센서·제어 복잡도 ↑
- 가격 ↑, 무게 ↑

### Hand Dexterity (손 기민성)
- 인간 손은 27 DoF + 풍부한 촉각 → 옷 정리·요리·세밀 작업
- 현재 휴머노이드: **22 DoF가 최고**
- 촉각 센서·정밀 액추에이터 한계
- 2027~2028 인간 수준 도달 목표

## 🔟 휴머노이드의 특수 난제 4가지

### 난제 1: **이족 보행 (Bipedal Locomotion)**
- 사람도 1~2년 학습 → 인간이 가장 늦게 익히는 기술
- 무게중심 (Center of Gravity) 제어
- ZMP (Zero Moment Point) 기반 제어 → 최근 RL 학습 기반
- 계단·경사·미끄러짐 대응 어려움

### 난제 2: **손 기민성 (Hand Dexterity)**
- 옷 단추·신발끈·요리 등 복잡 작업
- 27 DoF + 촉각 센서 모방 어려움
- **Apptronik Apollo, Tesla Gen 3 손**이 진전

### 난제 3: **배터리·열관리**
- 30 모터가 동시 작동 시 발열 폭발
- 8시간 연속 작동 = 최소 1.5 kWh 필요
- 무게 vs 작동시간 trade-off

### 난제 4: **AI 일반화 (Generalization)**
- 학습 안 한 환경·물체 적응
- 시뮬레이션 → 실제 (sim-to-real gap)
- 모델 크기 vs 추론 속도 trade-off
- BMW 파일럿에서 Figure 02 **팔뚝부 최대 고장 포인트** = 신뢰성 한계
