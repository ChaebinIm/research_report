## 1️⃣ 전력반도체란 무엇인가

### 정의
- **전기를 제어하는 반도체** — On/Off 스위칭, 변환, 증폭
- 일반 반도체 (CPU·메모리)는 **정보 처리**, 전력반도체는 **전기 처리**

### 비유로 이해하기
- 일반 반도체 = **뇌**
- 전력반도체 = **심장** (전기의 흐름·강약·방향 조절)

### 어디에 들어가나?
- **EV 배터리 → 모터** 변환 (인버터)
- **AI GPU 보드** 전원 (12V → 0.8V 변환)
- **태양광 패널 → 그리드** 변환 (인버터)
- **고속 충전기·어댑터·가전**
- **산업용 모터·로봇**

### 시장 규모
- 글로벌 전력반도체: **2024 ~$50B → 2030 ~$80B+** (CAGR 8~10%)
- 그중 **SiC: $2B → $10B (CAGR 30%+)**, GaN: $0.4B → $4B+

<a id="2"></a>
## 2️⃣ 일반 반도체 vs 전력반도체

| 축 | 일반 반도체 (Logic·Memory) | **전력반도체** |
|---|---|---|
| 역할 | 정보 처리·저장 | **전기 제어·변환** |
| 동작 전압 | 1V 이하 | **수십 V ~ 수천 V** |
| 동작 전류 | μA~mA | **A~kA** |
| 미세공정 | 2nm·3nm 경쟁 | **150~350nm** (오히려 큰 게 유리) |
| 핵심 지표 | 클럭 속도, TFLOPS | **R_DS(on), 스위칭 속도** |
| 대표 회사 | NVIDIA, TSMC, Samsung | Infineon, ST, onsemi, ROHM |

### 왜 전력반도체는 미세공정이 필요 없나?
- 정보가 아닌 **큰 전류·고전압을 다룸**
- 미세화하면 오히려 **발열·파손**
- 대신 **소재(SiC·GaN) + 구조** 혁신이 핵심

<a id="3"></a>
## 3️⃣ 핵심 기능 — 스위칭과 변환

### 4대 기본 기능
1. **스위칭 (Switching)** — On/Off 빠르게 (kHz~MHz)
2. **정류 (Rectification)** — AC → DC
3. **승압·강압 (Step-up/down)** — 전압 변환
4. **인버팅 (Inverting)** — DC → AC

### 비유: 댐의 수문
- **스위칭** = 수문 열고 닫기 (속도 = 효율)
- **정류** = 양방향 흐름 → 한 방향
- **승압/강압** = 수압 조절
- **인버팅** = 흐름 방향 바꿈

### 효율의 의미
- 95% 효율 → 5% 손실 = **열로 발산**
- 99% 효율 → 1% 손실
- AI DC 1MW 랙: 95% vs 99%면 **연 350MWh 차이** (수억원)
- → **"몇 % 효율 개선"이 수십조 시장 견인**

## 4️⃣ 전력반도체 소자 종류

### 주요 소자 5종

| 소자 | 풀네임 | 동작 | 용도 |
|---|---|---|---|
| **다이오드 (Diode)** | - | 한쪽 방향만 통과 | 정류·역류 방지 |
| **MOSFET** | Metal-Oxide-Semi FET | 빠른 스위칭, 저전압 | 0~600V |
| **IGBT** | Insulated Gate Bipolar Transistor | 고전류, 중속 | 600~6,500V (EV 인버터, 산업) |
| **Thyristor·SCR** | - | 큰 전력 | 고전압 송전 |
| **JFET·HEMT** | High Electron Mobility Transistor | 초고속 | GaN 채택 |

### 주요 소자 매트릭스

| 소자 | 동작 전압 | 스위칭 속도 | 전류 | 주 용도 |
|---|---|---|---|---|
| Si MOSFET | 0~650V | 빠름 | 작음 | 충전기·가전 |
| **Si IGBT** | 600~6,500V | 중간 | **큼** | **EV 인버터·산업** |
| **SiC MOSFET** | 600~1,700V | **빠름** | 큼 | **EV·DC·재생E** |
| **GaN HEMT** | 100~650V | **초고속** | 작음 | **충전기·AI DC·서버** |
| Si Diode | - | - | - | 정류 |

### IGBT vs SiC MOSFET vs GaN HEMT (가장 중요)

| 축 | Si IGBT | **SiC MOSFET** | **GaN HEMT** |
|---|---|---|---|
| **전압대** | 600V~6.5kV | 600V~1.7kV | 100V~650V |
| **스위칭** | ~20kHz | ~100kHz | **~1MHz** (10배+) |
| **효율** | ~95% | **97~98%** | **>99%** |
| **크기** | 큼 | 1/3 | 1/10 |
| **가격** | 저렴 | Si의 3~5x | Si의 5x+ |
| 주 용도 | EV·산업 | EV·DC·서버 | 충전기·AI DC |

<a id="5"></a>
## 5️⃣ 3대 소재 — Si vs SiC vs GaN

### Si (실리콘) — 60년+ 표준
- 전 세계 반도체 95%+
- 가공 쉬움, 저렴
- **고전압·고속에서 한계** → IGBT로 우회했지만 효율 한계

### SiC (탄화규소) — EV 슈퍼사이클
- **Wide Bandgap** (전기적 우수성)
- 200°C 이상 작동, 손실 1/3
- **Tesla Model 3 (2018)** 부터 채택 → **EV 인버터 표준화**
- 단점: **6인치 웨이퍼 → 8인치 전환 중**, 비쌈
- 주요 공급사: STM, Infineon, onsemi, ROHM, **Wolfspeed**

### GaN (질화갈륨) — 고속·고효율
- SiC보다 **스위칭 속도 10배 빠름**
- 100~650V 영역에서 압도적
- **휴대폰 충전기·노트북 어댑터** 시작 (2019~)
- **AI DC 800V HVDC**의 핵심 (NVIDIA 2025.5 채택)
- 주요 공급사: Infineon (GaN Systems 2023 인수), **Navitas**, **Innoscience**, **Power Integrations**, EPC

### Ga₂O₃ (산화갈륨) — 차차세대
- 더 큰 Bandgap (4.8 eV vs SiC 3.3)
- 2030+ 상용 예상
- 일본 Novel Crystal·미국 NG Wafer

### 소재 비교 표

| 축 | Si | SiC | GaN | Ga₂O₃ |
|---|---|---|---|---|
| Bandgap (eV) | 1.12 | 3.26 | 3.39 | **4.8** |
| 항복 전압 | 표준 | **10x** | 10x | 50x |
| 작동 온도 | 150°C | **200°C+** | 200°C+ | 300°C+ |
| 스위칭 속도 | 표준 | 5x | **10x** | 10x |
| 비용 | 1x | 3~5x | 5x | 50x (현재) |
| 양산 단계 | 성숙 | **램프 중** | **상용 시작** | 연구 |

<a id="6"></a>
## 6️⃣ 전력 변환 4대 회로

### 1. AC-DC (정류기, Rectifier)
- 콘센트 220V AC → 노트북 19V DC
- 사용: 가전 어댑터, 데이터센터 입력

### 2. DC-DC (Buck/Boost Converter)
- 12V → 5V (스마트폰)
- 800V → 12V → 0.8V (AI GPU 보드)
- 사용: AI 서버, 차량 ECU

### 3. DC-AC (인버터, Inverter)
- 배터리 DC → 모터 AC
- 사용: **EV 인버터 (가장 큰 시장)**, 태양광 인버터

### 4. AC-AC (사이클로컨버터)
- 60Hz → 50Hz 등 주파수 변환
- 사용: 산업용

### EV 1대당 변환 회로
- **Main Inverter** (배터리 → 모터): SiC MOSFET 6개
- **OBC** (On-Board Charger, AC → 배터리): SiC + GaN
- **DC-DC Converter** (HV 배터리 → 12V): Si/SiC
- **충전 인프라** (DCFC): SiC

→ EV 1대당 전력반도체 BOM **$700~1,200** (Si 차량 $200 대비 4~6배)

## 8️⃣ AI DC와 800V HVDC 혁명

### 왜 800V로 가는가?
- AI GPU 1대 = **1MW** (Rubin Ultra Kyber 랙)
- 기존 415V AC 배전 → 케이블 굵고 손실 큼
- **800V DC**로 올리면:
  - 케이블 1/2 굵기
  - 손실 -45%
  - 동(銅) 사용 -70%

### NVIDIA 2025.5 발표
- **Vera Rubin Ultra (2027 Kyber 랙)** 부터 800V HVDC
- Microsoft·Google·Meta 표준 채택 검토

### 800V 변환 단계
```
[발전소 송전 (수만V)] → [DC 변환소] → [800V DC] → [GPU 보드 12V] → [GPU 0.8V]
                                   ↑                ↑
                              GaN/SiC 핵심       GaN 다단 변환
```

### 수혜 기업
- **Navitas** (NVIDIA 800V GaN 핵심 공급) — 별도 리포트 참조
- Infineon (SiC + GaN 종합)
- onsemi (SiC)
- **MPS** (DC-DC 컨버터)

## 🔟 공정·제조 특징

### Wafer 사이즈 차이
| 분야 | Wafer | 비고 |
|---|---|---|
| 일반 반도체 | **300mm (12")** | 표준 |
| Si 전력반도체 | 200mm (8") | 양산 |
| **SiC** | **150mm (6") → 200mm (8")** | **2024~2025 8인치 전환 중** |
| **GaN-on-Si** | 200mm | Si 위에 GaN 성장 |
| GaN-on-SiC | 150mm | 고가 |

### SiC 8인치 전환의 의미
- 6인치 → 8인치 = **칩 수 2배+**
- 원가 30% 절감 → SiC 대중화 가속
- ST·Infineon·onsemi 모두 8인치 라인 증설 (2024~2026)

### Wolfspeed의 도박
- 세계 최초 **8인치 SiC 전용 팹** (모홀리·뉴욕)
- $5B+ 투자 → 2024 재무 위기 → 2025 Chapter 11 우려
- **8인치 전환의 high-stakes player**

### Fab 모델
- **IDM** (Integrated Device Manufacturer): Infineon·ST·onsemi·ROHM — 자체 제조
- **Fabless**: Navitas, EPC — TSMC·VIS 위탁
- **Foundry**: TSMC·VIS·SMIC가 SiC·GaN 생산
