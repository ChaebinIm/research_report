## 1️⃣ 반도체 제조 큰 그림 — 전공정 vs 후공정

### 두 단계로 나눔
```
[모래 (실리콘)] → [웨이퍼 (Wafer)] 
                       ↓
                 [전공정 (FEOL/BEOL)] ← 수백 개 공정 반복, 2~3개월
                       ↓
                 [웨이퍼 완성]
                       ↓
                 [후공정 (Packaging·Test)] ← OSAT
                       ↓
                 [칩 (완제품)]
```

### 전공정 (Front-End of Line, FEOL + BEOL)
- **웨이퍼 위에 회로를 그리고 쌓는 단계**
- TSMC·Samsung·Intel·SK하이닉스의 **팹(Fab)**에서 진행
- **8대 공정**: 노광·증착·식각·계측·세정·CMP·이온주입·확산
- 장비 시장의 **~87%**

### 후공정 (Back-End / OSAT)
- **완성된 칩을 자르고 패키징하고 테스트**
- ASE·Amkor·한미반도체 등
- 장비 시장의 **~13%** (AI·HBM 시대 급성장 중)

## 2️⃣ 웨이퍼 (Wafer) — 반도체의 도화지

### 웨이퍼란?
- **실리콘 원판**. 반도체 회로를 그리는 바탕
- **300mm (12인치)** 표준. 450mm는 경제성 부족으로 실패
- SiC·GaN 웨이퍼는 150~200mm

### 제조 공정
1. **모래 → 실리콘 정제** (순도 99.9999999% = 9N)
2. **단결정 성장** (Czochralski, Float Zone)
3. **잉곳(Ingot) 절단** → 얇은 원판
4. **연마** (CMP)
5. **에피택시** (고급 웨이퍼)

### 글로벌 Top 5 (~85% 점유)
| 기업 | 점유율 | 국가 |
|---|---|---|
| **신에츠화학** (4063.T) | ~18% | 日 |
| **SUMCO** (3436.T) | ~17% | 日 |
| **GlobalWafers** (6488.TW) | ~15% | 대만 |
| **Siltronic** (WAF.DE) | ~12% | 독일 |
| **SK실트론** (비상장) | ~12% | 한국 |

## 3️⃣ 8대 전공정 심화

```
[산화(Oxidation)] → [포토레지스트 도포] → [노광 (Litho)] → [현상] → [식각 (Etch)]
       ↓                                                                ↓
[이온 주입] ← [세정] ← [CMP] ← [증착 (Deposition)] ← [확산 (Diffusion)]
```

**한 층 만드는 데만 이 과정 반복**. 최첨단 반도체는 **1,000회 이상** 반복.

### 공정별 장비 비중 (WFE 내)
| 공정 | 비중 | 1위 | 특징 |
|---|---|---|---|
| **리소그래피** (노광) | **~22%** | **ASML** (>90%) | 가장 비싼 장비 |
| **증착** | ~25% | AMAT | CVD·PVD·ALD·Epi |
| **식각** | ~22% | **LAM** (~45%) | 깎아내기 |
| **계측·검사** | ~13% | **KLA** (~55%) | 품질 관리 |
| CMP (평탄화) | ~4% | AMAT / Ebara | 울퉁불퉁 표면 깎기 |
| 세정 | ~7% | **Screen** (~50%) | 오염 제거 |
| 이온주입·기타 | ~7% | AMAT / Axcelis | 도핑 |

## 4️⃣ 노광 (Lithography) — 반도체 핵심

### 노광이란?
- **빛으로 회로 패턴을 웨이퍼에 전사**
- 스텐실처럼 마스크(레티클)를 통해 빛을 비춤
- **파장이 짧을수록 더 작은 패턴** 가능

### 광원 세대
| 세대 | 파장 | 시대 | 대표 장비 |
|---|---|---|---|
| g-line | 436 nm | 1980s | - |
| i-line | 365 nm | 1990s | - |
| KrF | 248 nm | 2000s | - |
| **ArF** | 193 nm | 2010s (7nm까지) | ASML NXT |
| **ArFi (immersion)** | 193 nm + 물 | 7nm까지 | 멀티패터닝 |
| **EUV** | **13.5 nm** | 7nm~ (선단) | ASML NXE/EXE |

### EUV = 극자외선
- **13.5 nm 파장**으로 5nm 이하 공정 필수
- 한 대 **$150~200M** (2,000억+)
- **ASML이 세계 유일 제조** — Zeiss 미러 + Trumpf 광원

### High-NA EUV (신형)
- NA 0.33 → **0.55**로 확대
- 2nm 이하에 필수
- 한 대 **$350~400M**
- Intel이 첫 고객 (2023), SK하이닉스 2025.9 DRAM 설치, Samsung 2025 말~2026 상반기
- 2027까지 **10대+ 출하** 계획

### 이전 단계 — Coater/Developer (Track)
- 포토레지스트를 웨이퍼에 **코팅**하고, 노광 후 **현상**하는 장비
- **TEL이 시장 92% 독점**. ASML EUV 옆에 반드시 TEL 코터 붙음

### 마스크·레티클
- **블랭크 마스크 (Blank Mask)**: 가공 전 원판. Hoya (日) 독점, 한국 **에스앤에스텍** 도전
- **레티클**: 패턴 완성된 마스크. TSMC·Samsung이 내부 제작

<a id="5"></a>
## 5️⃣ 증착 (Deposition) — CVD·PVD·ALD·Epi

### 증착이란?
- **웨이퍼 위에 얇은 막(thin film)을 쌓는** 공정
- 금속·절연체·반도체 층 형성

### 주요 방식
| 방식 | 원리 | 용도 |
|---|---|---|
| **CVD** (화학 증착) | 가스 반응 | 절연층 (SiO₂·SiN) |
| **PVD** (물리 증착) | 플라즈마로 금속 튀김 | 금속 배선 (Cu·Al·W) |
| **ALD** (원자층 증착) | 분자 한 층씩 | **HBM·GAA·High-k 게이트** (선단 필수) |
| **Epi** (에피택시) | 결정 성장 | 채널 재료 (SiGe, GaN) |

### ALD가 AI 시대에 중요한 이유
- 3D 구조 (GAA 트랜지스터, HBM TSV)에서 **균일한 초박막** 필수
- 단일 원자 층 제어 가능
- **ASM International (ASMI)** 글로벌 55%+ 점유

### 장비 경쟁 구도
| 회사 | 강점 |
|---|---|
| **AMAT** | 증착 종합 1위, Epi 독보 |
| **LAM Research** | 금속 증착 |
| **TEL** | Coater + 증착 |
| **ASMI** | **ALD 독점급** |
| **원익IPS** (韓) | ALD·CVD, 삼성 주력 |
| **주성엔지니어링** (韓) | ALD·LPCVD |

<a id="6"></a>
## 6️⃣ 식각 (Etch) — 깎아내기

### 식각이란?
- **원하지 않는 부분을 깎아내는** 공정
- 노광으로 그린 패턴대로 정밀하게 제거

### 식각 종류
| 방식 | 원리 | 용도 |
|---|---|---|
| **Dry Etch** (플라즈마) | 가스 이온으로 깎음 | 선단 공정 주류 |
| Wet Etch | 화학 용액 | 세정 가까운 용도 |
| **Cryo Etch** | **극저온 (-100°C+)** 플라즈마 | **3D NAND 고단 식각 (필수)** |
| Atomic Layer Etch (ALE) | 분자 한 층씩 | 선단 3D |

### Cryo Etch의 부상
- 3D NAND 300단+ 시대에 필수
- **LAM Cryo 3.0** (2024) + **TEL 극저온 etcher** (2023)
- Samsung V10 (400단) NAND의 **first customer가 TEL** → LAM 독점 균열

### 식각 장비 TOP
| 회사 | 강점 |
|---|---|
| **LAM Research** | **Etch 글로벌 1위 (~45%)**, Vantex (3D NAND), SABRE (TSV 도금) |
| **TEL** | Cryo Etcher (삼성 V10 first customer) |
| AMAT | 메탈 식각 |
| **PSK** (韓) | Dry Stripper, Bevel Etch |

## 7️⃣ 계측·검사 (Metrology·Inspection)

### 계측 vs 검사
- **Metrology (계측)**: 치수·두께·조성 **측정**
- **Inspection (검사)**: 결함 **발견**

### 장비 유형
| 유형 | 원리 | 용도 |
|---|---|---|
| **광학 검사** | 가시광·UV | 패턴 결함 |
| **E-beam 검사** | 전자빔 | 고정밀 결함 (저속) |
| **AFM** | 원자힘 현미경 | **1nm 이하 해상도, 2nm 필수** |
| **CD-SEM** | 주사전자현미경 | 패턴 치수 |
| **Overlay / Reticle** | 정렬 측정 | 노광 정확도 |

### KLA 독점 구조
- **검사·계측 40%+ 점유**
- OP 마진 **41%+** (장비 업계 최고)
- AMAT·Hitachi High-Tech·Onto Innovation이 경쟁
- 한국: **파크시스템스** (140860) AFM 독보

## 8️⃣ 세정·CMP·이온주입

### 세정 (Cleaning)
- 공정 후 **파티클·오염물 제거**
- 웨이퍼 100번 공정이면 **200번 세정**
- **Screen Holdings (日)** 글로벌 1위 (~50%)
- 한국: **세메스** (삼성 내부), **케이씨텍**

### CMP (Chemical Mechanical Planarization)
- **화학 + 기계적 연마**로 표면 평탄화
- 증착 후 울퉁불퉁한 표면 필수 처리
- **AMAT + Ebara (日)** 양강
- 소모품: **슬러리 (Slurry)** — Cabot, Fujimi, SK엔펄스

### 이온주입 (Ion Implantation)
- 실리콘에 **불순물 (도펀트) 주입** → 전도성 조절
- 트랜지스터 만들 때 필수
- **AMAT, Axcelis**

<a id="9"></a>
## 9️⃣ 후공정 (Packaging) 심화

### 기본 패키징
- 웨이퍼 절단 → 다이(die) 분리 → 프레임에 올림 → 와이어 본딩 → 몰딩
- **ASE, Amkor, JCET** (OSAT)

### 고급 패키징 — AI 시대 핵심
| 기술 | 설명 | 사용 |
|---|---|---|
| **FCBGA** | Flip-Chip BGA | CPU·GPU·AI 칩 |
| **CoWoS** (TSMC) | Chip-on-Wafer-on-Substrate | **NVIDIA H100/B200/Rubin** |
| **CoWoS-L** (대형) | LSI interposer | Rubin 이후 |
| **SoIC** | 3D 칩 적층 | 차세대 |
| **HBM (High Bandwidth Memory)** | DRAM 3D 적층 + TSV | **AI 필수** |

### HBM 제조 핵심 공정
```
1. DRAM 다이 제조
2. TSV (Through-Silicon Via) 뚫기 ← LAM·AMAT·ASE
3. 다이 적층 (8-Hi → 12-Hi → 16-Hi)
4. TC 본딩 (Thermal Compression) ← 한미반도체 독점
5. 언더필
6. 테스트 ← Advantest·Teradyne
```

### TC 본더 (Thermal Compression Bonder)
- **HBM 제조의 심장**
- 다이 8~16개를 **초정밀 열·압력으로 접합**
- **한미반도체 (042700)** SK하이닉스 전용 독점 공급
- **한화세미텍** 삼성 2nd source 진입 중
- ASMPT, Besi, 신카와, TEL Synapse도 경쟁

### 하이브리드 본더 (차세대)
- **TSV 없이 직접 접합** (2026~)
- Besi + AMAT, TEL Synapse
- HBM4·SoIC 필수

## 1️⃣3️⃣ 글로벌 장비 빅5 한눈 보기

| 기업 | 본사 | FY25 매출 | OpM | 강점 |
