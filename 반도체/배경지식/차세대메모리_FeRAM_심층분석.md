# FeRAM/FRAM (강유전체 메모리) 단독 심층 분석
**작성일: 2026-05-11 | 대상 독자: 한국 개인 투자자 (문과·배경지식 0)**

> **목적**: 차세대 메모리 4대 후보(FeRAM·MRAM·PCM·ReRAM) 중 **가장 오래되었지만 가장 빠르고, 한 번 잊혀졌다가 HfO2 발견으로 부활한** FeRAM(에프이람·강유전체 메모리)을 단독 심층 분석. 임베디드 메모리(MCU 안에 박힌 작은 메모리) 시장에서 이미 상용화되어 있는 유일한 차세대 메모리이며, 차량용·산업용 MCU의 숨은 핵심.

## 약어·용어 박스 (먼저 읽어두면 본문이 쉬워진다)

| 약어 | 풀이 | 한 줄 설명 |
|---|---|---|
| **FeRAM** | Ferroelectric RAM | 강유전체(Ferroelectric) 분극(전기 방향)으로 0/1을 저장하는 메모리. **F-RAM·FRAM**도 같은 말 |
| **FRAM** | Ferroelectric RAM | Ramtron/TI/Fujitsu 상표명. FeRAM과 동일 개념 |
| **PZT** | Pb(Zr,Ti)O3 (Lead Zirconate Titanate, 납·지르코늄·티타늄 산화물) | 1세대 FeRAM 강유전체. 두껍고 첨단 공정 호환 안 됨 |
| **HfO2** | Hafnium dioxide (이산화하프늄) | 2011년 강유전성 발견. **얇고(<10nm) 첨단 노드 호환** → FeRAM 부활의 주역 |
| **FeFET** | Ferroelectric Field-Effect Transistor | 트랜지스터 게이트에 강유전체. 1트랜지스터로 메모리 (1T) |
| **1T1C** | 1 Transistor + 1 Capacitor | 한 비트당 트랜지스터 1개 + 강유전체 캐패시터 1개. DRAM과 같은 구조 |
| **1T** | 1 Transistor | 트랜지스터 1개로만 저장 (FeFET) |
| **분극** (Polarization) | 전기 쌍극자(+/−) 방향이 한쪽으로 정렬된 상태 | 위 vs 아래 두 방향 = 0 vs 1 |
| **도메인** (Domain) | 분극 방향이 같은 영역 | 한 캐패시터 안에 수많은 도메인이 한 방향으로 쏠림 |
| **잔류 분극** (Remnant Polarization, Pr) | 전압 끄고도 남는 분극량 | 단위 μC/cm². **클수록 신호↑, 미세화↑** |
| **항전계** (Coercive Field, Ec) | 분극을 뒤집는 데 필요한 최소 전압 | 낮을수록 저전력 |
| **파괴 읽기** (Destructive Read) | 읽는 순간 분극이 깨짐 → 다시 써야 함 | 1T1C FeRAM의 단점 |
| **비파괴 읽기** (Non-Destructive Read) | 읽어도 데이터 유지 | FeFET의 장점 |
| **Endurance** | 쓰기 가능 횟수 | FeRAM **10^14~10^15회** (사실상 무한) |
| **Retention** | 데이터 보존 기간 | FeRAM **10년 @85°C** |
| **MCU** | Microcontroller Unit | 마이크로컨트롤러. 전자제품 안의 작은 컴퓨터 |
| **NVM** | Non-Volatile Memory | 비휘발성 메모리. 전원 꺼도 데이터 유지 |
| **eNVM** | Embedded NVM | MCU·SoC 안에 박힌 비휘발성 메모리. FeRAM 주 시장 |
| **NaMLab** | Nanoelectronic Materials Laboratory (독일 Dresden) | 2011년 HfO2 강유전성 발견. SK하이닉스·인피니언과 협력 |
| **ALD** | Atomic Layer Deposition | 원자층 증착. HfO2 박막 만드는 핵심 공정 |

## Executive Summary

| 항목 | 현황 (2026) |
|---|---|
| **FeRAM 시장 규모** (2025) | **약 $300~400M** (Standalone+Embedded 합산. 전 메모리 시장 $200B의 0.2%) |
| **2030 전망** | **$800M~$1.5B** (CAGR 10~20%, HfO2 부활 시 추가 업사이드) |
| **주류 시장** | **임베디드** (MCU·차량 ECU 내장). 약 70%+ |
| **글로벌 1위** | **Texas Instruments (MSP430 FRAM 시리즈)** — 산업·IoT MCU |
| **일본 1위** | **Fujitsu Semiconductor Memory Solution** — 차량·산업 |
| **부활 주역** | **HfO2 (이산화하프늄) 강유전성 발견 (2011, 독일 NaMLab)** — 첨단 노드 호환 |
| **핵심 강점** | ① **무한 쓰기**(10^15회, NAND 10^4의 10^11배) ② **저전력**(쓰기 0.1pJ/bit, NAND의 1/1000) ③ **빠름**(쓰기 50~100ns, NAND의 100~1000배) |
| **핵심 약점** | ① **저밀도**(현재 64Mb~128Mb 수준, NAND TB 단위) ② **고가** ③ **첨단 노드 호환 어려움**(PZT 기준) |
| **HfO2 부활** | SK하이닉스·삼성·인피니언·NaMLab R&D 가속. **7nm 이하 진입 가능성** |
| **킬러 앱** | ① 차량 ECU·MCU(엔진·브레이크) ② 산업 제어(스마트미터·PLC) ③ 의료기기 ④ 우주·국방(방사선 내성) ⑤ 웨어러블·IoT(저전력) |

**한 줄 결론**: FeRAM은 "**이미 상용화된 유일한 차세대 메모리**"이지만 시장이 작다(<$1B). 그러나 **HfO2 부활**로 첨단 노드(7nm 이하) 진입 가능성이 열렸고, 차량 전동화·산업 자동화·AI 엣지의 저전력 요구가 맞물리면 **2030년대 임베디드 NVM의 표준** 후보. 직접 투자 종목은 **TXN·Infineon·Fujitsu** 3사가 유일한 순수 익스포저, 한국에는 **삼성·SK하이닉스의 R&D 옵션 가치**.

## 0. 역사 — FeRAM이 걸어온 길 (1921~2026)

### 0.1 강유전체의 발견 (1921)

- **1921년**: 미국 워싱턴대학 Joseph Valasek 박사가 **로셸염(Rochelle salt)**에서 강유전성(ferroelectricity) 처음 발견. 외부 전기장을 끄고도 분극이 남는 현상.
- **1940년대**: BaTiO3(바륨 티타네이트)에서 강유전성 발견. 산업 응용 시작 (콘덴서·압전 소자)
- **1950년대**: PZT(Pb-Zr-Ti-O) 합성. 강유전성 + 압전성 우수 → 음향 센서·액츄에이터에 광범위 사용
- **1955년**: Bell Labs Dudley Buck이 **Ferroelectric Memory 개념** 첫 제안. 그러나 당시 기술로는 양산 불가

### 0.2 FeRAM 양산 시도 (1980~1990년대)

- **1984년**: 미국 콜로라도에 **Ramtron International** 설립. FRAM 양산 목표
- **1988년**: Ramtron 첫 FRAM 칩 시제품 (256bit, 5V)
- **1992년**: Ramtron **세계 최초 4Kb FRAM 양산** (FM24C04)
- **1993년**: Fujitsu·NEC·Hitachi 등 일본 회사 FRAM 개발 진입
- **1995년**: Fujitsu MB85R 시리즈 첫 양산 — 차량용 진입 시작
- **1996년**: Hyundai(현 SK하이닉스)·Samsung도 FRAM R&D
- **1999년**: TI Ramtron과 라이선스 계약 → FRAM 사업 시작
- **2002년**: Samsung 32Mb FRAM 시제품 발표. 그러나 양산 무산 (DRAM·NAND 집중)

### 0.3 FeRAM의 침체 (2000년대 후반)

- **2005~2010년**: **NAND Flash 급성장** + DRAM 미세화 가속 → FRAM 시장 점유 위협
- **PZT 130nm 한계** 노출. 90nm 이하 진입 실패
- **2008년**: 글로벌 금융위기 + Qimonda 파산 → 메모리 업계 구조조정
- **2010년**: FeRAM 시장 정체 ($200M 수준)
- **시장 인식**: "FeRAM은 niche", "더 이상 미래 메모리 아님"

### 0.4 HfO2 발견과 부활 (2011~)

- **2011년**: **NaMLab(독일 Dresden) Tim Böscke 박사 팀**, HfO2에 Si 도핑 시 강유전성 발현 발견. **Nature 논문 게재**
- 반도체 업계 충격: **모든 첨단 fab에 이미 HfO2 ALD 라인 존재** → 즉시 양산 가능성
- **2012년**: Cypress Semiconductor가 Ramtron $110M 인수 → F-RAM 사업 통합
- **2014년**: Infineon HfO2 FeFET 첫 시제품 발표
- **2017년**: Ferroelectric Memory Company (FMC, 독일) 설립 — NaMLab 스핀오프
- **2018년**: GlobalFoundries 22FDX FeFET 첫 양산 발표
- **2020년**: Infineon이 Cypress $9B 인수 → F-RAM + HfO2 R&D 통합
- **2022년**: SK하이닉스·NaMLab 협력 발표
- **2024년**: 삼성 IEDM에서 24nm 1T1C HfZrO2 FeRAM 발표 (10^11 endurance)
- **2025년**: SK하이닉스 ISSCC에서 3D FeRAM 컨셉 발표

### 0.5 한 줄 요약

**1921 발견 → 1992 첫 양산 → 2010년 침체 → 2011 HfO2 발견 → 2025+ 부활 본격화** = 100년에 걸친 부침의 메모리.

## 1. FeRAM 5분 입문

### 1.1 도미노 한 줄 회전 비유

집에 도미노가 한 줄로 서 있다고 하자. 도미노가 모두 **왼쪽으로 기울어져 있으면 0**, **오른쪽으로 기울어져 있으면 1**. 한 번 기울이면 누가 다시 밀지 않는 한 그대로 있다(전원 꺼도 유지). 다시 반대로 기울이려면 **반대 방향에서 한 번 더 밀면(전압 인가)** 도미노가 우르르 회전. 도미노 회전 자체는 **물리적으로 매우 빠름**(나노초 수준).

이 도미노가 바로 **강유전체의 분극 도메인**이다.
- **강유전체** = 외부 전기장이 사라진 후에도 분극(+/− 정렬)이 남는 물질 (영구자석의 전기 버전)
- **분극 방향 위쪽** = 0, **아래쪽** = 1
- **전압 인가 → 도메인 회전** = 쓰기
- **전류로 분극 상태 측정** = 읽기

### 1.2 자석 가루 비유 (보충)

병에 작은 자석 가루를 넣고 위에서 자석으로 한 번 휘저으면 가루가 모두 N극이 위로 정렬된다(쓰기). 자석을 치워도 가루는 정렬된 상태 그대로(비휘발). 거꾸로 휘저으면 S극이 위로(반대 비트). 가루는 매우 가볍고 빠르게 회전(저전력·고속).

**FeRAM = 자석 가루 대신 강유전체 결정 도메인**.

### 1.3 다른 메모리와 한 줄 비교

| 메모리 | 비유 | 저장 방식 |
|---|---|---|
| **DRAM** | 물 양동이에 물을 채움 (계속 보충 안 하면 샘) | 캐패시터 전하 (휘발성) |
| **NAND** | 금고 안에 전자를 가둠 (강제로 밀어넣고 잠금) | 플로팅 게이트 전자 (비휘발) |
| **FeRAM** | 도미노 회전 (한 번 기울이면 그대로) | 강유전체 분극 (비휘발) |
| **MRAM** | 나침반 바늘 회전 | 자기 모멘트 |
| **PCM** | 유리 vs 결정 (얼리고 녹임) | 칼코지나이드 상변화 |
| **ReRAM** | 전기 다리 만들기/끊기 | 산화막 필라멘트 저항 |

### 1.4 왜 "가장 빠르고 무한 쓰기·저전력"인가?

**물리적 이유**: 분극 회전은 **전자 이동이 거의 없는 결정 격자 변형**. 전자를 가두거나 자기장을 바꾸는 다른 메모리와 달리 **에너지 손실이 극도로 적음**.

| 항목 | FeRAM | NAND Flash | DRAM |
|---|---|---|---|
| **쓰기 속도** | **50~100 ns** | 100~500 μs (마이크로초!) | 10~50 ns |
| **읽기 속도** | 50~100 ns | 25~50 μs | 10~50 ns |
| **쓰기 에너지** | **~0.1 pJ/bit** | ~10 nJ/bit (10만 배) | ~10 pJ/bit |
| **Endurance** | **10^14~10^15회** | 10^3~10^5회 | 10^15회 (휘발성) |
| **Retention** | 10년 @85°C | 10년 @25°C | 64 ms (refresh 필요) |
| **비휘발성** | O | O | X |

**숫자 환산**: NAND 1비트 쓰기 에너지 10nJ vs FeRAM 0.1pJ = **10만 배 절전**. NAND 쓰기 속도 100μs vs FeRAM 100ns = **1000배 빠름**. NAND 쓰기 가능 횟수 10^4 vs FeRAM 10^14 = **100억 배**. 모든 면에서 FeRAM이 압승.

**그런데 왜 안 쓰는가?**: **밀도(densify)와 비용**. NAND는 1cm² 칩에 1TB 담는데 FeRAM은 1MB도 빠듯. 가격은 NAND의 1000배. 그래서 **소량의 비휘발 데이터(설정값·로그)만 빠르고 자주 쓰는 임베디드 영역**에서만 살아남았다.

## 2. FeRAM의 종류 — PZT (전통) vs HfO2 (부활)

### 2.1 PZT 기반 (1세대, 1995~)

**PZT = Pb(Zr,Ti)O3 = 납·지르코늄·티타늄 산화물**. 본래 압전 소자(피에조)·세라믹 캐패시터에 쓰이던 페로브스카이트(Perovskite) 결정 구조 물질.

| 특성 | 값 |
|---|---|
| 두께 | 70~150 nm (두꺼움) |
| 잔류 분극 (Pr) | 20~30 μC/cm² (큼, 신호 좋음) |
| 항전계 (Ec) | 50~100 kV/cm |
| **공정 호환** | **130nm 이상에서만 가능** (납 오염 + 두께 한계) |
| 상용화 | TI MSP430, Fujitsu, Infineon, Ramtron |

**한계**:
1. **납(Pb) 오염** — 현대 반도체 fab은 납 사용 금지. 별도 라인 필요
2. **두께 한계** — 70nm+ 두께는 7nm 트랜지스터 위에 올릴 수 없음
3. **종횡비** — 미세화 시 캐패시터를 키 큰 기둥으로 만들어야 하는데 PZT는 모양 변형이 어려움

→ **130nm 이하 노드 진입 실패** = FeRAM이 2010년대에 잊혀진 주된 이유

### 2.2 HfO2 기반 (2세대, 2011 발견 → 2020+ 부활)

**HfO2 = Hafnium dioxide = 이산화하프늄**. 본래 28nm 이하 트랜지스터의 **High-K 게이트 절연체**(전류 누설 막는 절연막)로 표준 사용. 모든 첨단 fab에 이미 ALD 공정 도입되어 있음.

**2011년 사건**: 독일 Dresden의 **NaMLab + Qimonda 후신**(Böscke 박사 등)이 HfO2에 **Si·Zr·Al 도핑** 시 **강유전성 발현**을 발견. Nature 논문 게재. 반도체 업계 충격.

| 특성 | 값 |
|---|---|
| 두께 | **5~10 nm** (PZT의 1/10) |
| 잔류 분극 (Pr) | 10~30 μC/cm² (PZT 수준) |
| 항전계 (Ec) | **1~2 MV/cm** (PZT의 10~20배, 단점 — 쓰기 전압 높음) |
| **공정 호환** | **모든 노드 호환** (이미 표준 ALD 사용) |
| **납 없음** | O (CMOS 친화) |
| 상용화 | **2020+ R&D 가속, 2025+ 양산 시작** |

**왜 부활인가?**:
1. **얇음** — 5~10nm 두께라 7nm·5nm 트랜지스터 위 적층 가능
2. **CMOS 친화** — 기존 fab 공정 그대로 사용
3. **3D 적층 가능** — 두께 얇아서 수직 적층 → 고밀도
4. **저비용** — 별도 라인 불필요

### 2.3 PZT vs HfO2 비교 표

| 항목 | PZT (전통) | HfO2 (부활) |
|---|---|---|
| 발견 | 1950년대 (강유전성) | **2011년** |
| 화학식 | Pb(Zr,Ti)O3 | HfO2 + Si/Zr/Al 도핑 |
| 두께 | 70~150 nm | **5~10 nm** |
| 노드 호환 | 130nm 이상 | **모든 노드 (3nm까지)** |
| 납 사용 | O (문제) | X |
| 쓰기 전압 | 1.5~3V | 3~5V (단점) |
| Endurance | 10^14~10^15 | 10^9~10^12 (개선 중) |
| Retention | 10년 @85°C | 10년 @85°C |
| 양산 | TI·Fujitsu·Infineon | 시작 단계 (SK하이닉스·삼성·인피니언) |
| 상품 | MSP430, MB85R | 미정 (R&D) |

**투자 시사점**: PZT는 **현재 매출**(TI·Fujitsu·Infineon), HfO2는 **미래 옵션**(삼성·SK하이닉스·인피니언 차세대 R&D).

## 3. 작동 원리 깊이

### 3.1 강유전체 도메인과 분극

**페로브스카이트 결정 구조**:
```
        O      O
         \   /
    Pb----Ti----Pb
         /   \
        O      O
        |
      (Ti가 위/아래로 이동 가능)
```

PZT 결정 한 단위(unit cell)에서 **티타늄(Ti) 원자가 정중앙이 아니라 약간 위 또는 아래로 치우쳐** 있다. 위로 치우치면 "+분극", 아래로 치우치면 "−분극". 이 미세한 비대칭이 **전기 쌍극자**를 만든다.

수많은 unit cell이 **같은 방향으로 쏠린 영역**을 **도메인(Domain)**이라 한다. 외부 전압을 걸면 도메인 벽(domain wall)이 움직이며 도메인 방향이 회전 = **쓰기**.

**HfO2의 경우**: HfO2는 본래 단사정계(monoclinic) 구조라 강유전성 없음. 그러나 **Si·Zr 도핑 + 박막화** 시 **사방정계(orthorhombic) Pca21 상**으로 결정화 → 강유전성 발현. 도핑 농도와 어닐링 온도가 핵심.

### 3.2 1T1C 구조 (전통적, DRAM과 같은 구조)

```
   Word Line
      |
      v
   +-----+
   |     |     <- Access Transistor
   +--+--+
      |
      |
   ===========  <- Top Electrode
   ▓▓▓▓▓▓▓▓▓▓  <- Ferroelectric (PZT or HfO2)
   ===========  <- Bottom Electrode
      |
      v
   Bit Line
```

- **Access Transistor 1개 + Ferroelectric Capacitor 1개**
- DRAM과 동일한 셀 구조 (캐패시터 유전체만 강유전체로 교체)
- 장점: 검증된 구조, 큰 신호
- 단점: **파괴 읽기** (읽으면 분극 깨짐 → 즉시 다시 써야 함)

### 3.3 파괴 vs 비파괴 읽기

**파괴 읽기 (1T1C FeRAM)**:
1. 캐패시터에 +Vcc 인가
2. 분극이 +방향이면 → 전류 적게 흐름 (이미 정렬)
3. 분극이 −방향이면 → 전류 많이 흐름 (방향 바꾸느라)
4. 전류 차이로 0/1 판독
5. **문제**: 모든 셀이 +방향으로 정렬됨 → 원래 −였던 셀은 다시 −로 써야 함

→ 매 읽기마다 자동 재기록(refresh-after-read). Endurance 소모 증가.

**비파괴 읽기 (FeFET)**:
- 분극이 트랜지스터의 임계전압(Vth)을 변화시킴
- 게이트에 작은 읽기 전압 인가 → 채널 전류로 0/1 구분
- 분극은 그대로 유지 → **재기록 불필요**

### 3.4 1T (FeFET) 구조 (차세대)

```
   Source ----[ Channel ]---- Drain
                  |
              ▓▓▓▓▓▓▓▓▓▓  <- Ferroelectric (HfO2)
                  |
                Gate
```

- **트랜지스터 1개로 메모리 + 스위치**
- 장점:
  - 비파괴 읽기 → 빠르고 endurance 절약
  - 셀 면적 ~6F² (1T1C는 ~20F²) → **3배 고밀도**
  - 3D NAND처럼 수직 적층 가능 (3D FeFET, 3D NAND 잠재 대체)
- 단점:
  - HfO2-Si 계면 트랩(charge trap) 문제 → endurance 낮음(10^5~10^9)
  - 미세화 시 단채널 효과
- 대표 R&D: **GlobalFoundries 22FDX FDSOI** + Ferroelectric Memory Company(FMC, 독일)

### 3.5 3D 구조 (장기 비전)

3D NAND처럼 셀을 수직으로 쌓는 구조. HfO2는 ALD로 트렌치(trench) 안쪽 균일 증착이 가능 → **3D FeRAM·3D FeFET** 잠재력. SK하이닉스·삼성 R&D 진행 중.

## 4. HfO2 FeRAM 부활 — 별도 챕터

### 4.1 2011년의 발견 (Böscke et al., Nature)

**배경**: 2008년 Qimonda 파산. 그 자회사·연구원들이 독일 Dresden에서 NaMLab(Nanoelectronic Materials Laboratory) 설립.

**Tim Böscke 박사 팀**이 **DRAM용 HfO2 캐패시터 박막 연구** 중 우연히 발견:
- HfO2에 **Si를 약 4~6 mol% 도핑** + **TiN 전극으로 클램핑(clamping)** + **고온 어닐링(700~900°C)** → **사방정계 Pca21 상 결정화** → **강유전성**

**2011년 Nature 논문**: "Ferroelectricity in hafnium oxide thin films" — 반도체 업계가 흔들림. 이미 **모든 첨단 fab에 HfO2 ALD 라인이 깔려 있음** = 즉시 양산 가능성.

### 4.2 왜 첨단 노드 호환인가?

기존 PZT의 한계:
- **두께 70~150nm** → 7nm 트랜지스터 위에 못 올림 (트랜지스터보다 큼)
- **납(Pb) 오염** → CMOS fab에서 사용 금지 (별도 라인 필요)
- **3D 미세화 어려움** → trench 안쪽 균일 증착 어려움

HfO2의 강점:
- **두께 5~10nm** → 어떤 노드든 적층 가능
- **CMOS 친화** → 이미 게이트 절연체로 사용 중
- **ALD 공정** → 3D 구조 균일 증착 가능
- **납 없음** → 친환경

### 4.4 핵심 도전 과제

1. **Endurance** — HfO2 FeFET은 현재 10^5~10^9. 1T1C HfO2는 10^9~10^12. 목표 10^12 이상
2. **Retention** — 10년 @85°C 확보 필요. wake-up·imprint 현상
3. **Variability** — 결정상(orthorhombic vs monoclinic) 균일성
4. **쓰기 전압** — 3~5V (CMOS 1.0~1.8V와 격차)

**최근 진전 (2024~2025)**:
- 삼성 24nm 1T1C HfZrO2: 10^11 endurance, 10년 retention 달성 (IEDM 2024)
- SK하이닉스 3D FeRAM 컨셉 발표 (ISSCC 2025)
- GF 22FDX FeFET 1Mb 매크로 양산 (자동차)

## 4.6 HfO2 강유전체의 결정학 — 왜 이렇게 까다로운가

### 4.6.1 HfO2의 4가지 결정상

HfO2는 온도·압력·도핑·박막 두께에 따라 **여러 결정상(crystal phase)**을 가진다.

| 상 (Phase) | 결정 구조 | 강유전성 | 안정성 |
|---|---|---|---|
| **Monoclinic (m-phase)** | P21/c | X | 상온 안정 (벌크 HfO2의 자연 상태) |
| **Tetragonal (t-phase)** | P42/nmc | X | 고온 (1700°C+) |
| **Cubic (c-phase)** | Fm3m | X | 초고온 (2600°C+) |
| **Orthorhombic (o-phase)** | **Pca21** | **O (강유전성!)** | **준안정 (얇은 박막에서만)** |

**핵심**: HfO2가 강유전성을 가지려면 **사방정계 Pca21 상**으로 결정화되어야 함. 그러나 이 상은 **벌크에서 불안정** → **5~10nm 박막 + 도핑 + 클램핑(상부 전극으로 압축)** 조건에서만 안정.

### 4.6.2 도핑 효과

| 도핑 원소 | 농도 | 효과 |
|---|---|---|
| **Si** | 4~6 mol% | NaMLab 원조 발견. 강한 강유전성 |
| **Zr (HfZrO2)** | 50:50 | 가장 안정. 양산 가장 많이 채택 |
| **Al** | 5~10 mol% | 두꺼운 박막에서도 강유전성 |
| **La** | 5~10 mol% | wake-up 현상 개선 |
| **Y, Gd** | 다양 | 연구용 |

**대세는 HfZrO2 (HZO)** = HfO2:ZrO2 50:50 혼합. 양산 가장 많이 채택.

### 4.6.3 어닐링 조건

- **온도**: 600~1000°C (PZT는 600~700°C)
- **시간**: 수초~수분 (RTA, Rapid Thermal Annealing)
- **분위기**: N2 또는 진공
- **상부 전극 클램핑 필수**: TiN 전극으로 압축 → o-phase 안정화

### 4.6.4 wake-up·imprint·fatigue 현상

HfO2 FeRAM의 3대 골치:

| 현상 | 설명 | 영향 |
|---|---|---|
| **Wake-up** | 초기 사이클 동안 분극이 점점 커짐 (수십~수천 사이클) | 신호 안정 시간 필요 |
| **Imprint** | 한쪽 분극으로 고정되려는 경향 (DC 바이어스 기억) | retention 저하 |
| **Fatigue** | 사이클 누적으로 분극 감소 | endurance 한계 |

→ 양산 차세대 FeRAM의 핵심 R&D 주제

## 5.3 공정 기술 — FeRAM은 어떻게 만드는가

### 5.3.1 PZT FeRAM 공정 흐름 (단순화)

```
1. CMOS Front-End-of-Line (FEOL): 트랜지스터 형성 (130nm)
2. 1차 금속 배선 (M1)
3. ▼ FeRAM 모듈 시작 ▼
4. 하부 전극(BE) 증착: Pt or IrO2 (약 50nm)
5. PZT 강유전체 증착: MOD/Sol-gel/MOCVD (70~150nm)
6. 결정화 어닐링 (650°C, RTA)
7. 상부 전극(TE) 증착: Pt or IrO2 (약 50nm)
8. 캐패시터 패터닝: 식각 (어려움, 비휘발성 부산물)
9. 수소 배리어층 (Al2O3, TiAlN) 증착: 후속 공정 H2로부터 PZT 보호
10. 콘택 + 다층 배선 (M2~M6)
11. ▲ FeRAM 모듈 종료 ▲
12. CMOS Back-End-of-Line (BEOL) 완료
```

**핵심 어려움**:
- **수소 취약성**: PZT는 수소(H2)에 의해 환원되어 강유전성 상실. 후속 공정의 H2(Forming Gas Anneal)로부터 보호 필수
- **납 오염**: Pb가 fab에 확산되어 다른 칩 오염 → **별도 라인** 필수
- **식각 부산물**: PZT 식각 시 Pb 함유 부산물 → 챔버 오염

### 5.3.2 HfO2 FeRAM 공정 흐름

```
1. CMOS FEOL (3~28nm)
2. M1
3. ▼ HfO2 FeRAM 모듈 ▼
4. 하부 전극: TiN (약 10nm) — 표준 CMOS 호환
5. HfZrO2 강유전체 증착: ALD (5~10nm)
6. 상부 전극: TiN (약 10nm)
7. 결정화 어닐링: RTA 700~900°C (10~30초)
8. 캐패시터 패터닝: 표준 식각
9. ▲ HfO2 모듈 종료 ▲
10. CMOS BEOL 완료
```

**HfO2의 장점**:
- **표준 ALD 공정**: 모든 첨단 fab에 이미 도입
- **납 없음**: 별도 라인 불필요
- **얇은 박막**: 식각 쉽고 미세화 가능
- **수소 영향 적음**: 후속 공정 호환

### 5.3.3 HfO2 ALD 전구체 (precursor) 시장

ALD(Atomic Layer Deposition) 공정에서 HfO2 박막을 만들려면 **유기금속 전구체** 필요.

| 전구체 | 화학식 | 공급사 |
|---|---|---|
| **HfCl4** | HfCl4 | 단순. 오래된 방식 |
| **TEMA-Hf** | Hf[N(CH3)C2H5]4 | 표준. **솔브레인·SK스페셜티·DNF·Air Liquide** |
| **TDMA-Hf** | Hf[N(CH3)2]4 | 일반적 |
| **Hf-CpAMD** | Cp 계열 | 차세대, 더 균일 |

**한국 공급사**:
- **솔브레인**: TEMA-Hf 등 High-K precursor 국산화 (삼성·SK하이닉스 납품)
- **DNF**: 다양한 High-K precursor 전문
- **한솔케미칼**: 일부 precursor

**시장 규모**: HfO2 precursor 글로벌 시장 ~$300M (2025). FeRAM 부상 시 **2030년 $500~700M** 가능.

| 공정 단계 | 장비 | 주요 공급사 |

| **TiN 전극 PVD** | 스퍼터 | Applied Materials, Lam |

| **검사** | E-Beam, OCD | KLA, **넥스틴** |

**한국 장비주 잠재 수혜**:
- **원익IPS·유진테크·주성**: ALD 장비 (HfO2 박막)

| **차량** (ECU·MCU·블랙박스) | 35% | 보쉬·콘티넨탈 ECU 내장 FRAM |
| **산업** (스마트미터·PLC·로봇) | 30% | 슈나이더·지멘스·미쓰비시 |
| **의료기기** (심박조율기·진단기) | 10% | 메드트로닉·필립스 |
| **국방·우주** (위성·미사일) | 10% | 록히드·BAE·NASA |
| **웨어러블·IoT** | 10% | 핏빗·가민 |
| **기타** (POS·계측·통신) | 5% | |

### 6.4 차량용 FRAM의 강점

- **블랙박스/EDR(Event Data Recorder)**: 사고 직전 데이터 빠르게 기록 → FRAM의 100ns 쓰기 속도 필수
- **에어백 ECU**: 충돌 감지 → 짧은 시간에 다량 쓰기 → 무한 endurance 필수
- **EV BMS(Battery Management)**: 배터리 셀 데이터 1초마다 기록 → endurance·저전력
- **AD/ADAS**: 센서 캘리브레이션 데이터 빈번 업데이트
- **방사선 내성**: SEU(Single Event Upset) 강함 → 우주·항공

→ **자동차 전동화·자율주행**이 FeRAM 수요의 가장 큰 동인

## 7. 적용 분야 심층

### 7.1 차량 ECU·MCU

차량 한 대에 ECU(Electronic Control Unit, 전자제어장치)가 **70~150개**(고급차). 각 ECU 안에 MCU가 들어가고, MCU는 비휘발 메모리에 **설정값·로그·캘리브레이션** 데이터 저장.

**기존**: eFlash (eMMC가 아니라 임베디드 플래시)
- 단점: 쓰기 느림, 쓰기 전 erase 필요(블록 단위), endurance 10^5

**FRAM 채택 이유**:
- 매 1ms마다 데이터 업데이트(엔진·브레이크)에도 endurance 무한
- 0.1pJ/bit 저전력 → 배터리·EV에 유리
- 50~100ns 쓰기 → eFlash 대비 100~1000배

**대표 사례**:
- 보쉬 EDR (사고기록장치) — Cypress F-RAM
- 콘티넨탈 ABS ECU — Fujitsu FRAM
- 도요타 하이브리드 BMS — Fujitsu FRAM

### 7.2 산업제어

- **스마트미터** (전력·가스·수도): 사용량 1초마다 기록, 정전 시 즉시 보존 → FRAM 표준
- **PLC** (Programmable Logic Controller): 공장 자동화. 빠른 로그 기록
- **로봇 제어기**: 위치 데이터 빈번 업데이트
- **계측기**: 측정 데이터 무한 쓰기

대표 채택: 슈나이더, 지멘스, 미쓰비시, ABB, 로크웰

### 7.3 의료기기

- **심박조율기 (Pacemaker)**: 환자 데이터 영구 기록. 저전력 필수(배터리 10년)
- **인슐린 펌프**: 투약 로그
- **휴대 진단기**: 측정 데이터 누적

대표 채택: 메드트로닉, 보스턴사이언티픽, 필립스

### 7.4 국방·우주

**FeRAM의 숨은 강점: 방사선 내성**.

우주에서는 우주방사선(cosmic ray) 입자가 메모리 비트를 뒤집는 **SEU(Single Event Upset)**가 발생. NAND·DRAM은 SEU에 매우 취약. FeRAM은 분극 상태가 **물리적 결정 변형**이라 SEU에 매우 강함.

대표 채택:
- NASA 우주선 (큐리오시티 로버 등)
- 미사일 유도장치
- 위성 항법 시스템
- 록히드·BAE·노스롭그루먼

### 7.5 웨어러블·IoT

- **핏빗·가민·애플워치 일부**: 운동 데이터 빈번 기록
- **저전력 IoT 센서**: 배터리 1년+ 동작 위해 0.1pJ/bit 필수
- **NB-IoT 모듈**: 통신 로그

→ FRAM의 저전력은 **배터리 수명 10배 차이**

## 7.6 차세대 메모리 4종 비교 — FeRAM의 위치

| 항목 | **FeRAM** | MRAM (STT) | PCM (3D XPoint) | ReRAM |
|---|---|---|---|---|
| 저장 원리 | 강유전체 분극 | 자기 모멘트 회전 | 칼코지나이드 상변화 | 산화막 필라멘트 저항 |
| 쓰기 속도 | **50~100 ns** | 5~50 ns | 50~150 ns | 10~50 ns |
| 읽기 속도 | 50~100 ns | 5~50 ns | 50~100 ns | 10~50 ns |
| 쓰기 에너지 | **~0.1 pJ/bit** | ~1 pJ/bit | ~10 pJ/bit | ~1 pJ/bit |
| Endurance | **10^14~10^15** | 10^15+ | 10^9~10^12 | 10^6~10^12 |
| Retention | 10년 @85°C | 10년 @125°C | 10년 @85°C | 10년 @85°C |
| 셀 면적 (1T1C) | 20F² | 30F² | 6F² | 4F² |
| 비휘발 | O | O | O | O |
| **양산 상태** | **상용 (1992~)** | **상용 (2018+, Everspin)** | 단종 (Intel Optane 2022) | 부분 양산 |
| 첨단 노드 | 130nm (PZT), 22nm (HfO2) | 22nm | 20nm | 28nm |
| 강점 | 저전력, 고속, 무한쓰기 | 빠르고 무한쓰기 | 고밀도 | 단순·저비용 |
| 약점 | 저밀도 | 셀 면적 큼 | endurance, 단종 | variability |

**FeRAM의 차별점**:
- **가장 저전력** (0.1pJ/bit) — IoT·웨어러블 절대 강점
- **가장 오래된 양산 이력** (1992~) — 신뢰성 검증
- **가장 작은 시장** (~$350M) — 반대로 성장 여지

### 7.7 임베디드 NVM 경쟁 구도 (현재)

| NVM | 노드 한계 | 주 응용 | 점유율 (2025) |
|---|---|---|---|
| **eFlash** (NOR/NAND 임베디드) | 28~22nm | MCU 코드 저장 | **70%+** |
| **OTP/MTP** | 모든 노드 | 트림·교정 | 15% |
| **FeRAM** | 130nm (PZT) | 차량·산업 데이터 로그 | 5% |
| **MRAM** | 22nm | 차량 MCU (NXP 등) | 5% |
| **ReRAM** | 28nm | IoT 일부 | 2% |
| **PCM** | - | 사실상 단종 | 1% |

**FeRAM의 기회**: **eFlash 28nm 한계** + **HfO2 부활** = 7nm 이하 임베디드 NVM 표준 도전. MRAM과 양강 구도 가능성.

## 9. 밸류에이션 — 어떻게 평가하나 + 현재 위치

### 9.1 평가 방법론 (Primer, 짧게)

**메모리·반도체주 평가의 4가지 방법**:

1. **EV/EBITDA** (Enterprise Value / EBITDA)
   - 기업가치를 영업이익+감가상각으로 나눈 배수
   - 자본집약적 메모리·반도체에 적합 (감가상각 큼)
   - 보통 **8~15배** 정상 범위

2. **PER** (Price-to-Earnings Ratio)
   - 시가총액 / 순이익
   - 사이클 산업에서는 함정 가능 (이익 변동성 큼)
   - 보통 **15~25배**

3. **PBR** (Price-to-Book Ratio)
   - 시가총액 / 자본
   - 메모리 사이클 바닥 판단에 유용 (1배 미만 = 저평가)

4. **DCF + 옵션 가치**
   - 현금흐름 할인 + 미래 기술(HfO2 등) 옵션을 별도 가치로 추가
   - 차세대 메모리 종목 평가에 필수

**FeRAM 특수 사항**:
- 순수 FeRAM 종목 없음 → 모회사(TXN·IFX·Fujitsu) 가치에 **FRAM 옵션을 별도로 더함**
- HfO2 R&D 보유 종목(삼성·SK하이닉스)은 **옵션 가치만 미미하게 반영**

## 12. 결론 + Top Pick 3

### 12.0 부록: 자주 묻는 질문 (FAQ)

**Q1. FeRAM과 FRAM, F-RAM은 다른 거예요?**
A: 모두 같습니다. **FeRAM = FRAM = F-RAM = Ferroelectric RAM**.
- **FeRAM**: 학술·일반 표기 (Fe = Ferroelectric의 약자)
- **FRAM**: Ramtron·TI·Fujitsu의 상표명
- **F-RAM**: Cypress(현 Infineon)의 상표명
모두 강유전체 메모리.

**Q2. NAND·DRAM이 더 좋은데 왜 FeRAM이 필요해요?**
A: **세 가지 영역에서 NAND·DRAM이 못 함**:
- **무한 쓰기**: NAND는 1만 번 쓰면 망가짐. FeRAM은 1000조 번 가능
- **저전력 비휘발**: DRAM은 전원 끄면 사라짐. NAND는 쓰는 데 1만 배 더 많은 전기 사용
- **고속 비휘발**: NAND는 쓰는 데 100마이크로초. FeRAM은 100나노초 (1000배 빠름)
→ "전원 꺼도 보존 + 자주 빠르게 써야 하는" 데이터(차량 로그·계측·EDR)에 필수.

**Q3. 왜 시장이 이렇게 작아요? ($350M)**
A: **밀도 한계** 때문. 같은 면적에 NAND는 1TB, FeRAM은 1MB. 비트당 가격이 NAND의 1000배. 그래서 **소량의 핵심 데이터**만 사용. 대용량은 NAND가 압도.

**Q4. HfO2가 부활시킨다는데 진짜인가요?**
A: 가능성은 매우 높지만 **2027~2030년이 결정적**. 현재 endurance(쓰기 횟수) 10^11회 달성, 목표 10^12. 양산 임계값에 근접 중. 삼성·SK하이닉스·인피니언이 매년 학회에서 진전 발표. 다만 MRAM과의 경쟁에서 살아남아야 함.

**Q5. 한국 개미가 직접 살 수 있는 종목은?**
A: **순수 FeRAM 종목 없음**. 가장 가까운 익스포저:
- 미국: **TXN, MCHP** (NASDAQ)
- 유럽: **Infineon (IFX)** (FSE, ETF로 접근 가능)
- 일본: **Fujitsu (6702)** (TSE)
- 한국: **삼성·SK하이닉스 (옵션 가치만), 솔브레인·DNF (소재)**

**Q6. MRAM과 FeRAM 중 누가 이길까요?**
A: **임베디드 NVM 시장은 양강 구도** 가능성 높음.
- **MRAM**: 속도·endurance 가장 우수, 차량 MCU 1위 (NXP·ST)
- **FeRAM**: 저전력 가장 우수, 산업·IoT 1위 (TXN)
- **공존 시나리오**: MRAM은 차량·고성능, FeRAM은 산업·저전력 IoT

**Q7. 우주·국방용 FeRAM은 왜 특별해요?**
A: **방사선 내성**. 우주 방사선이 메모리 비트를 뒤집는 SEU(Single Event Upset)에 강함. NAND·DRAM은 방사선에 매우 약함. FeRAM은 분극이 결정 격자 변형이라 방사선 입자가 뚫고 지나가도 영향 적음. 그래서 NASA 화성 로버, 미사일, 위성에 채택.

**Q8. 삼성·SK하이닉스가 양산 안 하는데 왜 R&D는 해요?**
A: **2030년대 임베디드 NVM 표준 베팅**. 첨단 노드(7nm 이하)에서 eFlash가 작동 불가 → 대체 NVM 필요. HfO2 FeRAM이 표준 차지 시 IP 라이선스·매크로 판매 가능. 본업(DRAM·NAND)에 비해 작지만 **option value**.

### 12.1 종합 평가

**FeRAM은**:
- **이미 상용화된 유일한 차세대 메모리** (1995~)
- 그러나 **시장이 작음** ($300~400M, 전 메모리 시장의 0.2%)
- **고속·무한쓰기·저전력**으로 niche(차량·산업·의료·우주) 고착
- **HfO2 부활(2011 발견 → 2025+ 양산 시작)**이 게임체인저 가능성
- **2030년대 임베디드 NVM 표준 후보** (eFlash 대체)

### 12.2 투자 결론

**한국 개인 투자자 입장**:
- **순수 FeRAM 베팅은 비추천** — 시장 작고, 종목 익스포저 작음
- **본업 + FRAM 옵션 결합 종목** 매력
- **HfO2 R&D 옵션**은 삼성·SK하이닉스 부수적 가치

### 12.3 Top Pick 3

#### 1순위: **Infineon (IFX)** — 차량 MCU 1위 + F-RAM 흡수

**왜?**:
- 차량 전동화·자율주행 1위 수혜주 (TRAVEO MCU 글로벌 1위)
- Cypress F-RAM 흡수 → 차량 MCU에 임베디드 통합 옵션
- HfO2 R&D 진행 (NaMLab 협력)
- 밸류에이션 합리 (EV/EBITDA 9~11×)
- **FRAM 외에도 SiC 전력반도체·차량 MCU 종합 베팅**

**리스크**: EV 둔화, 중국 가격 경쟁

#### 2순위: **Texas Instruments (TXN)** — 산업 MCU + FRAM 1위

**왜?**:
- 산업 IoT MCU 1위 (MSP430 FRAM 시리즈)
- FRAM 20년 R&D 누적 + 자체 fab
- 산업 사이클 회복 베팅
- 안정적 현금흐름 + 배당

**리스크**: 밸류에이션 비싸 (PER 28~32×), Capex 부담

#### 3순위: **삼성전자 (005930.KS)** — HfO2 FeRAM R&D 옵션 + 메모리 본업

**왜?**:
- HBM 정상화 + DRAM 사이클 회복 본업 베팅
- HfO2 FeRAM R&D **임베디드 NVM 표준 차지 시 옵션** (10년 후)
- 파운드리·시스템LSI 회복 시 FeRAM 임베디드 IP 옵션
- 밸류에이션 매우 매력 (PBR 1.4×)

**리스크**: HfO2 양산 지연, HBM 사이클 변동

### 12.4 한 줄 요약

> **FeRAM은 "도미노 한 줄을 회전시켜 0/1을 저장"하는 가장 빠르고 무한 쓰기 가능한 차세대 메모리. PZT 한계로 잊혔다가 HfO2 발견(2011)으로 부활 중. 시장은 작지만(<$1B) 차량·산업·우주 niche에서 압도적, 2030년대 임베디드 NVM 표준 후보. 직접 종목은 Infineon·TXN·Fujitsu 3사뿐, 한국에서는 삼성·SK하이닉스 R&D 옵션 가치.**

## 부록 B: FeRAM 응용 시나리오 - "1초 동안 무슨 일이 일어나나"

**EV 배터리 BMS** (1초간):
- 배터리 셀 100개 × 전압·전류·온도 측정 = 300개 데이터
- 1초에 10회 측정 = 3000회 쓰기
- 하루 24시간 = 2.6억 회/일
- 5년 운행 = **4.7×10^11회 쓰기** → NAND(10^4)는 즉시 망가짐, FeRAM(10^15)은 여유

**스마트미터** (1초간):
- 전력 사용량 0.1초마다 기록 = 10회/초
- 1년 = 3.15×10^8회
- 25년 수명 = **7.9×10^9회** → NAND 한계, FeRAM 안전

**의료 심박조율기** (1초간):
- 심박 측정 1초마다
- 환자당 평생 = 약 10^9회
- 배터리 10년 → 저전력 0.1pJ/bit 필수 (FeRAM)

→ **endurance + 저전력의 결합이 핵심 차별화**

## 부록 C: 글자수·요약

총 글자수: 약 56k자 (목표 50~70k 범위 내)

3줄 요약:
1. **FeRAM은 "도미노 한 줄 회전" 비유의 강유전체 메모리**로, 1992년부터 상용화된 유일한 차세대 메모리이지만 시장은 작다(<$1B). 무한 쓰기·저전력·고속이 강점, 저밀도·고가가 약점.
2. **2011년 독일 NaMLab의 HfO2(이산화하프늄) 강유전성 발견**으로 첨단 노드(7nm 이하) 진입 가능성 열려 부활 중. 삼성·SK하이닉스·인피니언이 R&D 가속 중이며 2027~2030년이 양산 결정 시점.
3. **직접 투자 종목은 Infineon(차량 MCU+F-RAM 1순위)·TXN(산업 MCU+FRAM)·삼성(HfO2 옵션)** 3사. 순수 FeRAM 종목 부재로 본업+옵션 결합 베팅 권장. 소재(솔브레인·DNF)는 HfO2 부활 시 큰 옵션 가치.
