## 0. 들어가며 — 이 보고서를 읽기 전에

차세대메모리 4형제(PCM·MRAM·FeRAM·ReRAM) 중에서 **ReRAM(저항변화메모리, Resistive RAM)** 은 가장 늦게 학계에 등장했지만, 가장 빨리 "AI 시대의 진짜 메모리"로 주목받고 있는 친구다.

왜? 다른 차세대메모리들이 "기존 D램·낸드를 대체할 후보"에 머무는 동안, ReRAM은 **"메모리이면서 동시에 연산을 하는 소자"**, 즉 **뉴로모픽 컴퓨팅(인간 뇌 모방 연산)** 의 핵심 부품으로 자리 잡았기 때문이다.

GPU 시대(NVIDIA)가 2020-2025년이었다면, 2027-2035년은 **"메모리 안에서 AI가 계산을 끝내는"** 새로운 컴퓨팅 패러다임 시대가 될 것이고, 그 중심에 ReRAM이 있다.

본 보고서는 ReRAM이 무엇인지, 어떻게 동작하는지, 누가 만들고 있는지, 그리고 한국 개인 투자자가 어떻게 이 파도에 올라탈 수 있는지를 **초등학생도 이해할 비유 + 애널리스트 수준의 깊이** 로 풀어낸다.

## 2. ReRAM 5분 입문 — 산화막 안에 물길을 내자

### 2-1. 비유로 이해하기 — "도시락통 안에 막대를 세웠다 눕히는 게임"

상상해보자. 작은 도시락통이 있다. 뚜껑은 금속, 바닥도 금속, 그 사이에는 **얇은 산화막(녹슨 막)** 이 끼어 있다. 정상 상태에서 산화막은 전기를 통하지 않는다 (= 절연체).

그런데 뚜껑과 바닥에 강한 전압을 걸어주면, 산화막 속의 산소 원자들이 한쪽으로 밀려나면서 **"산소가 빠진 길(산소 결손)"** 이 생긴다. 이 길은 전기를 잘 통한다. 즉, **도시락통 안에 작은 전선(필라멘트)** 이 만들어진 것이다.

- 전선이 만들어진 상태 → **저항이 낮음 (LRS, Low Resistance State)** → "1"
- 전선이 끊어진 상태 → **저항이 높음 (HRS, High Resistance State)** → "0"

이 "막대를 세웠다 눕히는" 동작이 곧 ReRAM의 쓰기/지우기다.

### 2-2. 기존 메모리와 어떻게 다른가

| 메모리 | 저장 원리 | 비유 |
|--------|----------|------|
| **D램** | 콘덴서(작은 컵)에 전자(물)를 채움. 시간 지나면 증발 → 끊임없이 보충 필요 | 구멍 뚫린 컵에 물 붓기 |
| **SRAM** | 트랜지스터 6개로 1비트를 잡고 있음. 전기 끊으면 사라짐 | 6명이서 손잡고 서있기 |
| **낸드** | 절연막에 전자를 가두기. 전기 꺼도 유지됨 | 자물쇠 달린 금고에 동전 넣기 |
| **PCM (상변화)** | 칼코겐화물을 녹였다 식혔다 → 결정/비결정 상태 | 사탕을 녹였다 굳히기 |
| **MRAM** | 자석의 N/S 방향으로 저장 | 나침반 돌리기 |
| **FeRAM** | 강유전체의 전기 분극 방향으로 저장 | 자석 대신 전기 막대 돌리기 |
| **ReRAM** | 산화막 안에 전기 물길을 내거나 끊기 | **도시락통 안에 막대 세우기** |

### 2-3. ReRAM의 3대 매력

#### 매력 1: CMOS 호환성 (= "기존 공장에서 그대로 만들 수 있다")

- ReRAM 셀은 **금속-산화막-금속(MIM, Metal-Insulator-Metal)** 구조다.
- 이 구조는 기존 반도체 공장이 매일 만드는 **배선층(BEOL, Back-End-Of-Line)** 위에 그냥 얹을 수 있다.
- 비유: 이미 지어진 아파트 옥상에 옥탑방 하나 더 짓는 격. 기초 공사가 필요 없다.
- 결과: **신규 투자 부담이 적고**, 기존 22nm·28nm 파운드리 라인에서 바로 양산 가능.
- → **TSMC가 22nm eReRAM을 양산할 수 있는 이유**가 바로 이것.

#### 매력 2: 다중상태(MLC) — "한 셀에 여러 비트"

- 필라멘트의 굵기/길이를 조절하면 저항값을 **2단계가 아니라 4단계, 8단계, 16단계**로 만들 수 있다.
- 즉, 한 셀에 1비트(2단계)가 아니라 2비트(4단계), 3비트(8단계), 4비트(16단계)를 넣을 수 있다.
- 이를 **MLC (Multi-Level Cell)** 라고 한다.
- 더 나아가 저항을 **연속적으로** 조절하면 → **아날로그 메모리** = 뉴로모픽 시냅스가 된다.

#### 매력 3: 저전력 + 비휘발

- 쓰기 전압: ~1-3V (낸드 ~20V 대비 압도적으로 낮음)
- 쓰기 에너지: ~pJ (피코줄, 10^-12 J) 수준
- 비휘발: 전원 꺼도 10년 이상 데이터 유지
- → **IoT 센서, 웨어러블, 엣지 AI** 등 배터리 동작 기기에 최적

| 약어 | 풀이 | 한국어 | 한 줄 설명 |
|------|------|--------|-----------|

## 3. ReRAM의 종류 — 산화막? 이온? 뉴로모픽?

ReRAM은 "필라멘트가 무엇으로 만들어지는가"에 따라 크게 3가지 갈래로 나뉜다.

### 3-1. OxRAM (Oxide-based ReRAM) — 주류

- **재료**: HfO2 (산화하프늄), TaOx (산화탄탈), TiOx, ZrOx, AlOx 등
- **원리**: 산화막 속 산소 원자가 빠져나가면서 만들어지는 **산소 결손(Oxygen Vacancy)** 이 모여 전도성 필라멘트 형성
- **장점**:
  - HfO2는 이미 D램·로직 게이트에 쓰이는 표준 재료 → 공정 호환성 최상
  - 비교적 안정적이고 내구성 양호 (10^9-10^12 cycles)
  - 미세화에 유리
- **단점**:
  - 같은 셀이라도 매번 필라멘트가 다른 경로로 형성됨 → **변동성(Cycle-to-Cycle Variation)** 큼
  - 초기 활성화(Forming) 단계에서 높은 전압 필요
- **대표 기업**: TSMC, Panasonic, Fujitsu, 삼성전자, SK하이닉스 (R&D), Crossbar
- **적용**: 임베디드 ReRAM (MCU 코드 저장), Standalone ReRAM, 뉴로모픽

### 3-2. CBRAM (Conductive Bridge RAM) — 이온 이동형

- **재료**: 고체 전해질 (GeS, GeSe, Ag2S 등) + 활성 전극 (Ag, Cu)
- **원리**: 활성 전극에서 **금속 이온(Ag+, Cu+)** 이 전해질 안으로 이동 → 다리(Bridge) 형성 → 저항 낮아짐
- **장점**:
  - 저항 비율(LRS/HRS) 차이가 매우 큼 → 읽기 마진 우수
  - 저전력
  - MLC에 유리
- **단점**:
  - 금속 이온의 확산 → 데이터 보존성(retention) 문제 가능
  - 활성 전극 재료(Ag, Cu)가 CMOS 공정 오염원이 될 수 있음 → 별도 격리 공정 필요
- **대표 기업**: **Adesto Technologies (Renesas 인수, 2020)**, 일부 IBM·HP 연구
- **적용**: 초저전력 IoT, 보안 칩, 임베디드

### 3-3. 뉴로모픽 ReRAM — 특수 목적

- **재료**: 위 OxRAM/CBRAM 베이스에 추가 엔지니어링
- **원리**: 필라멘트의 굵기/길이를 **연속적으로** 조절 → 저항이 아날로그 값을 가짐 → **인공 시냅스**로 작동
- **장점**:
  - 한 셀이 시냅스 역할 = 곱셈 + 덧셈을 동시에 처리
  - 메모리에서 직접 연산 (CIM, Compute-in-Memory)
  - GPU 대비 100-1000배 에너지 효율
- **단점**:
  - 정밀도(precision)가 8bit FP 대비 떨어짐 → 추론용은 OK, 학습은 어려움
  - 변동성 제어가 매우 어려움
  - 현재 학계·R&D 단계, 본격 상용화 2028+ 예상
- **대표 기업**: **IBM (TrueNorth, NorthPole)**, **Intel (Loihi 2)**, **Mythic AI**, **SambaNova** (간접), Crossbar (시냅스 IP), Weebit Nano (R&D)
- **적용**: 엣지 AI 추론 가속기, 자율주행 추론, 음성·영상 인식

### 3-4. 비교 표

| 구분 | OxRAM | CBRAM | 뉴로모픽 ReRAM |
|------|-------|-------|---------------|
| **필라멘트 재료** | 산소 결손 | 금속 이온 (Ag, Cu) | 산소 결손 (정밀 제어) |
| **CMOS 호환성** | 매우 좋음 | 보통 (오염 이슈) | 좋음 |
| **저항 비율** | 10x-100x | 100x-10000x | 연속값 (아날로그) |
| **속도 (Set/Reset)** | ns-μs | ns | μs-ms |
| **내구성** | 10^9-10^12 | 10^6-10^9 | 10^6 (아날로그 모드) |
| **MLC 적합성** | 좋음 | 매우 좋음 | 우수 (연속값) |
| **상용화 현황** | 양산 (TSMC, Panasonic) | 양산 (Adesto/Renesas) | R&D + 시제품 |
| **킬러 앱** | 임베디드 MCU | IoT, 보안 | AI 추론 가속기 |
| **대표 기업** | TSMC, Panasonic, Crossbar | Adesto/Renesas | Mythic, IBM, Weebit |

## 4. 작동 원리 깊이 — Set, Reset, 그리고 Forming

### 4-1. ReRAM 셀의 3가지 동작

ReRAM 셀에는 3가지 핵심 동작이 있다.

#### Forming (초기화) — "처음 한 번, 큰 망치로"

- 새로 만들어진 ReRAM 셀은 산화막이 **완전히 절연 상태**다 (저항 ~GΩ).
- 처음 한 번, **높은 전압 (3-5V)** 을 걸어주면 산화막이 부분적으로 "깨지면서" 전도성 길이 형성된다.
- 이것이 **Forming**. 망치로 한 번 세게 쳐서 길을 트는 것.
- 이후로는 낮은 전압으로도 길을 다시 만들었다 끊을 수 있다.
- **단점**: Forming 전압이 높아 **회로 설계가 까다롭다**. → Forming-free ReRAM (Forming 단계가 필요 없는 셀)이 R&D 핵심 과제.

#### Set (쓰기) — "길을 만들자"

- 양극(+)에서 음극(-) 방향으로 전압 인가 (~1-2V)
- 산소 이온이 한쪽으로 이동 → 산소 결손 채널이 연결됨
- → 저항이 낮아짐 (LRS, ~kΩ)
- = **데이터 "1" 쓰기**

#### Reset (지우기) — "길을 끊자"

- 반대 방향 또는 더 높은 전류로 전압 인가
- 산소 이온이 다시 채널로 돌아옴 → 필라멘트 일부 끊어짐
- → 저항이 높아짐 (HRS, ~MΩ)
- = **데이터 "0" 쓰기**

#### Read (읽기) — "조심스럽게 살짝"

- 매우 낮은 전압 (~0.1-0.3V) 인가
- 흐르는 전류량 측정 → LRS인지 HRS인지 판별
- 읽기 전압이 너무 높으면 **읽기 방해(Read Disturb)** 로 셀이 의도치 않게 Set/Reset될 수 있음

### 4-2. I-V 곡선 (전압-전류 특성) 이해

ReRAM의 동작을 그림으로 그리면 **히스테리시스(Hysteresis)** 곡선이 나온다.

```
전류
 ↑
 |     ┌─── LRS (낮은 저항, 가파른 기울기)
 |    /
 |   /
 |  /     ← Set 발생 (전압 1V 부근에서 저항 급감)
 | /  ___
 |/__/   \___ HRS (높은 저항, 완만한 기울기)
 |
 +─────────────→ 전압
       0V    1V    2V
```

- 전압을 0 → +2V로 올리면 어느 순간 갑자기 전류가 폭증 (Set, LRS 진입)
- 전압을 0 → -2V로 내리면 다시 전류가 급감 (Reset, HRS 복귀)
- 이 반복 루프가 ReRAM의 **데이터 쓰기/지우기**

### 4-3. 변동성 문제 — ReRAM 최대 난제

**같은 셀이라도 매번 다르게 반응한다**는 게 ReRAM의 가장 큰 골칫거리다.

#### Cycle-to-Cycle Variation (사이클 간 변동)

- 같은 셀에서 100번 Set/Reset을 반복하면, 100번의 LRS 저항값이 모두 다름
- 일부는 1kΩ, 일부는 5kΩ, 일부는 10kΩ
- → 읽기 마진(margin)이 줄어들고, 오류율 증가

#### Cell-to-Cell Variation (셀 간 변동)

- 같은 칩 안의 100만 개 셀이 각자 다른 특성
- 같은 Set 펄스를 줘도 어떤 셀은 1kΩ, 어떤 셀은 10kΩ
- → 칩 전체 수율 하락

#### 왜 변동성이 큰가?

- 필라멘트는 **확률적(stochastic)** 으로 형성됨 (정확한 위치 제어 불가)
- 산소 결손이 어디에 모일지는 부분적으로 무작위
- 비유: 매번 같은 자리에 비를 뿌리는데, 빗물이 흐르는 길은 매번 미세하게 다르다.

#### 해결 노력

- **재료 엔지니어링**: HfO2 두께·도핑·계면 제어
- **회로 보상**: ECC (오류정정) 강화, Iterative programming (목표 저항에 도달할 때까지 펄스 반복)
- **셀 구조**: 1T1R (1 트랜지스터 + 1 저항) 구조로 전류 정밀 제어
- **AI 보정**: 머신러닝으로 변동성 패턴 학습 후 보정

### 4-4. MLC (다중상태 셀) — ReRAM의 비밀 무기

ReRAM의 저항은 **연속적인 값**을 가질 수 있다. 이를 활용하면 한 셀에 여러 비트를 저장할 수 있다.

| 모드 | 상태 수 | 비트/셀 | 밀도 향상 | 도전 과제 |
|------|--------|---------|----------|----------|
| **SLC** | 2 (HRS, LRS) | 1 bit | 기준 | 안정적 |
| **MLC (4-state)** | 4 | 2 bits | 2배 | 변동성 제어 |
| **TLC (8-state)** | 8 | 3 bits | 3배 | 정밀 펄스 제어 |
| **QLC (16-state)** | 16 | 4 bits | 4배 | 고도 ECC 필수 |
| **Analog (연속)** | ∞ | - | 시냅스 모드 | 정밀도와 노이즈 트레이드오프 |

- 낸드플래시는 이미 QLC까지 갔지만 (한 셀에 4비트), 셀당 5-6V 전압이 필요하고 속도가 느림.
- ReRAM은 1-2V로 MLC가 가능 → **저전력 + 고밀도** 의 환상의 조합 가능성.
- 단, 변동성 문제로 양산 MLC는 아직 어려움. 2028+ 예상.

### 4-5. 1T1R 셀 구조

| 구성 요소 | 역할 |
|---------|------|
| **선택 트랜지스터(T)** | 셀 ON/OFF 제어, Sneak current 차단 |
| **저항 소자(R)** | 데이터 저장 (HRS/LRS) |

- 1T1R 구조는 선택성이 우수하지만, **셀 면적이 큼** (트랜지스터가 셀 면적 결정).
- 더 작게 하려면 **선택 소자 + 저항 소자**를 모두 BEOL에 올리는 **Selector + ReRAM** 구조가 필요 → R&D 진행 중.
- Selector는 OTS (Ovonic Threshold Switching, 칼코겐화물) 또는 MIEC (Mixed Ionic Electronic Conductor) 등이 후보.

## 5. 기술 로드맵 — ReRAM은 어디까지 가나

### 5-1. 노드(공정 미세화) 로드맵

| 시기 | 노드 | 셀 면적 | 밀도 | 양산 회사 |
|------|------|--------|------|----------|
| 2013 | 130nm | ~0.2 μm² | 64Mb | Panasonic |
| 2017 | 40nm | ~0.05 μm² | 1Gb | Panasonic, Fujitsu |
| 2020 | 28nm | ~0.025 μm² | 4Gb | TSMC eReRAM |
| 2023 | 22nm | ~0.018 μm² | 8Gb | TSMC, GlobalFoundries |
| 2025 | 16nm | ~0.01 μm² | 16Gb | (R&D) TSMC, 삼성 |
| 2027E | 12nm | ~0.006 μm² | 32Gb | (개발 중) |
| 2030E | 7nm | ~0.003 μm² | 64Gb-128Gb | (목표) |

- 낸드(2025년 200+층 3D NAND, ~64-128Gb 셀)와 비교하면 밀도는 한참 뒤떨어짐.
- 하지만 ReRAM은 **수직 적층(3D ReRAM)** 도 가능해, 2028+ 3D ReRAM이 상용화되면 낸드와 비슷한 밀도 도달 가능.

### 5-2. 속도 로드맵

| 동작 | 현재 (2026) | 2030E |
|------|------------|-------|
| Set | 50-100 ns | 10-20 ns |
| Reset | 50-100 ns | 10-20 ns |
| Read | 5-20 ns | 2-5 ns |
| Forming | 1-10 μs | 100 ns |

- 비교: D램 ~10ns, 낸드 ~10-100μs, MRAM ~5-20ns
- ReRAM은 **D램에 근접하는 속도 + 비휘발성** → 이론적으로 "Storage Class Memory" 후보

### 5-3. 내구성 로드맵

| 시기 | Endurance | 비교 |
|------|-----------|------|
| 2015 | 10^6 | 낸드 SLC 수준 |
| 2020 | 10^9 | DRAM 사용 패턴 일부 가능 |
| 2025 | 10^11-10^12 | DRAM 대체 가능 영역 진입 |
| 2030E | 10^14+ | DRAM 완전 대체 가능 |

(참고: D램 endurance ~10^16, 낸드 SLC ~10^5, MLC ~10^4, QLC ~10^3)

### 5-4. 데이터 보존성 (Retention)

- 현재 양산품: 85°C에서 10년 보장
- 자동차용 (AEC-Q100): 150°C에서 10년 (개발 중)
- → **차량용 MCU 임베디드 메모리**로 확산 중 (Renesas, NXP)

## 6. 시장 규모 — 임베디드 vs Standalone vs 뉴로모픽

### 6-1. ReRAM 시장 분류

ReRAM 시장은 크게 3가지로 나뉜다.

#### (1) 임베디드 ReRAM (eReRAM)
- 정의: MCU·SoC 칩 안에 내장된 ReRAM (코드 저장용)
- 대체 대상: 임베디드 플래시(eFlash)
- 시장 규모 (2026E): 약 3억 달러
- 성장률: CAGR 30-40%
- 주요 기업: TSMC, GlobalFoundries, Panasonic, Fujitsu, Renesas

#### (2) Standalone ReRAM
- 정의: 독립된 메모리 칩 (낸드·D램과 경쟁)
- 시장 규모 (2026E): 약 1억 달러 (매우 작음)
- 성장률: 본격 개화 2028+
- 주요 기업: Crossbar, Weebit Nano (잠재), 삼성·SK하이닉스 (R&D)

#### (3) 뉴로모픽/CIM ReRAM
- 정의: AI 추론 가속기 안의 ReRAM 어레이 (시냅스 역할)
- 시장 규모 (2026E): 약 1억 달러
- 성장률: CAGR 60-100% (잠재)
- 주요 기업: Mythic AI, IBM, Crossbar, Weebit (IP 라이선스)

### 6-2. 시장 규모 전망 (TAM)

| 연도 | 임베디드 (M$) | Standalone (M$) | 뉴로모픽/CIM (M$) | 합계 (M$) |
|------|--------------|-----------------|-------------------|----------|
| 2024 | 200 | 50 | 50 | 300 |
| 2025 | 250 | 70 | 80 | 400 |
| 2026E | 320 | 100 | 130 | 550 |
| 2027E | 450 | 150 | 220 | 820 |
| 2028E | 650 | 250 | 400 | 1,300 |
| 2029E | 950 | 400 | 750 | 2,100 |
| 2030E | 1,400 | 700 | 1,400 | 3,500 |
| 2032E | 2,800 | 1,500 | 4,000 | 8,300 |
| 2035E | 6,000 | 4,000 | 12,000 | 22,000 |

(출처: 종합 추정 — Yole, Coughlin Associates, IDTechEx, Objective Analysis 보고서 평균)

> **읽는 법**: 2026년 5.5억 달러 → 2030년 35억 달러 → 2035년 220억 달러. 9년 만에 **40배** 성장. 단, 모든 차세대메모리 중 **변동성이 가장 큰** 전망.

### 6-3. 한국 메모리 산업 매출과 비교

- 삼성전자 메모리 매출 (2024): 약 90조 원
- SK하이닉스 매출 (2024): 약 65조 원
- 한국 메모리 합계: ~155조 원 (= 약 1,200억 달러)
- ReRAM 2030E 시장: 35억 달러 = 한국 메모리 시장의 **3% 수준**
- ReRAM 2035E 시장: 220억 달러 = 한국 메모리 시장의 **18% 수준**

→ ReRAM이 메모리 패권을 뒤집을 정도는 아니지만, **AI 추론 가속기 시장(2030E 1,000억 달러)** 의 중요한 한 부분을 차지할 전망.

## 7. 적용 분야 — ReRAM은 어디에 쓰이나

### 7-1. 임베디드 MCU 코드 저장 (현재 주력)

#### 시장 배경
- 자동차·산업·IoT용 MCU(마이크로컨트롤러)는 **펌웨어 코드를 저장**할 비휘발성 메모리가 필요
- 기존엔 **임베디드 플래시(eFlash)** 사용 → 28nm 이하 미세화가 매우 어려움 (별도 마스크 12-15개 추가, 비용 폭증)
- 28nm·22nm 이하 MCU는 **eFlash 대체재** 가 절실 → ReRAM, MRAM, FRAM이 후보

#### ReRAM의 강점
- BEOL 단순 추가 → 마스크 4-5개만 추가
- 22nm까지 양산 검증 (TSMC)
- 자동차 신뢰성 인증 가능

#### 시장 규모
- 임베디드 메모리 전체: 2026E 약 100억 달러
- 그중 eReRAM: 2026E 3.2억 달러 (점유율 ~3%) → 2030E 14억 달러 (점유율 ~10%)

#### 대표 응용
- **차량용 MCU** (Infotainment, ADAS) — Renesas, NXP가 채택 검토
- **산업용 MCU** (모터 제어, 센서 허브) — STMicro, Microchip
- **스마트카드, IC카드** — 보안 + 비휘발성

### 7-2. IoT 센서 — 초저전력 + 비휘발

- IoT 센서는 배터리 동작 → 전력 1mW도 아껴야 함
- ReRAM은 쓰기 에너지 ~pJ → 낸드 대비 100-1000배 저전력
- 또한 비휘발 → 데이터 유지에 전력 0
- 응용: 스마트홈 센서, 웨어러블, 환경 모니터링

### 7-3. 뉴로모픽 컴퓨팅 — 미래 킬러 앱

(상세 내용은 9장에서 별도 다룸)

- 인공지능 추론 가속기 안의 시냅스 역할
- 메모리에서 직접 곱셈·덧셈 → CIM (Compute-in-Memory)
- GPU 대비 **100-1000배 에너지 효율**

### 7-4. AI 추론 (Edge AI)

- 클라우드 AI(서버)는 GPU/HBM 조합이 강세
- 엣지 AI(휴대폰, 카메라, 자동차)는 **저전력 + 작은 모델** 필요
- ReRAM 기반 가속기는 전력 1W 이하로 NN 추론 가능
- 응용: 자율주행 카메라, 음성 비서, 산업 검사 카메라

### 7-5. Storage Class Memory (D램과 낸드 사이)

- D램(빠르지만 비휘발 X) ↔ 낸드(비휘발이지만 느림) 사이의 **Storage Class Memory(SCM)** 시장
- Intel Optane(3D XPoint, PCM 기반)이 시도했으나 2022년 단종
- ReRAM이 그 자리를 노릴 수 있으나, **양산 밀도/비용 경쟁력 미흡** → 단기 가능성 낮음

### 7-6. 보안 칩 — PUF (Physical Unclonable Function)

- ReRAM의 **변동성을 역이용** → 칩마다 고유한 "지문" 생성
- PUF는 복제 불가능한 ID로, 보안 칩·암호화 칩에 사용
- Crossbar가 PUF 솔루션 출시
- 시장은 작지만 단가 높음

### 7-7. 적용 분야 매트릭스

| 분야 | 시장 크기 | ReRAM 적합도 | 양산 시점 | 주요 경쟁자 |
|------|---------|------------|----------|-----------|
| eMCU 코드 | ★★★ | ★★★★★ | 양산 중 | eFlash, MRAM |
| IoT 센서 | ★★ | ★★★★ | 2026-2027 | FRAM, MRAM |
| 뉴로모픽 | ★★★★ | ★★★★★ | 2028+ | SRAM 기반 가속기 |
| Edge AI 추론 | ★★★★ | ★★★★ | 2027-2028 | NPU + SRAM |
| Storage Class Memory | ★★★★★ | ★★ | 2030+ | (Optane 종료, 공백) |
| 보안 PUF | ★ | ★★★★★ | 양산 중 | SRAM PUF |

## 8. 종목 심층 분석

ReRAM은 4대 차세대메모리 중 **가장 다양한 플레이어**가 활동 중이다. 미국·호주·일본·대만·한국의 대표 기업들을 하나씩 뜯어본다.

### 8-1. Crossbar (비상장) — 미국 ReRAM 대표 Pure Play

#### 회사 개요
- **설립**: 2010년, 미국 산타클라라
- **창업자**: Wei Lu (UMich 교수, ReRAM 창시자 중 한 명)
- **포지션**: ReRAM IP 라이선싱 + 시제품 칩
- **펀딩**: 누적 ~$80M (2024 기준)
- **상장 여부**: **비상장** (한국 개인투자자 직접 매수 불가)

#### 기술
- 자체 ReRAM 셀 구조 ("Field Programmable Resistive Switching")
- Si 기반 + 금속 필라멘트 (CBRAM 변형)
- 1T1R + selector 구조
- 28nm 및 그 이하 노드에서 IP 검증 완료

#### 사업 모델
- **IP 라이선싱**: 파운드리(SMIC, Microsemi) 및 메모리 회사에 IP 제공
- **PUF 솔루션**: 보안 칩
- **AI 가속기 IP**: 뉴로모픽 ReRAM IP (Reciva 라인)

#### 투자 포인트
- ✅ ReRAM 진영의 **상징적 회사**, 학계·산업계 영향력 큼
- ✅ IP 모델 → 경박단소, 마진 높음
- ❌ 비상장 → 직접 투자 불가
- ❌ 매출 규모 작음 (~$30-50M 추정)
- ❌ 라이선스 수익이 양산 매출로 직결되지 않음

#### 한국 투자자 접근법
- 직접 투자 불가
- **간접 노출**: SMIC가 Crossbar IP 사용 → SMIC 주가에 일부 반영. 하지만 SMIC 자체가 미국 제재 종목.
- IPO 가능성: 2027-2028 거론되나 확정 X

### 8-2. Weebit Nano (WBT.AX) — 호주 ReRAM Pure Play (★ 핵심 종목)

#### 회사 개요
- **본사**: 이스라엘 (R&D), 호주 ASX 상장
- **티커**: WBT (호주 증권거래소)
- **시가총액**: 약 5-7억 호주달러 (2026년 5월 기준 추정, 약 5,000억 원)
- **설립**: 2015년
- **CEO**: Coby Hanoch
- **주력 기술**: **SiOx 기반 ReRAM** (산화규소 사용 — Rice University 라이선스)

#### 왜 SiOx인가?
- **CMOS 표준 재료** → 별도 신소재 도입 불필요
- 공정 단순화, 양산 비용 낮음
- 다만 HfO2 대비 변동성 큼 (현재 개선 중)

#### 사업 모델
- **IP 라이선싱**: 파운드리·IDM에 ReRAM IP 제공
- **임베디드 ReRAM**: 28nm·22nm·16nm
- **2024년 마일스톤**: SkyWater Technology(미국 파운드리)에서 130nm ReRAM 양산 시작 → 첫 매출 발생

#### 주요 파트너
- **SkyWater Technology** (미국, 130nm 양산)
- **GlobalFoundries** (22FDX 공정 통합 진행 중, 2026년 양산 목표)
- **DB HiTek** (한국, 130nm 평가 진행 중) — **한국 투자자 주목 포인트**
- **Onsemi** (자동차용 MCU 적용 검토)

#### 자금 조달
- **IPCEI (Important Projects of Common European Interest)** — EU의 메모리 자금 지원 프로그램에서 일부 자금 수혜
- 호주 정부 R&D 보조금
- 2024년 자본 조달 ~$30M

#### 재무 (2025 fiscal, 추정)
- 매출: 약 $5-10M (라이선스 + 로열티)
- 적자: 약 $20-30M (R&D 집중)
- 현금: 약 $30-40M

#### 밸류에이션
- 시총 ~$400M (USD)
- EV/Sales: 매출 대비 수십 배 → **전형적 옵션 가치 종목**
- Bull case (2030 매출 $200M, EV/Sales 10x): 시총 $2,000M = **5배 상승**
- Bear case (2028 자금 고갈, 회사 매각/청산): 0.3-0.5배

## 9. 뉴로모픽 컴퓨팅과 ReRAM의 결합

### 9-1. 뉴로모픽 컴퓨팅이란?

**뉴로모픽 컴퓨팅(Neuromorphic Computing)** 은 인간의 뇌 구조를 모방한 컴퓨팅 방식이다.

- 인간 뇌: 약 860억 개 뉴런(신경세포) + 100조 개 시냅스(연결)
- 시냅스는 두 뉴런 사이의 "연결 강도"를 저장 → 학습/기억의 본질
- 뇌의 에너지 소비: 약 20W (전구 1개 수준)
- 같은 작업을 GPU로 하면 → 수 kW 필요

뉴로모픽 칩은 이 뇌 구조를 실리콘으로 구현하려는 시도다.

### 9-2. 왜 ReRAM이 뉴로모픽의 핵심인가?

ReRAM의 **저항이 연속적으로 변할 수 있는 특성**은 시냅스의 "연결 강도"와 똑같다.

| 시냅스 | ReRAM |
|--------|-------|
| 연결 강도 (가중치) | 저항값 (R) |
| 학습 → 강도 변화 | Set/Reset → 저항 변화 |
| 신호 전달 = 강도 × 입력 | 전류 = 전압 × (1/R) (옴의 법칙) |
| 100조 개 병렬 처리 | ReRAM 어레이 병렬 처리 |

특히 **ReRAM 어레이(2차원 격자) 구조**에서 행에 입력 전압을 인가하고 열에서 전류를 읽으면, **한 번의 동작으로 수천 개 곱셈+덧셈을 동시 수행**한다. 이것이 **CIM (Compute-in-Memory)** 의 핵심.

### 9-3. CIM vs 기존 컴퓨팅

#### 기존 컴퓨팅 (Von Neumann 구조)
1. CPU/GPU가 메모리에서 데이터 읽음
2. CPU/GPU가 계산
3. 결과를 메모리에 다시 씀
4. 반복

→ 데이터 이동에 **에너지의 90%** 소비됨 ("메모리 벽" 문제)

#### CIM (Compute-in-Memory)
1. 메모리(ReRAM 어레이)에 가중치 저장
2. 입력 신호를 메모리에 직접 인가
3. 메모리가 계산 결과(전류)를 출력
4. 데이터 이동 없음

→ 에너지 효율 **100-1000배** 향상

### 9-5. 한계와 과제

ReRAM 뉴로모픽이 본격화되려면 몇 가지 난제가 있다.

#### 정밀도 한계
- ReRAM 저항값의 정밀도는 **3-4 bit 수준** (8 bit FP보다 낮음)
- AI 추론 정확도가 약간 떨어질 수 있음
- 해결: 복합 정밀도 (Mixed Precision), HW-aware Training

#### 변동성
- 같은 가중치를 여러 셀에 저장해도 결과가 약간씩 다름
- 해결: 다중 셀 평균, ECC, 보정 회로

#### 학습(Training)에는 부적합
- ReRAM은 추론(Inference)에는 강하나 학습은 정밀도 문제로 어려움
- 학습은 GPU/HBM에서, 추론은 ReRAM CIM에서 → **하이브리드** 구조가 표준

## 10. 밸류에이션 — 어떻게 평가하나 + 현재 위치

### 10-1. 평가 방법론 Primer

ReRAM 종목 평가는 **3가지 방법**을 혼용한다.

#### 방법 1: EV/Sales (매출 대비 기업가치)
- **적용 대상**: 매출이 적고 적자인 R&D 회사 (Weebit, 4DS)
- **공식**: 시가총액 + 부채 - 현금 = EV → EV / 매출 = EV/Sales
- **벤치마크**: 반도체 IP 회사 일반 EV/Sales 5-15배, 고성장기엔 30-100배

#### 방법 2: 옵션 가치 (Real Option)
- **적용 대상**: 양산 시점 불확실, 성공 시 큰 수익 (모든 ReRAM 회사)
- **개념**: 매출이 0이어도, **성공 확률 × 미래 가치** 로 평가
- **공식 (단순)**: 옵션 가치 = 성공 확률 × 성공 시 가치 + (1 - 성공 확률) × 실패 시 가치
- **예시 (Weebit)**:
  - 성공 시 시총 (2030년 매출 $200M, EV/Sales 15x): $3,000M
  - 실패 시 시총 (자금 고갈, 청산): $50M
  - 성공 확률 15% 가정: 0.15 × $3,000M + 0.85 × $50M = $493M
  - 현재 시총 ~$400M → 시장은 성공 확률 ~12% 반영

#### 방법 3: 기술 마일스톤 기반 평가
- **적용 대상**: 매출 전 R&D 회사
- **개념**: 기술 진척도(시제품 → 양산 → 첫 수주 → 매출 인식)에 따라 단계적 가치 부여
- **마일스톤 예시**:
  - 시제품 검증 → 시총 $50-100M
  - 첫 양산 라인 가동 → $200-500M
  - 첫 매출 인식 → $500M-1B
  - 본격 매출 ($50M+) → $1-3B
  - 흑자 전환 → $3-10B

## 11. 9대 동인 + 9대 리스크

### 11-1. 9대 동인 (왜 ReRAM이 뜨나)

#### 동인 1: AI 추론 시장 폭발
- 2030년 AI 추론 가속기 시장 $1,000억 달러 → ReRAM CIM의 직접 무대
- 클라우드는 GPU/HBM, 엣지는 ReRAM이라는 분업 구도

#### 동인 2: eFlash 한계
- 28nm 이하 임베디드 플래시 비용 폭증 → ReRAM이 자연스러운 후계자
- 차량용 MCU 수요 증가 → ReRAM 채택 가속

#### 동인 3: CMOS 호환성
- 기존 파운드리 라인 그대로 사용 → 신규 투자 부담 적음
- TSMC·GFS·UMC가 적극 채택

#### 동인 4: 자동차 전동화 + 자율주행
- 차량 1대당 MCU 수 100개 → 200개로 증가 (전기차·ADAS)
- 차량용 임베디드 메모리 시장 폭발

#### 동인 5: 엣지 AI 디바이스 확산
- 스마트폰·웨어러블·IoT·로봇 모두 엣지 AI 채택
- 저전력 + 비휘발 → ReRAM 최적

#### 동인 6: 미·중 기술 패권
- 미국: ReRAM IP에 대규모 R&D 투자 (DARPA, NSF, IBM)
- 중국: SMIC가 Crossbar IP 사용, 자체 ReRAM 개발
- 유럽: IPCEI 프로그램으로 Weebit 등 지원

#### 동인 7: 뉴로모픽 컴퓨팅 본격화
- IBM NorthPole, Intel Loihi 2 → 뉴로모픽 칩 시장 형성
- 차세대 뉴로모픽 칩에 ReRAM 통합

#### 동인 8: MLC + 3D 적층 기술 진보
- 한 셀에 4비트 + 수직 적층 32층 → 낸드 수준 밀도 가능
- 2030+ Standalone ReRAM도 경쟁력 확보

#### 동인 9: ESG·저전력 트렌드
- 데이터센터 전력 소비 폭증 → 저전력 메모리 수요
- ReRAM은 D램 대비 1/10 수준 전력

### 11-2. 9대 리스크 (왜 실패할 수 있나)

#### 리스크 1: 변동성 문제 미해결
- Cycle-to-cycle, cell-to-cell 변동성 → 양산 수율 저하
- 만약 해결 안 되면 → 임베디드까지만 머무름

#### 리스크 2: 내구성 한계
- 현재 10^9-10^12 → D램 대체에는 부족 (10^16 필요)
- 캐시 메모리·D램 시장 진입 불가

#### 리스크 3: 경쟁 메모리 (MRAM, FRAM)
- MRAM은 더 빠르고 내구성 우수 → 임베디드 시장에서 경쟁
- FRAM은 초저전력 + 저속 → IoT 시장에서 경쟁
- ReRAM의 차별화 포인트가 약화될 수 있음

#### 리스크 4: 양산 비용
- BEOL 추가 마스크 비용 + 수율 문제 → 가격 경쟁력 미흡
- eFlash 대비 비용 우위 확보까지 시간 필요

#### 리스크 5: 표준화 부재
- ReRAM 표준이 없음 (HfO2 vs TaOx vs SiOx 등)
- 시장 분열로 규모의 경제 어려움

#### 리스크 6: 뉴로모픽 채택 지연
- AI 학습 패러다임이 GPU 중심으로 굳어짐
- ReRAM CIM이 주류 진입 못 할 가능성

#### 리스크 7: 미·중 갈등
- 중국 ReRAM 회사가 미국 IP 사용 제한받을 시 시장 축소
- 반대로 미국이 중국 ReRAM 양산 도울 경우 문제

#### 리스크 8: 자금 고갈 (스타트업)
- Weebit·4DS 등 스타트업은 매년 적자
- 추가 자금 조달 → 주식 희석 → 주가 하락

#### 리스크 9: 대형 메모리 회사 진입
- 삼성·SK하이닉스가 본격 진입 시 → 스타트업 가치 희석
- 또는 인수 → 인수 가격이 시가 대비 낮을 수 있음

### 13-7. 최종 정리 한 페이지

**ReRAM이란?** 산화막 안에 전기 물길을 만들었다 끊었다 하면서 저항으로 데이터를 저장하는 비휘발성 메모리.

**왜 중요한가?** ① 임베디드 메모리(MCU 코드) eFlash 후계자 ② AI 추론 + 메모리 인 컴퓨팅 ③ 뉴로모픽 시냅스 모방.

**시장 규모?** 2026 5.5억 달러 → 2030 35억 달러 → 2035 220억 달러.

**최대 강점?** CMOS 호환성 + 다중상태 + 저전력.

**최대 약점?** 변동성 + 양산 노하우 부족.

**Top Pick?** ① Weebit Nano (직접) ② TSMC (간접+안정) ③ 원익IPS (한국 장비 옵션).

**전략?** 포트폴리오 1-5% 옵션형 베팅, 2028+ 뉴로모픽 시대 본격 개화 시 비중 확대.
