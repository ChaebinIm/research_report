### 1.2 회사 이름의 의미 — "한미(韓美)"

**한미** = 한국 + 미국 (한미약품과 같은 이름이지만 다른 회사).

1980년 설립 당시 **"한국 + 미국 시장 동시 진출"** 비전.

### 1.3 한 줄 정체성 3가지

#### 정체성 #1: HBM TC본더 글로벌 독점 공급자

**한미반도체 = HBM 양산의 절대 필수 장비 회사**.

- HBM 메모리 = AI GPU의 핵심 부품 (NVIDIA Rubin·AMD MI400 필수)
- TC본더 = HBM 만들 때 반드시 필요한 장비
- **TC본더 = 한미반도체가 사실상 독점** (글로벌 점유율 70~90%)

비유: **AI GPU = 자동차, HBM = 자동차 엔진, TC본더 = 엔진 만드는 공장 설비**. 자동차가 폭발하면 → 엔진 공장 설비도 폭발 수혜.

#### 정체성 #2: SK하이닉스의 "슈퍼 을"

**"슈퍼 을(乙)"** = 거래에서 보통 "을(공급자)"이지만 사실상 갑(구매자)을 능가하는 위치.

- 일반: 구매자(SK하이닉스) > 공급자(한미반도체)
- 한미반도체: **SK하이닉스가 한미 장비 없으면 HBM 양산 불가**
- 사실상 한미가 SK하이닉스보다 갑

**2025년 갈등 사례:**
- 4월 SK하이닉스가 한미 인력 파견 거부
- 한미 매출 일시 감소
- 결국 SK하이닉스가 한미와 화해 시도

비유: **세계 최고 셰프(SK하이닉스)가 특수 칼(한미반도체) 없이는 요리 못 함** → 칼 회사가 갑.

#### 정체성 #3: 영업이익률 50%+ 의 압도적 마진

- 반도체 장비 평균 영업이익률 15~25%
- **한미반도체 영업이익률 50%+** (글로벌 최고 수준)
- 비교: ASML 30%, Applied Materials 25%

→ **반도체 장비 회사 중 가장 높은 마진**. 독점적 위치의 증거.

## 2. HBM이 뭔지부터 — AI 메모리 혁명

### 2.1 메모리 반도체 기본

> 🔍 **용어 박스 #1 — 메모리 vs 비메모리**
> - **메모리**: 데이터를 잠시 저장 (DRAM·NAND). SK하이닉스·삼성·마이크론.
> - **비메모리**: 데이터를 처리 (CPU·GPU·AP). NVIDIA·Intel·AMD·TSMC.
>
> 비유: **메모리 = 책상 위 공간, 비메모리 = 책상에서 일하는 사람**. 둘 다 필요.

### 2.2 HBM이란?

**HBM (High Bandwidth Memory, 고대역폭 메모리)** = AI GPU 옆에 붙여 매우 빠른 데이터 전송을 가능하게 한 차세대 DRAM.

#### 일반 DRAM vs HBM

| 항목 | 일반 DRAM | HBM |
|---|---|---|
| 구조 | 1층 칩 | **여러 층(8~16층) 수직 적층** |
| 대역폭 | 30 GB/s | **1,200 GB/s+ (40배)** |
| 크기 | 일반 | 작음 |
| 위치 | 메인보드 | **GPU 옆 직접 부착 (interposer)** |
| 가격 | 일반 | **5~10배 비쌈** |
| 용도 | PC·서버 | **AI GPU 전용** |

비유:
- 일반 DRAM = **1층 빌딩, 평지 도로**
- **HBM = 16층 빌딩, 8차선 고속도로** — 같은 면적에서 16배 정보 저장 + 40배 빠른 전송

### 2.3 HBM 시장 폭발

| 연도 | 글로벌 HBM 시장 | 비고 |
|---|---|---|
| 2023 | $5B | NVIDIA H100 시작 |

| 2025 | $35B | Blackwell 본격 |

→ **HBM = 메모리 시장 사상 최대 슈퍼사이클**.

## 3. TC본더 — HBM 양산의 결정적 장비

### 3.1 TC본더가 뭔가요?

**TC본더 (TC Bonder, Thermo-Compression Bonder, 열압착 본더)** = **여러 개의 칩을 위로 쌓아 올리고 압력과 열로 붙이는 장비**.

비유: **샌드위치를 만드는 기계** — 빵 → 햄 → 치즈 → 빵을 위에서 누르고 데워서 붙임. 다만 칩 버전.

### 3.2 HBM 제조 공정

```
[DRAM 칩 1층 만들기]
   ↓
[1층 위에 2층 쌓기] ← TC본더 사용 ⭐
   ↓
[1+2층 위에 3층 쌓기] ← TC본더
   ↓
[... 8~16층까지 반복] ← TC본더
   ↓
[GPU 옆에 부착 (Interposer)]
   ↓
[HBM 완성]
```

→ **HBM 1개당 TC본더 7~15회 사용** = TC본더 없으면 HBM 양산 불가능.

### 3.3 TC본더의 기술 난이도

#### 핵심 요구
1. **정밀도**: 마이크론(μm) 단위 정렬 (머리카락 굵기의 1/100)
2. **온도 제어**: 200~300°C 정밀 제어
3. **압력 제어**: 칩 깨지지 않도록 정밀 압력
4. **속도**: 1시간에 수백 개 칩 처리
5. **신뢰성**: 24시간 무사고 가동

#### 누가 만들 수 있나?
- **한미반도체** (한국, 70~90% 글로벌 점유) ⭐
- **한화 (한화정밀기계)** (한국, 추격 중)
- **BESI** (네덜란드, 10%)
- **ASMPT** (홍콩, 5%)

→ **한국 2사가 글로벌 90%+** = K-반도체 장비의 보석.

### 3.4 한미반도체 TC본더의 글로벌 점유

| 회사 | 글로벌 TC본더 점유 |
|---|---|
| **한미반도체** | **70~90%** ⭐ |
| 한화정밀기계 (한국) | 10~15% (SK하이닉스 듀얼 발주) |
| BESI (네덜란드) | 5~10% |
| ASMPT (홍콩) | 5% |

→ **사실상 독점 = ASML 수준의 해자**.

### 3.5 TC본더 가격

- 1대당 약 **20~50억원**
- HBM 팹 1개에 **수십~수백 대** 필요
- 한미반도체 분기 매출 = TC본더 수십~수백 대 판매

### 3.6 양산 모델 라인업

## 4. SK하이닉스와의 관계 — "슈퍼 을" 의 의미

### 4.1 SK하이닉스 = 한미반도체의 최대 고객

#### 매출 비중 (2025)
- SK하이닉스: ~40%
- 마이크론·해외: ~60%

#### 매출 비중 (2026E)
- SK하이닉스: ~34%
- 마이크론·해외: **~66%** (해외 비중 증가)

→ **단일 고객 의존도 점진 감소** = 안정성 ↑.

### 4.2 "슈퍼 을" 사건 (2025-04)

#### 배경
- SK하이닉스가 한미반도체 외 한화정밀기계에 듀얼 발주 시작
- 한미반도체 의존도 줄이려는 시도

#### 사건
- 2025-04 SK하이닉스가 한미반도체 엔지니어 파견 거부
- 한미반도체 일시 매출 충격
- **한미 반발 = 양산 차질 가능성 시그널**

#### 결과
- SK하이닉스가 한미반도체와 화해 모색
- 한화에도 발주 유지하지만 한미가 여전히 메인
- **결국 SK하이닉스가 한미를 무시할 수 없음을 입증**

비유: **세계적 셰프 식당이 특수 칼 회사와 갈등 → 결국 칼이 없어 일주일간 영업 차질 → 셰프가 화해 신청**.

## 6. 신제품 듀얼 TC본더 "그리핀"

### 6.1 그리핀(Griffin)이란?

**그리핀** = 한미반도체의 **듀얼 헤드 TC본더** (2024 출시).

- 기존: 1개 본딩 헤드로 칩 1개씩 처리
- **그리핀: 2개 본딩 헤드로 칩 2개 동시 처리** → **속도 2배**

비유: **세탁기 1통 → 2통 동시 세탁** — 처리량 2배.

### 6.2 그리핀의 의미

#### HBM 적층 가속
- HBM3E: 12층 (기존 그리핀 적용)
