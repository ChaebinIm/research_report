> **부제**: 문과생도 5시간이면 이해하는 손익계산서·재무상태표·현금흐름표
> **작성일**: 2026년 5월 11일
> **분량**: 약 80,000자
> **대상**: 회계 배경지식 0인 한국 개인 투자자
> **시리즈 위치**: 투자기초 시리즈 01 (다음 02. 밸류에이션 / 03. 분식회계)

## 0. 시작하기 전에 — 이 가이드를 읽는 법

이 가이드는 **회계라는 단어를 처음 들어보는 사람**을 위해 썼다. 손익계산서가 뭔지, 자산이 뭔지, 매출이 뭔지 하나도 몰라도 된다. 대신 끝까지 읽으면 다음을 할 수 있다.

1. 삼성전자 실적 발표 자료를 펼쳐서 "아, 영업이익이 둔화됐는데 매출은 늘었네. 그럼 마진이 깎인 거고, 원인은 메모리 가격 하락이겠구나" 정도는 1분 안에 판단.
2. 어느 회사 재무상태표를 보고 "이 회사 부채비율이 300%인데 매출은 안 늘고 있네. 이자도 못 갚을 위험이 있다"라고 직관할 수 있음.
3. 현금흐름표를 보고 "순이익은 1조인데 영업현금흐름은 -3,000억이다. 이건 매출채권이 폭발적으로 쌓인 거고, 회수 못하면 분식 의심" 같은 패턴 인식.
4. DART 사업보고서를 펼쳐서 어디부터 읽어야 할지 안다.
5. 미국 10-K를 펼쳐서 MD&A·Risk Factors·Notes의 기능을 안다.

배경지식 0을 가정하므로 "전문가는 다 아는 내용"도 자세히 풀어 쓴다. 대신 진도가 빠르므로 **표·비유·예시**를 적극 사용한다. 어렵다고 느끼면 비유 부분만 먼저 훑고 다시 돌아오라.

> **읽는 순서 추천**
> - 1독: 1~3장(서론 + 5분 입문 + 한빛카페 1년차)만 읽고 잠시 쉬기
> - 2독: 4~6장(IS·BS·CF 항목 풀이) 차근히
> - 3독: 7~10장(비율·연결·IFRS·DART) 정독
> - 4독: 11~14장(실제 종목·헷갈림·체크리스트·용어사전) 사전처럼 활용

## 1. 서론 — 왜 재무제표를 배워야 하나

### 1.1 재무제표 = 회사의 진단서

병원에 갔다고 치자. 의사가 환자를 보고 "이 환자 어디가 아픈가" 판단하려면 무엇이 필요한가? 차트, X-ray, MRI, 혈액검사 결과지. 이 모든 것이 환자의 **객관적 상태**를 숫자·이미지로 압축한 자료다.

회사도 똑같다. 삼성전자라는 거대한 환자(연간 매출 300조 원, 직원 27만 명, 사업장 70개국)의 건강 상태를 알려면 **압축된 객관 자료**가 필요하다. 그게 바로 **재무제표**다.

- 환자 차트 = 손익계산서 (1년 동안 어떻게 살았나)
- X-ray = 재무상태표 (오늘 현재 몸 상태가 어떤가)
- 혈액검사 = 현금흐름표 (혈액(현금)이 잘 돌고 있나)

**재무제표 모르고 주식 사는 것 = 차트 안 보고 약 처방하는 의사.** 단기 매매라면 차트만 보고 살 수도 있지만, 1년·3년·10년 들고 갈 종목이라면 재무제표 없이는 도박이다.

### 1.2 왜 "초등학생 눈높이"로 배워야 하나

회계는 고등학교에서 안 가르친다. 대학에서도 경영학과·회계학과 아니면 안 배운다. 그런데 주식 시장에 참여하는 한국 개인 투자자 1,400만 명 중 회계학 전공자는 5%도 안 된다. **즉 95%는 배운 적이 없다.**

문제는 회계가 **언어**라는 점. 영어를 모르고 미국 신문 읽는 것처럼, 회계를 모르면 사업보고서가 외계어처럼 보인다. "당기순이익은 전기 대비 38.5% 증가했으나, 매출채권 회전일수가 65일에서 92일로 악화되어 영업현금흐름은 -1,240억 원을 기록했다"라는 한 문장이 **무서운 경고**인지 **별것 아닌 일시적 현상**인지 구분이 안 된다.

이 가이드는 그 언어를 가르친다. 어렵게 가르치는 책은 시중에 많다. **쉽고 깊게** 가르치는 책은 드물다. 우리는 후자를 노린다.

### 1.3 모든 분석의 0순위 토대

이 가이드는 시리즈 01번이다. 후속 가이드는 다음과 같다.

| 번호 | 제목 | 역할 |
|---|---|---|
| 01 | **회계·재무제표 입문** (이 글) | 회사의 숫자를 읽는 법 |
| 02 | 밸류에이션 입문 | PER·PBR·EV/EBITDA·DCF로 가격을 매기는 법 |
| 03 | 분식회계 패턴 50선 | 거짓말하는 회사 찾는 법 |

**01 없이 02·03은 무용지물.** 손익계산서를 못 읽는데 PER이 무슨 뜻인지 알 수 없고, 매출채권을 모르면 분식회계의 가장 흔한 패턴(매출 부풀리기)을 못 잡는다.

또한 동일 디렉토리에는 100여 개의 종목·산업 분석 보고서가 있다(반도체·조선·금융·바이오 등). 그 보고서들 모두 **"이 회사 ROE는 18%, 부채비율 65%, FCF 1조 2,000억"** 같은 표현이 자주 나온다. 이 가이드를 안 읽고 그 보고서들을 읽으면 **소설을 그림책으로 보는 셈**이다.

### 1.4 이 가이드가 다루지 않는 것

- **세금 신고**: 개인 종합소득세나 법인세 신고는 회계의 한 분야지만, 투자자 입장에서 보는 회계와 다르다. 별도 책 참고.
- **회계감사 실무**: 외부감사인이 어떻게 감사하는지는 우리 관심이 아니다.
- **고급 평가 기법(스톡옵션 블랙숄즈·복합금융상품 분리 등)**: 02·03번 가이드 또는 더 깊은 책에서 다룬다.

이 가이드는 **"투자 의사결정에 필요한 회계"**만 다룬다. 그래서 80,000자다. 욕심내면 200,000자도 가능하지만, 우리는 첫 진입자를 위한 디딤돌을 만든다.

## 2. 3대 재무제표 5분 입문 (비유로 먼저)

회계의 본체는 의외로 단순하다. **3개의 표**가 전부다.

```
┌─────────────────────────────────────────────┐
│ 손익계산서 (Income Statement, IS)          │
│ "1년 동안 얼마 벌었나" 영화                 │
└─────────────────────────────────────────────┘
┌─────────────────────────────────────────────┐
│ 재무상태표 (Balance Sheet, BS)             │
│ "12월 31일 현재 재산 사진"                  │
└─────────────────────────────────────────────┘
┌─────────────────────────────────────────────┐
│ 현금흐름표 (Cash Flow Statement, CF)       │
│ "1년 동안 통장에 들어왔다 나간 영화"        │
└─────────────────────────────────────────────┘
```

### 2.1 손익계산서 = 영화

영화는 **시간의 흐름**을 보여준다. 손익계산서도 마찬가지로 **기간**(보통 1년 또는 분기)을 보여준다.

> **비유**: 카페를 1년 운영했다고 치자. 1년 동안 커피를 1억 원어치 팔았고, 원두·우유 사는 데 3,000만 원, 알바 인건비 2,000만 원, 임대료 1,500만 원, 전기·가스 500만 원 썼다. 그래서 **순이익은 3,000만 원**. 이걸 한 장의 종이에 정리한 게 손익계산서.

| 항목 | 금액 (만원) |
|---|---|
| 매출 | 10,000 |
| (-) 매출원가 (원두·우유) | -3,000 |
| **매출총이익** | **7,000** |
| (-) 판관비 (인건비·임대·전기) | -4,000 |
| **영업이익** | **3,000** |
| (-) 영업외비용 | 0 |
| (-) 법인세 | 0 (간단화) |
| **순이익** | **3,000** |

핵심: **위에서 아래로 빼나간다.** 매출에서 원가 빼면 매출총이익, 거기서 판관비 빼면 영업이익, 거기서 이자·세금 빼면 순이익.

### 2.2 재무상태표 = 사진

사진은 **한 순간**을 보여준다. 재무상태표도 마찬가지로 **특정 시점**(보통 12월 31일 자정)의 재산 상태.

> **비유**: 12월 31일 밤 12시에 카페 문을 닫고 안에 뭐가 있나 봤다. 카운터 통장에 1,500만 원 있고, 외상값 받을 거 500만 원, 원두 재고 200만 원, 인테리어·기계 5,000만 원. 합쳐서 7,200만 원이 **자산**. 이 중에 은행 빚 3,000만 원이 **부채**. 나머지 4,200만 원이 **내 돈(자본)**.

| **자산** | **부채 + 자본** |
|---|---|
| 현금 1,500 | 단기차입 3,000 |
| 매출채권 500 | **부채 합계 3,000** |
| 재고 200 | 자본금 2,000 |
| 유형자산 5,000 | 이익잉여금 2,200 |
|  | **자본 합계 4,200** |
| **자산 합계 7,200** | **부채+자본 합계 7,200** |

핵심: **왼쪽(자산) = 오른쪽(부채+자본)**. 이게 회계의 가장 중요한 등식.

### 2.3 현금흐름표 = 통장 영화

통장 거래내역을 1년 모은 영화. **순이익과 다른 이유**가 핵심.

> **비유**: 카페 1년 매출 1억인데, 그 중 1,500만 원은 외상(아직 입금 안 됨). 그러면 손익계산서엔 1억 매출 잡혀도 통장에 들어온 건 8,500만 원. 반대로 작년 외상값 500만 원이 올해 입금되면 통장엔 9,000만 원 들어온 셈. 또 인테리어 5,000만 원 1월에 한 번에 결제했지만 손익엔 5년 나눠서 1,000만 원만 비용. 통장은 5,000만 원 한 번에 빠져나갔다.

이 차이를 정리한 게 현금흐름표.

| 구분 | 내용 | 금액 (만원) |
|---|---|---|
| 영업활동 CF | 본업으로 번 현금 | +3,500 |
| 투자활동 CF | 인테리어·기계 산 돈 | -5,000 |
| 재무활동 CF | 은행 대출 받음 | +3,000 |
| **순현금 증감** |  | **+1,500** |
| 기초 현금 | 1월 1일 잔고 | 0 |
| 기말 현금 | 12월 31일 잔고 | 1,500 |

이 표 하나로 "이 회사가 본업에서 진짜로 돈을 벌고 있는지" 본질을 파악한다.

### 2.4 핵심 등식 — 이거 하나만 외우자

**자산 = 부채 + 자본**

영어로는 Assets = Liabilities + Equity. 줄여서 A = L + E.

> **왜 이게 등식이냐?**: 카페 안에 있는 모든 재산(자산)은 **빌려온 돈(부채)** 또는 **내 돈(자본)** 둘 중 하나로 사들인 거다. 다른 출처가 없다. 하늘에서 자산이 떨어지지 않는다.

이 등식이 깨지면 회계가 망가진다. 모든 거래는 **양쪽이 동시에** 움직인다. 이걸 **복식부기**라고 한다.

> **복식부기 1초 예시**:
> - 은행에서 1,000만 원 빌림 → 자산(현금) +1,000, 부채(차입) +1,000. 양쪽 동시.
> - 원두 100만 원 카드로 삼 → 자산(재고) +100, 부채(미지급) +100.
> - 커피 50만 원 팔고 현금 받음 → 자산(현금) +50, 자본(이익잉여금) +50 (매출 잡히고 순이익으로 누적).
> - 알바비 30만 원 현금 지급 → 자산(현금) -30, 자본(이익잉여금) -30 (비용 잡혀서 순이익 깎임).

**모든 거래는 양쪽 동시에. 등식은 항상 성립.** 이게 회계의 뼈대.

## 3. 가상 카페 "한빛카페" 5년 일대기

이제 본격 실전. 가상 카페 **"한빛카페"**를 5년간 따라가며 모든 회계 거래를 추적한다. 여기가 이 가이드의 **핵심**이다. 다른 장은 다 잊어도 이 장만 이해하면 80% 끝.

### 3.1 1년차 — 창업부터 첫 흑전까지

#### Q1 (1~3월): 창업

**1월 2일: 자본금 납입.** 사장 김한빛 씨가 자기 돈 5,000만 원, 친구가 3,000만 원, 부모님이 2,000만 원 — 총 1억을 모았다. "한빛커피(주)"라는 법인을 세우고 이 돈을 회사 통장에 넣었다.

| 거래 | 자산 | = | 부채 | + | 자본 |
|---|---|---|---|---|---|
| 자본금 1억 납입 | 현금 +1억 | = | (변동 없음) | + | 자본금 +1억 |

> **용어 박스**: **자본금** = 주주가 회사에 처음 넣은 돈. 회사 입장에선 "갚을 의무 없는 돈". 단, 사업이 잘 되어 회사가 청산하면 잉여 재산을 주주가 가져간다.

**1월 10일: 은행 대출.** 인테리어·기계 살 돈이 부족해서 은행에서 5,000만 원을 5년 만기, 연이자 5%로 빌림.

| 거래 | 자산 | = | 부채 | + | 자본 |
|---|---|---|---|---|---|
| 차입금 5천만 | 현금 +5,000만 | = | 장기차입금 +5,000만 | + | (변동 없음) |

**1월 15~31일: 가게 임차·인테리어·기계 구입.**
- 보증금 2,000만 원 (집주인에게 맡기는 돈, 나갈 때 돌려받음)
- 인테리어 4,000만 원 (벽지·바닥·조명·테이블·의자 등)
- 에스프레소 머신·그라인더·냉장고 3,000만 원

| 거래 | 자산 변동 |
|---|---|
| 보증금 2,000만 지급 | 현금 -2,000만 / 임차보증금 +2,000만 |
| 인테리어 4,000만 지급 | 현금 -4,000만 / 인테리어자산(유형자산) +4,000만 |
| 기계 3,000만 지급 | 현금 -3,000만 / 기계장치(유형자산) +3,000만 |

**3월 1일 오픈 직전 재무상태표:**

| 자산 | 만원 | 부채+자본 | 만원 |
|---|---|---|---|
| 현금 | 6,000 | 장기차입금 | 5,000 |
| 임차보증금 | 2,000 | 자본금 | 10,000 |
| 인테리어 | 4,000 |  |  |
| 기계장치 | 3,000 |  |  |
| **합계** | **15,000** | **합계** | **15,000** |

> **포인트**: 양쪽 합계 일치. 또 자산 1.5억 중 현금 6,000만, 나머지 9,000만은 보증금·시설로 "묶여 있는" 돈. 이걸 **유동성이 떨어진다**고 표현한다.

#### 감가상각 — 돈은 안 나가는데 비용

**감가상각**(Depreciation)이란? **장기간 쓸 자산은 돈을 한꺼번에 비용 처리하지 않고, 사용 기간에 나눠서 비용 처리한다.**

### 3.3 3년차 — 피크 (영업이익 폭증)

브랜드 인지도 상승. 3호점도 오픈. 매출이 4.5억으로 폭증.

**3년차 손익:**
- 매출: **45,000만 원**
- 영업이익: 13,500만 원 (마진 30% — 한빛카페 역대 최고)
- 순이익: 약 10,500만 원

**의사결정: 자사주? 배당?**
- 누적 이익잉여금이 18,000만 원 가까이 되어 사장이 "주주에게 환원할 돈" 고민.
- 옵션: 배당 5,000만 원 + 자사주 매입 3,000만 원
- 배당 = 회사 현금이 주주에게로 직접 → 자본 감소
- 자사주 = 회사가 자기 주식 사들임 → 자본 항목 중 "자사주" 음수로 잡힘 → 자본 감소

> **자사주의 효과**: 시중 유통 주식 수 감소 → EPS 상승 (분모 줄어듦). 즉 주가 부양 효과.
> 다만 자사주는 **소각**해야 진짜 효과. 보유만 하면 나중에 다시 팔거나 임직원 보상으로 줄 수 있어서 유통주식수 다시 늘어날 위험.

| 연차 | 매출 | 영업이익 | 영업이익률 | 순이익 | 비고 |
|---|---|---|---|---|---|
| 1년 | 13,800 | 2,776 | 20.1% | 2,055 | 창업 |
| 2년 | 27,000 | 7,100 | 26.3% | 5,280 | 2호점 |
| 3년 | 45,000 | 13,500 | 30.0% | 10,500 | 피크 |
| 4년 | 42,000 | 6,500 | 15.5% | 3,500 | 위기 |
| 5년 | 48,000 | 15,000 | 31.3% | 11,500 | 턴어라운드 |
| **누적** | **175,800** | **44,876** | **25.5%** | **32,835** | |

#### 5년간 자본 추적 — "내 돈이 어떻게 굴러갔나"

| 시점 | 자본금 | 이익잉여금 | 자사주 | 자본 합계 | 비고 |

| 0년 (창업) | 10,000 | 0 | 0 | 10,000 | |

> **투자 시사점**: 좋은 회사는 매년 자본(특히 이익잉여금)이 꾸준히 누적된다. 자본이 줄어드는 회사 = 적자 또는 과잉 배당. 자본이 정체되는 회사 = 성장이 멈춘 신호.

## 4. 손익계산서 항목별 풀이

이제 한빛카페 흐름을 알았으니, 실제 상장사 손익계산서에 나오는 모든 항목을 깊게 파보자.

### 4.1 매출 (Revenue / Sales)

> **약어**: Rev (Revenue), 한국에선 **매출액·영업수익**.

**한 줄 정의**: 회사가 본업으로 고객에게 판매한 재화·서비스의 총액(부가세 제외).

**인식 시점이 핵심.** 언제 매출로 잡아야 하나? 이게 회계의 가장 미묘하고 분쟁 많은 부분.

#### 인식 시점별 분류

| 비즈니스 | 인식 시점 |
|---|---|
| **제품 판매** (전자제품·자동차) | 인도 시점 (배송 완료) |
| **서비스 제공** (컨설팅·청소) | 서비스 완료 또는 진행률 기준 |
| **장기 공사** (조선·건설) | 진행률 기준 (매기 누적 진행률 × 총공사매출) |
| **소프트웨어 라이선스** (영구) | 인도 시점 |
| **SaaS 구독** | 기간 안분 (월 100만 12개월 = 매월 100만씩) |
| **광고** (네이버·카카오) | 광고 노출 시점 |
| **로열티** (라이선스·게임) | 계약 조건에 따라 (일시 또는 기간) |

> **IFRS 15 5단계 모델**: 2018년 1월 이후 모든 회사가 따라야 하는 매출 인식 기준.
> 1. 계약 식별
> 2. 수행의무 식별
> 3. 거래가격 산정
> 4. 거래가격 배분
> 5. 수행의무 충족 시 매출 인식
> 핵심: **고객에게 통제권이 이전되는 시점**에 매출.

#### 분식 신호

매출 인식을 부정하게 앞당기는 게 가장 흔한 분식. 사례:
- **밀어내기 (Channel stuffing)**: 분기 말에 대리점에 강제로 재고 떠넘기기. 다음 분기 반품·취소 폭증.
- **부풀리기 (Round-tripping)**: A가 B에게 100억 팔고 B가 A에게 100억 팔기. 둘 다 매출 잡지만 실제 부가가치 없음.
- **가공매출**: 존재하지 않는 거래처와 거래 위장. 한국 대우조선해양·옵티머스 등.

**투자자 체크 포인트**: 매출 증가율이 매출채권 증가율보다 현저히 낮으면 의심. 즉 매출은 천천히 늘고 외상은 폭증 → 가공매출 가능성.

### 4.2 매출원가 (Cost of Goods Sold, COGS)

> **약어**: COGS (Cost of Goods Sold). 한국에선 **매출원가**.

**한 줄 정의**: 매출을 만드는 데 직접 들어간 비용.

#### 구성

- **제조업**: 원재료비 + 직접노무비 + 제조간접비 (공장 임대·감가·전기)
- **유통업**: 매입원가 + 운송비
- **서비스업**: 직접 인건비
- **소프트웨어**: 호스팅·서버비·CSP(클라우드 서비스 제공자) 비용

> **포인트**: COGS가 매출과 함께 **변동**해야 정상. COGS가 매출보다 빨리 증가 = 마진 압박. 매출보다 천천히 증가 = 마진 개선.

#### 재고 평가법

원재료 가격이 변하면 재고 가치도 변한다. 어떻게 평가하나?
- **선입선출 (FIFO)**: 먼저 산 거 먼저 출고. 인플레이션 시기 매출원가 낮게 잡힘 → 마진 좋게 보임.
- **가중평균**: 모든 재고 평균 가격. 균형.
- **개별법**: 자동차·고가품. 각 재고별 원가 추적.
- **후입선출 (LIFO)**: 나중에 산 거 먼저 출고. K-IFRS·미국 GAAP 대부분 금지(미국 일부 허용).

> **투자 시사점**: 원자재 가격 급등 시기, FIFO 쓰는 회사는 마진 일시적으로 좋게 보일 수 있음. 가중평균은 천천히 반영. 비교 시 주의.

### 4.3 매출총이익 (Gross Profit)

> **공식**: 매출총이익 = 매출 - 매출원가
> **매출총이익률 (Gross Margin)** = 매출총이익 / 매출 × 100

**의미**: 제품 자체의 수익성. 판관비 빼기 전 1차 마진.

#### 산업별 일반적 GP 마진

| 산업 | GP 마진 | 비고 |
|---|---|---|
| 명품 (LVMH·Hermès) | 65~70% | 높은 브랜드 프리미엄 |
| 소프트웨어 (MS·Adobe) | 80~90% | 한계비용 거의 0 |
| 제약 (오리지널) | 70~85% | 특허 보호 |
| 반도체 (TSMC) | 50~55% | 자본집약 |
| 한국 메모리 (삼성·하이닉스) | 30~50% (사이클) | 가격 변동 큼 |
| 자동차 (현대·도요타) | 15~25% | 부품 외주 많음 |
| 유통 (이마트·코스트코) | 10~25% | 박리다매 |
| 조선 (HD현대중공업) | 5~15% | 후행적 원가 |
| 건설 | 5~15% | 원자재·인건비 변동 |

**투자 시사점**: 같은 산업 내에서 GP 마진이 평균보다 현저히 높으면 = 경쟁우위(브랜드·기술). 현저히 낮으면 = 경쟁력 약화 또는 진입 단계.

### 4.4 판관비 (SG&A)

> **약어**: SG&A (Selling, General & Administrative expenses). 한국에선 **판매비와 관리비**.

**한 줄 정의**: 본업 운영에 들어가지만 매출원가가 아닌 모든 비용.

#### 주요 항목

| 항목 | 설명 | 한빛카페 예 |
|---|---|---|
| 인건비 | 본사·관리·영업 직원 급여 | 알바·점장·매니저 |
| 임대료 | 사무실·매장 임차료 | 카페 임대료 |
| 광고선전비 | 마케팅·광고 | 인스타 광고 |
| 연구개발비 (R&D) | 신제품·기술 개발 | (해당 없음) |
| 감가상각비 | 비제조 자산 | 인테리어 감가 |
| 무형자산상각비 | 특허·브랜드 상각 | (해당 없음) |
| 운반비 | 배송·물류 | 배달앱 수수료 |
| 접대비·여비 | 영업 활동 | 거래처 미팅 |
| 통신비·수도광열비 | | 카페 전기·가스 |
| 보험료 | | 사업자 보험 |
| 세금과 공과 | (법인세 외 세금) | 재산세·등록세 |
| 대손상각비 | 떼인 외상값 처리 | (해당 없음) |
| 주식보상비용 | 임직원 스톡옵션·RSU | (해당 없음) |

> **R&D는 비용? 자산?**: 한국·국제 회계기준상 **연구**는 비용 / **개발**은 일정 조건 충족 시 자산화. 미국 GAAP은 거의 전부 비용. 그래서 미국 빅테크(MS·Google) 손익에서 R&D가 매출의 15~20%로 크게 보이는 반면, 한국 셀트리온은 일부 자산화로 비용이 적게 보일 수 있음.

#### 변동비 vs 고정비

- **변동비**: 매출과 함께 움직임 (운반비·결제 수수료·판매 인센티브)
- **고정비**: 매출과 무관하게 발생 (본사 인건비·임대·감가)

매출 급증 시기에는 고정비 분산 효과로 이익이 폭발. 매출 급감 시기에는 고정비가 부담 — **운영레버리지(operating leverage)**.

### 4.5 영업이익 (Operating Income / EBIT)

> **공식**: 영업이익 = 매출총이익 - 판관비 = 매출 - 매출원가 - 판관비
> **약어**: EBIT (Earnings Before Interest and Taxes), OP (Operating Profit), OI

**한 줄 정의**: 본업으로 번 이익. 이자·세금 빼기 전.

**왜 중요한가?** 회사의 **핵심 수익력**을 보여준다. 이자·세금은 자본구조와 세제에 따라 변하니, 회사 본질을 보려면 영업이익이 가장 깨끗.

#### 영업이익률 (Operating Margin, OPM)

> **공식**: OPM = 영업이익 / 매출 × 100

**산업별 일반 수준** (참고):
- 명품·SW: 25~35%
- 메모리 호황기: 40~50% / 불황기: -10~10%
- 자동차: 5~10%
- 유통: 2~6%
- 조선·건설: 3~10%

> **삼성전자 사례**: 2024년 매출 약 300조, 영업이익 약 32조 → OPM 약 10.7%. 메모리 회복기여서 평소보다 좋은 수준. 2023년 메모리 침체기엔 OPM 2~4%까지 하락.

### 4.6 EBITDA — 감가상각 빼기 전 영업이익

> **약어**: EBITDA = Earnings Before Interest, Taxes, Depreciation, and Amortization
> 발음: "이비ㅌ다(또는 이빗다·이빗따)"

**공식**: EBITDA = 영업이익 + 감가상각비 + 무형자산상각비

**왜 별도로 보나?**
- 감가상각·무형상각은 **현금이 안 나가는 비용**. 그래서 "현금 기준 본업 이익"을 보려면 빼야.
- 자본집약적 산업(통신·항공·반도체·조선) 평가에 자주 씀.
- M&A 가격 평가의 표준(EV/EBITDA Multiple).

**한빛카페 1년차 EBITDA**:
- 영업이익 2,776 + 감가상각 1,024 = **3,800만 원**
- EBITDA 마진 = 3,800/13,800 = 27.5%

> **EBITDA의 함정**: 워런 버핏은 EBITDA를 싫어한다. "감가상각이 진짜 비용이 아니라고 누가 그랬나? 결국 그 자산을 다시 사야 한다." 즉 EBITDA만 보면 자본집약 회사의 진짜 비용을 놓칠 수 있다. 보조 지표로 쓰되 영업이익도 같이.

### 4.7 영업외이익·비용 (Non-operating Income/Expense)

본업과 무관한 손익.

#### 주요 항목

| 항목 | 설명 |
|---|---|
| **이자수익·이자비용** | 예금 이자 / 차입금 이자 |
| 배당금수익 | 보유 주식 배당 |
| **외환차익·차손** | 환율 변동으로 인한 손익 |
| 외화환산이익·손실 | 결산 시 외화 자산·부채 환산 |
| **유형자산처분이익·손실** | 부동산·기계 매각 차익 |
| **지분법이익·손실** | 관계회사 지분 평가 |
| **금융자산평가이익·손실** | 보유 주식·채권 평가 |
| 손상차손 | 자산 가치 회복 불가 시 |
| 충당부채 환입·전입 | 미래 손실 추정 변동 |
| **기타** | 일회성 손익 다수 |

> **포인트**: 영업외 항목은 **변동성이 크다**. 환율 1원만 변해도 외환차손익 수십억 발생. 그래서 투자자는 "본업 영업이익 vs 영업외 손익"을 분리해서 본다.

#### 이자비용 (Interest Expense)

> **공식**: 이자비용 = 차입금 평균 잔액 × 평균 이자율

**투자 시사점**: 이자비용이 영업이익에 비해 너무 크면 = 부채 과다 위험. **이자보상배율 = 영업이익/이자비용**. 이게 1배 미만이면 본업 이익으로 이자도 못 갚는 위험 회사. 5배 이상이면 안정.

### 4.8 세전이익·법인세·순이익

#### 세전이익 (EBT, Earnings Before Tax)
- = 영업이익 + 영업외이익 - 영업외비용

#### 법인세 (Income Tax)
- 한국 법인세율: 과세표준 2억 이하 9%, 2~200억 19%, 200~3,000억 21%, 3,000억 초과 24% (지방세 포함 대기업 약 27.5%)
- 미국: 연방 21% + 주마다 다름 (평균 약 25%)
- 실효세율 = 법인세 / 세전이익. 보통 한국 대기업 22~25%, 미국 빅테크 12~18%(해외 절세 구조).

#### 순이익 (Net Income, NI)

> **공식**: 순이익 = 세전이익 - 법인세

**한 줄 정의**: 모든 비용·세금을 다 빼고 주주에게 귀속될 최종 이익.

**EPS (Earnings Per Share, 주당순이익)** = 순이익 / 평균 발행주식수

**한빛카페 1년차 EPS**: 만약 주식 1,000주 발행했다면 EPS = 2,055만/1,000 = **2.055만 원/주**.

상장사는 EPS가 가장 중요한 지표 중 하나. 시장은 EPS 성장률을 보고 주가를 매김.

### 4.9 영업이익률 vs 순이익률

| 지표 | 공식 | 의미 |
|---|---|---|
| 매출총이익률 | 매출총이익/매출 | 제품 자체 수익성 |
| 영업이익률 (OPM) | 영업이익/매출 | 본업 운영 효율 |
| 순이익률 (NPM) | 순이익/매출 | 최종 주주 귀속 비율 |

**OPM과 NPM의 차이가 크다 = 영업외 손익이 크다**. 환율·이자·일회성 비용·세금 부담 등.

## 5. 재무상태표 항목별 풀이

이제 사진(BS) 각 항목을 깊이 보자. 자산은 위에서 아래로 **유동성 순서**(현금화 빠른 것 → 느린 것)로 배열.

### 5.1 자산 — 유동자산

#### 5.1.1 현금 및 현금성자산 (Cash and Cash Equivalents)

- 통장 잔액
- MMF·CD 등 3개월 이내 만기 단기 금융상품
- **가장 유동성 높은 자산**

> **투자 시사점**: 회사가 현금이 너무 많으면? 좋은 일이지만 **자본효율성 저하** 신호일 수도. 애플은 보유 현금 2,000억 달러 중 일부를 자사주 매입에 쓰고 있다. 한국 SK텔레콤·KT&G도 비슷.
> 반대로 현금이 너무 적으면? 단기 자금경색 위험. 매출채권 회수 지연·매입채무 결제 일정 따라 돈이 막힐 수 있음.

#### 5.1.2 단기금융상품·단기투자자산

- 만기 3개월~1년 예금·채권
- 단기적으로 운용 중인 여유자금

#### 5.1.3 매출채권 (Accounts Receivable, AR)

> **한 줄 정의**: 고객에게 물건·서비스 팔고 아직 못 받은 돈(외상값).

> **DSO (Days Sales Outstanding, 매출채권 회전일수)** = (매출채권 / 매출) × 365
> 한빛카페 1년차: (1,200/13,800) × 365 = 약 32일. 즉 평균 32일 만에 외상이 회수된다.

**산업별 DSO 일반 수준**:
- 식음료·소비재 (현금 매출 많음): 10~30일
- B2B 제조업: 60~90일
- 건설·조선: 100~300일 (장기 프로젝트 정산)
- 광고·미디어: 60~120일

> **분식 신호**: DSO가 갑자기 늘면 위험. 매출은 잡았는데 돈이 안 들어옴 = 가공매출 또는 매출 부풀리기. 한국 옵티머스·라임 사태 등 다수 사례.

**대손충당금**: 받지 못할 가능성이 있는 외상값에 대해 미리 충당금을 잡는다. 대손율은 산업·고객 신용도에 따라 다름.

#### 5.1.4 재고자산 (Inventory)

- 원재료 + 재공품(작업 중) + 제품 + 상품(매입 후 미판매)

> **DIO (Days Inventory Outstanding, 재고자산 회전일수)** = (재고 / 매출원가) × 365

**산업별 DIO**:
- 패션 (계절성): 90~180일
- 가전·전자: 30~90일
- 식품 (신선): 7~30일
- 자동차: 30~60일
- 반도체: 60~150일 (사이클 영향)

> **재고 급증 시 의심해야 할 것**:
> 1. 판매 부진 (안 팔리는 상품 쌓임)
> 2. 미래 수요 예측 (선제 비축 — 의도적이면 OK)
> 3. 분식 (매출원가 안 잡고 재고로 남겨 이익 부풀리기)

**평가손실 (재고자산평가손실)**: 재고가치가 시장가보다 낮아지면 차이만큼 손실. 한국 메모리 반도체 회사가 가격 폭락 시기에 자주 인식.

#### 5.1.5 기타 유동자산

- 선급비용 (미리 낸 임대료·보험료)
- 미수수익 (받을 이자·배당 등)
- 단기대여금
- 부가세 환급채권 등

### 5.2 자산 — 비유동자산

#### 5.2.1 유형자산 (Property, Plant and Equipment, PP&E)

- 토지·건물·기계장치·차량·집기·인테리어
- **감가상각 대상** (토지 제외)

| 자산 | 일반 내용연수 |
|---|---|
| 토지 | 무한 (감가 안 함) |
| 건물 | 30~50년 |
| 기계장치 | 5~15년 |
| 차량 | 5~7년 |
| 집기·비품 | 3~5년 |
| 컴퓨터·IT장비 | 3~5년 |

**감가상각 방법**:
- **정액법**: 매년 동일 금액 (4,000만/5년 = 매년 800만) — 가장 흔함
- **정률법**: 잔존가치 × 일정 비율. 초기 많이 상각, 후기 적게.
- **생산량비례법**: 사용량(생산량)에 비례. 광산·유전 등.

#### 5.2.2 무형자산 (Intangible Assets)

- 특허·상표·소프트웨어·라이선스·고객관계
- **상각 대상** (영업권 제외)

> **포인트**: K-IFRS는 영업권을 매년 상각하지 않고 **손상검사**만 한다. 미국 GAAP도 동일.
> 일반 무형자산은 내용연수에 따라 상각.

#### 5.2.3 영업권 (Goodwill) — M&A의 흔적

> **공식**: 영업권 = 인수가격 - 인수한 회사 순자산 공정가치

**예시**: A회사가 B회사를 1,000억에 인수. B회사 순자산(자산-부채) 공정가치 600억. 그러면 영업권 400억 = "B회사의 브랜드·고객·기술 등 측정 안 된 가치를 A가 인정한 프리미엄".

> **투자 시사점**:
> - 영업권이 자산의 30% 이상 = M&A 의존 성장 모델. 신중히.
> - 영업권 손상차손 발생 = 그 M&A가 실패했다는 자백. 큰 일회성 손실.
> - 한국 사례: SK하이닉스가 키옥시아 지분 인수 후 손상차손 잡은 적 있음. 한진해운·STX 등 M&A 실패 사례 다수.

#### 5.2.4 관계회사·종속회사 투자

- 지배지분(50% 초과) → 종속회사 → 연결재무제표 합산
- 유의적 영향력(20~50%) → 관계회사 → 지분법 평가
- 단순 투자(20% 미만) → 금융자산 분류

#### 5.2.5 장기금융자산·이연법인세자산 등

- 만기 1년 초과 채권·예금
- 이연법인세자산: 일시적 세무차이로 미래 환급 예상

### 5.3 부채 — 유동부채

> **유동부채**: 1년 이내 갚아야 할 빚.

#### 5.3.1 단기차입금 (Short-term Borrowings)
- 1년 이내 만기 은행 대출
- 회전식 운영 자금

#### 5.3.2 매입채무 (Accounts Payable, AP)
- 원재료·상품 매입하고 아직 안 갚은 외상

> **DPO (Days Payable Outstanding, 매입채무 회전일수)** = (매입채무 / 매출원가) × 365
> 한빛카페 1년차 DPO = (200/4,140) × 365 = 약 18일

**투자 시사점**: DPO 길수록 회사가 협상력 강함(거래처에 결제 미루는 힘). 애플·월마트 DPO 60~90일. 작은 회사는 30일 이내가 일반.

#### 5.3.3 미지급금·미지급비용

- 미지급금: 비영업적 채무 (자산 매입 후 미결제)
- 미지급비용: 발생했지만 아직 결제 안 한 비용 (미지급 임대료·이자·인건비)

#### 5.3.4 미지급법인세

- 결산 시 인식한 법인세 중 아직 납부 안 한 부분

#### 5.3.5 단기충당부채

- 1년 내 발생할 가능성 높은 손실 (제품 보증·소송 등) 미리 적립

#### 5.3.6 유동성장기부채

- 장기차입금 중 1년 이내 만기 도래 부분

### 5.4 부채 — 비유동부채

#### 5.4.1 장기차입금·사채 (Long-term Debt, Bonds)
- 만기 1년 초과 부채
- 사채(社債)는 회사가 발행한 채권 (회사채)

> **신용등급별 금리** (2026년 5월 기준 추정):
> - AA+ ~ AAA (삼성전자급): 국채+30~50bp
> - A ~ AA (대기업 일반): 국채+50~150bp
> - BBB (중견): 국채+150~300bp
> - BB 이하: 국채+300bp~ (정크본드)

#### 5.4.2 장기충당부채

- 퇴직급여충당부채 (직원 퇴직금 누적)
- 장기소송충당부채
- 환경복구충당부채 (광산·정유)

#### 5.4.3 이연법인세부채

- 회계상 이익 > 세무상 이익일 때 미래에 낼 세금

### 5.5 자본 (Equity) 항목별

#### 5.5.1 자본금 (Paid-in Capital, Common Stock)

- 발행주식수 × 액면가
- 한국은 액면가 5,000원·500원·100원 등. 상장사는 보통 5,000원.

#### 5.5.2 자본잉여금 (Additional Paid-in Capital, APIC)

- 발행가 - 액면가 = 주식발행초과금
- 예: 액면 5,000원짜리 주식을 50,000원에 발행하면 차액 45,000원이 자본잉여금.

#### 5.5.3 이익잉여금 (Retained Earnings, RE)

- 회사 설립 이후 누적된 순이익에서 배당으로 나간 것 뺀 잔액
- **회사가 진짜로 번 돈이 쌓인 항목**

#### 5.5.4 자기주식 (Treasury Stock)

- 회사가 자기 회사 주식을 산 것
- 자본 항목에 **음수**로 잡힘 (자본 감소)
- 소각하면 자본금·이익잉여금·자본잉여금 등이 감소

> **자사주의 의미**:
> - 매입 = 주가 부양·EPS 상승
> - 보유 = 임직원 보상 등에 활용 가능
> - 소각 = 영구적 유통주식 감소 → 영구 EPS 상승

#### 5.5.5 기타포괄손익누계액 (Accumulated Other Comprehensive Income, AOCI)

- 손익으로 안 잡고 자본에 직접 반영되는 항목
- 매도가능금융자산평가손익, 해외사업환산손익 등

#### 5.5.6 비지배지분 (Non-controlling Interests, NCI)

- 연결 시 자회사의 외부 주주(소수 주주) 지분

### 5.6 K-IFRS vs 미국 GAAP — 핵심 차이

| 항목 | K-IFRS | 미국 GAAP |
|---|---|---|
| 재고 평가 | LIFO 금지 | LIFO 허용 |
| 재평가 | 유형자산 재평가 허용 | 원가법만 |
| 영업권 | 매년 손상검사 | 매년 손상검사 (동일) |
| 개발비 | 조건부 자산화 | 거의 전부 비용 |
| 리스 | IFRS 16 — 운용리스도 자산·부채 | ASC 842 — 유사 |
| 기타포괄손익 항목 | 일부 차이 | 일부 차이 |

> **포인트**: 미국 상장사·한국 상장사 **직접 비교 시 주의**. 동일 회사라도 IFRS vs GAAP에 따라 영업이익·자산이 다르게 표시될 수 있음.

## 6. 현금흐름표 항목별 풀이

### 6.1 영업활동 현금흐름 (Cash Flow from Operations, CFO)

> **한 줄 정의**: 본업으로 들어오고 나간 현금.

#### 직접법 vs 간접법

**직접법**: 매출에서 받은 현금, 매입에 쓴 현금, 인건비에 쓴 현금 등을 직접 보여줌. 이해 쉬우나 작성 어려움.

**간접법**: 순이익에서 출발 → 비현금 항목·운전자본 변동 조정. 대부분 회사가 이 방식.

#### 간접법 구성

| 항목 | 의미 |
|---|---|
| 순이익 | 손익계산서 최종 |
| (+) 비현금 비용 | 감가상각·무형상각·대손상각·주식보상비용·이연법인세 |
| (+/-) 운전자본 변동 | 매출채권↑(-)·재고↑(-)·매입채무↑(+)·미지급비용↑(+) |
| (+/-) 영업외 항목 조정 | 외환차손익·자산처분손익 등 |
| (-) 법인세 납부액 | (영업CF에서 차감) |
| (-) 이자 지급액 | (영업CF에서 차감 — 한국 IFRS 보통) |
| **= 영업CF** | |

> **운전자본 (Working Capital)**: 매출채권 + 재고 - 매입채무. 본업 운영에 묶이는 돈.
> - 매출채권 증가 = 외상이 늘어 현금 안 들어옴 → CF 마이너스
> - 재고 증가 = 창고에 돈 묶임 → CF 마이너스
> - 매입채무 증가 = 결제 미뤘으니 현금 남음 → CF 플러스

> **CCC (Cash Conversion Cycle, 현금화 사이클)** = DSO + DIO - DPO
> 짧을수록 좋음. 음수면 최고(매출채권보다 매입채무가 길어 협상력 강함). 애플·월마트가 음수.

### 6.2 투자활동 현금흐름 (Cash Flow from Investing, CFI)

#### 주요 항목

- **CapEx (Capital Expenditure)**: 유형자산·무형자산 취득. **투자 마이너스**.
- 자산 처분: 부동산·기계 매각. 플러스.
- M&A: 자회사 인수. 마이너스.
- 금융자산 매수·매각.

> **투자 시사점**:
> - CapEx > 감가상각 = 사업 확장 단계. 미래 매출 늘어날 가능성.
> - CapEx < 감가상각 = 유지보수만 하는 단계. 정체 또는 쇠퇴.
> - SK하이닉스·삼성전자는 매년 CapEx 30~50조 — 자본집약적.

### 6.3 재무활동 현금흐름 (Cash Flow from Financing, CFF)

#### 주요 항목

| 항목 | 의미 | CF 방향 |
|---|---|---|
| 차입금 차입 | 은행·사채 차입 | + |
| 차입금 상환 | 차입금 갚음 | - |
| 유상증자 | 신주 발행 | + |
| 자사주 매입 | | - |
| 자사주 처분 | | + |
| 배당 지급 | | - |
| 리스부채 상환 | | - |

### 6.4 순이익 vs 영업CF — 왜 다른가

이 차이가 회계의 가장 중요한 통찰.

**한빛카페 1년차**:
- 순이익 2,055만
- 영업CF 2,259만
- 차이 +204

**원인 분해**:
- (+) 감가상각 1,024 (현금 안 나간 비용)
- (+) 매입채무 200 (현금 안 빠진 비용)
- (+) 미지급법인세 380 (현금 안 빠진 비용)
- (-) 매출채권 -1,200 (현금 안 들어온 매출)
- (-) 재고자산 -200 (현금 묶임)

**합계 +204**. 손익은 "발생주의(accrual)"로 잡히고 현금은 "현금주의(cash)"로 잡히는 차이.

> **핵심 통찰**:
> - **순이익 > 영업CF가 지속** = 매출은 잡혔지만 현금 안 들어옴 → 분식 위험
> - **영업CF > 순이익** = 본업이 견실, 운전자본 효율 좋음 → 좋은 신호

### 6.5 FCF (Free Cash Flow, 잉여현금흐름)

> **공식**: FCF = 영업CF - CapEx
> (또는 더 엄격하게: 영업CF - 자본적 지출 - 운전자본 증가)

**의미**: 본업으로 번 현금 중 사업 유지·확장에 쓰고 남은 것. **주주에게 환원할 수 있는 진짜 돈**.

> **주식의 본질적 가치 = 미래 FCF의 현재가치 합** (DCF 모델 — 02번 가이드에서 다룸)

**산업별 FCF 특성**:
- 자본 가벼운 SW (MS·Adobe): 매출의 30~40%가 FCF
- 자본 무거운 통신·반도체: FCF 변동 큼, 호황기에만 양수
- 인프라·항공: FCF 만성 적자도 있음

#### FCF 응용 — FCF 수익률 (FCF Yield)
- = FCF / 시가총액
- 5% 이상이면 매력적인 수준 (단, 산업 비교 필수)

## 7. 재무비율 풀이 — 핵심 12가지

### 7.1 수익성 비율 (Profitability)

#### ROE (Return on Equity, 자기자본이익률)
> **공식**: ROE = 순이익 / 평균 자기자본 × 100

**한국 우량기업 기준**:
- 15% 이상 = 우수
- 10~15% = 양호
- 5~10% = 평이
- 5% 미만 = 부실 또는 자본 비효율

**워런 버핏 기준**: 장기 ROE 15% 이상 + 부채 적은 회사 선호.

> **한빛카페 5년차 ROE**: 11,500/30,000(평균자본 추정) ≈ 38%. 작은 사업이라 매우 높음. 대기업은 10~20%면 우수.

#### ROA (Return on Assets, 총자산이익률)
> **공식**: ROA = 순이익 / 평균 총자산 × 100

ROE보다 보수적. 부채로 자본 레버리지 키우는 효과를 제외.

#### ROIC (Return on Invested Capital, 투하자본이익률)
> **공식**: ROIC = NOPAT / (자기자본 + 유이자부채) × 100
> NOPAT = 영업이익 × (1 - 세율)

**의미**: 회사에 들어간 모든 자본(자기 + 빌린 돈) 대비 영업이익 효율.

**중요성**: ROIC > WACC(가중평균자본비용) → 가치 창출. 그 반대 → 가치 파괴.

#### OPM·NPM·매출총이익률 — 앞서 다룸

### 7.2 효율성 비율 (Efficiency)

#### 자산회전율 (Asset Turnover)
> **공식**: 자산회전율 = 매출 / 평균 총자산

산업별 차이 큼. 유통업(이마트) 2~3배, 자본집약 통신·반도체 0.3~0.7배.

#### 재고회전율 (Inventory Turnover)
> **공식**: 재고회전율 = 매출원가 / 평균 재고

높을수록 빠른 재고 회전 = 효율 좋음. 산업별 차이 큼.

#### 매출채권회전율 (AR Turnover)
> **공식**: 매출 / 평균 매출채권

높을수록 외상 빨리 회수 = 좋음.

### 7.3 안정성 비율 (Solvency)

#### 부채비율 (Debt-to-Equity Ratio)
> **공식**: 부채비율 = 총부채 / 자기자본 × 100

**한국 일반 기준**:
- 100% 미만 = 안정
- 100~200% = 보통
- 200~400% = 주의
- 400% 초과 = 위험

> **단, 산업별 다름**: 금융·건설·항공은 본질적으로 부채비율 높음. 동종 비교 필수.

#### 유동비율 (Current Ratio)
> **공식**: 유동비율 = 유동자산 / 유동부채 × 100

150% 이상이 일반적 안정 기준. 100% 미만 = 단기 자금경색 위험.

#### 당좌비율 (Quick Ratio)
> **공식**: 당좌비율 = (유동자산 - 재고) / 유동부채 × 100

재고는 빨리 안 팔릴 수 있으므로 제외. 100% 이상이 안정.

#### 이자보상배율 (Interest Coverage Ratio)
> **공식**: 영업이익 / 이자비용

- 5배 이상 = 안정
- 1~5배 = 위험 가능
- 1배 미만 = **본업 이익으로 이자도 못 갚음 → 좀비기업**

### 7.4 성장성 비율 (Growth)

#### 매출 성장률 (Revenue Growth)
- YoY (Year-on-Year, 전년 동기 대비)
- QoQ (Quarter-on-Quarter, 전 분기 대비)
- 3년·5년 CAGR (연평균 성장률)

#### EPS 성장률
- 주당순이익이 얼마나 빨리 성장하나
- 자사주 매입으로 주식수 줄면 EPS 성장 가속

### 7.5 DuPont 분해 — ROE를 3등분

> **DuPont 공식**:
> ROE = (순이익/매출) × (매출/총자산) × (총자산/자본)
>     = 순이익률 × 자산회전율 × 레버리지

**의미**:
- ROE는 **마진·효율·레버리지**의 곱.
- 어디서 ROE가 오는지 분석.

#### 한빛카페·삼성전자·SK하이닉스 비교 (가상 예시)

| 회사 | 순이익률 | 자산회전율 | 레버리지 | ROE |
|---|---|---|---|---|
| 한빛카페 5년차 | 24% | 1.3 | 1.4 | 약 44% |
| 삼성전자 (좋은 해) | 12% | 0.7 | 1.7 | 약 14% |
| SK하이닉스 (호황기) | 28% | 0.5 | 1.8 | 약 25% |
| SK하이닉스 (불황기) | -10% | 0.4 | 2.0 | -8% |
| 명품 LVMH | 16% | 0.5 | 2.5 | 약 20% |
| 월마트 | 2.5% | 2.5 | 2.5 | 약 16% |

**통찰**:
- 같은 ROE 16%라도 명품(고마진+낮은 회전+레버리지)과 월마트(저마진+빠른 회전+레버리지)는 비즈니스 모델이 정반대.
- ROE만 보면 안 되고 어디서 왔는지 봐야.

## 8. 연결재무제표 vs 별도재무제표

### 8.1 개념

- **별도재무제표**: 한 회사 단독의 재무제표.
- **연결재무제표**: 모회사 + 자회사들의 재무를 합산.

### 8.2 한빛카페 시나리오

3년차에 한빛카페가 2호점을 분사해 "한빛커피2호점(주)"를 만들고 본사가 80% 지분 보유.

**별도재무제표 (한빛카페 본사 단독)**:
- 자산에 "관계회사 투자" 1,500만 (2호점 지분)
- 손익엔 본사 매출만

**연결재무제표 (한빛카페 + 2호점 합산)**:
- 매출·자산·부채 모두 합산
- 2호점의 비지배지분(20%)을 자본에 별도 표시
- 내부 거래(본사가 2호점에 원두 판매 등)는 상계 제거

> **투자자 입장**: 보통 **연결재무제표를 보는 게 정답**. 그게 그룹 전체의 진짜 모습. 단, 한국 일부 분석은 별도도 함께 본다 (배당가능이익 등).

### 8.3 지분율별 분류

| 지분율 | 관계 | 회계 처리 |
|---|---|---|
| 50% 초과 | 종속회사 | 연결 (모든 항목 합산 + 비지배지분) |
| 20~50% | 관계회사 | 지분법 (자산에 투자평가, 손익에 지분법손익) |
| 20% 미만 | 단순투자 | 금융자산 (시가 평가 또는 원가) |

## 9. K-IFRS 핵심 변화 5가지 (간단 부록)

### 9.1 IFRS 15 — 매출 인식 (2018년 1월~)

5단계 모델로 매출 인식 통일. 주요 영향:
- 통신사 결합상품 매출 분배 변경
- 건설사 진행률 인식 강화
- SW 라이선스+유지보수 분리 인식

### 9.2 IFRS 16 — 리스 (2019년 1월~)

**가장 큰 변화**: 운용리스도 **사용권자산·리스부채로 BS에 인식**.

**예시**: 항공사가 비행기를 운용리스로 빌려 쓰면 예전엔 매월 임차료만 비용. 이제는 자산·부채에 잡힘.

**영향**:
- 부채비율 상승 (외관상)
- EBITDA 증가 (감가·이자로 분리되므로)
- ROA 변동

### 9.3 IFRS 17 — 보험 (2023년 1월~)

보험 부채 측정 방식 전면 개편. **CSM (Contractual Service Margin, 보험계약서비스마진)** 등장.
- 미래 이익을 부채로 잡고 기간에 따라 인식
- 한국 보험사(삼성생명·한화생명) 실적·자본 큰 변동

### 9.4 IFRS 9 — 금융상품 (2018년 1월~)

- 금융자산 분류 단순화 (3개 범주)
- 손상은 **예상신용손실(ECL)** 모델로 — 발생 전에 미리 충당
- 은행·금융사 영향 큼

### 9.5 IFRS 3 — 사업결합 (영업권)

- 인수가격 vs 순자산 공정가치 차이 = 영업권
- 영업권 매년 상각 안 함, 손상검사만

### 9.6 IFRS 18 — 재무제표 표시 (2027년 1월~)

> **2026년 5월 현재 가장 임박한 변화.**

기존 IAS 1을 대체하는 새 기준. 주요 내용:
- **영업이익(Operating Profit)을 정의된 소계로 의무 표시**
- **금융수익·비용 전 이익(Profit before financing and income taxes)** 별도 표시
- 손익계산서 항목을 **영업·투자·재무 활동 3가지 카테고리**로 분류
- **MPM (Management-defined Performance Measures)** 공시 의무화 — 회사가 발표하는 "조정 영업이익" 등에 대한 정의·조정 내역 공개
- 손익 표 내 분류 일관성·집계 표시 강화

**투자자에게 미치는 영향**:
- 영업이익 비교 가능성 향상
- 회사별로 "non-GAAP" 지표 임의 사용 줄어듬
- 글로벌 비교가 더 깔끔해짐

> **한국 적용**: K-IFRS도 2027년 1월 1일 이후 시작 사업연도부터 동일 적용 예정. 2027년 1분기 보고서부터 새 양식.

### 9.7 한국 가상자산 회계 (2024~ 시범)

암호화폐 보유·발행에 대한 K-IFRS 해석 확립 진행 중. 가상자산을 **무형자산** 또는 **재고자산**으로 분류. 가상자산거래소(빗썸·업비트)가 직접 영향.

### 11.1 삼성전자 1Q26 손익·재무상태·현금흐름 풀이 (가상 시나리오)

**가정 데이터** (2026년 1분기, 추정):

### 11.2 SK하이닉스 ROE 분해

### 11.3 HD현대중공업 — 잔고와 매출 인식

**조선업의 특수성**: LNG선·컨테이너선은 건조에 2~3년. 매출은 **진행률 기준** 인식.

- 수주잔고 (Backlog): 1,500억 달러 (3년치 일감)
- 매출은 매년 진행률에 따라 인식
- 원자재 가격 변동·환율 변동에 따라 **공사손실충당금** 잡힐 위험

**투자 포인트**:
- 잔고가 매출의 3배 이상이면 향후 3년 가시성 좋음
- 단, 수주 시점 환율(원/달러)와 인도 시점 환율 차이로 헤지 효과 중요

### 11.4 셀트리온 — 연구개발비 자산화 vs 비용화

바이오시밀러·신약 개발은 R&D 비용이 매출의 20~30% 수준.
- K-IFRS에선 **개발비** 일정 조건 충족 시 자산화 가능 → 무형자산
- 미국 GAAP에선 거의 전부 비용

**셀트리온 사례**: 자산화한 개발비가 무형자산에 수천억 누적. 만약 개발 실패하면 **손상차손**으로 일시 비용화 → 큰 손실.

**투자 시사점**: 바이오 회사 평가 시 자산화 정책 확인. 자산화 비율 높으면 손익이 좋아 보이지만 미래 손상 위험.

### 11.5 카카오 — 주식보상비용·자사주

**주식보상비용 (Stock-based Compensation, SBC)**: 임직원에게 스톡옵션·RSU 지급. 손익에 비용 잡히지만 **현금 안 나감** → 영업CF 가산.

카카오 같은 IT 기업은 SBC가 매출의 5~15% 수준. 많을수록:
- 손익 감소 (실제로는 비현금)
- 향후 행사 시 **유통주식수 증가 → EPS 희석**
- 이를 막으려 자사주 매입 동시 진행 → 현금 유출

**투자 시사점**: SBC를 빼고 보는 "Adjusted EPS"가 너무 낙관적일 수 있음. 희석 효과까지 봐야.

## 12. 자주 헷갈리는 5가지

### 12.1 매출 인식 시점 (제품·서비스·구독)

| 비즈니스 | 인식 |
|---|---|
| 제품 (TV·자동차) | 인도 시점 |
| 서비스 (컨설팅) | 진행률 |
| 장기공사 (조선) | 진행률 |
| SW 영구 라이선스 | 인도 시점 |
| SaaS (클라우드 SW) | 기간 안분 |
| 광고 (네이버) | 노출 시점 |

### 12.2 감가상각 vs CapEx

- CapEx = 자산 매입 (현금 유출, BS에 자산 잡힘)
- 감가상각 = 자산을 사용 기간에 비용으로 나눔 (현금 안 나감, IS에 비용 잡힘)
- 둘은 별개. 한꺼번에 일어나기도, 시점이 다르기도.

**예시**: 1월 1억 짜리 기계 구입. CapEx 1억 (CFI에 -1억). 감가는 5년 나눠서 매년 2,000만씩 IS 비용.

### 12.3 매출채권 회전 (외상의 함정)

매출 100, 매출채권 30 → DSO = 110일. 매출이 50% 늘어 150이 되었는데 매출채권 80 → DSO = 195일. **회수가 늦어지고 있다는 신호**. 부도 위험·분식 위험 모두 의심.

### 12.4 일회성 비용 vs 반복 비용

회사는 본업 손익이 안 좋을 때 "일회성 비용"이라고 표현 — 하지만:
- 같은 일회성이 매년 발생 = 사실상 반복
- "구조조정 비용"이 5년 연속 → 실은 만성 적자

**투자자 체크**: 5년 손익에서 "일회성"이라는 항목들을 다 합쳐서 평균 내면 진짜 본업 이익이 나옴.

### 12.5 자사주 매입·소각 효과

- **매입만**: 자사주가 BS 자본 음수 항목으로. EPS 일시 상승. 그러나 다시 처분하면 효과 사라짐.
- **소각**: 자본금·자본잉여금·이익잉여금 감소. 영구적 유통주식 감소 → EPS 영구 상승.

> **삼성전자 사례**: 2024년 10조 자사주 매입 발표. 그 중 절반은 소각. → 영구 EPS 상승 효과.

## 13. 체크리스트 — 1분 만에 회사 진단 10항목

종목 보고서 펴기 전에 다음 10가지를 체크하라.

| # | 체크 항목 | 정상 기준 | 위험 신호 |
|---|---|---|---|
| 1 | **매출 성장률** | 5년 연속 양수 | 2~3년 감소 |
| 2 | **영업이익률 (OPM)** | 동종 평균 이상 | 2년 연속 감소 |
| 3 | **ROE** | 10% 이상 | 5% 미만 또는 음수 |
| 4 | **부채비율** | 200% 미만 | 400% 이상 |
| 5 | **이자보상배율** | 3배 이상 | 1배 미만 |
| 6 | **DSO** | 동종 수준 | 갑자기 늘어남 |
| 7 | **재고회전일수 (DIO)** | 동종 수준 | 갑자기 늘어남 |
| 8 | **영업CF vs 순이익** | 영업CF ≥ 순이익 | 영업CF < 순이익 지속 |
| 9 | **FCF** | 양수 지속 | 음수 지속 |
| 10 | **감사의견** | 적정 | 한정·부적정 |

10개 모두 통과 = 안전. 2~3개 빨간불 = 신중. 5개 이상 = 패스.

## 14. 용어 사전 (50+)

(가나다·알파벳 순)

- **AOCI (기타포괄손익누계액)**: 손익에 안 잡고 자본에 직접 반영되는 항목 누적.
- **AP (Accounts Payable, 매입채무)**: 매입 후 미결제 외상값.
- **AR (Accounts Receivable, 매출채권)**: 판매 후 미수령 외상값.
- **BPS (Book Value Per Share, 주당순자산)**: 자기자본 / 발행주식수.
- **CapEx (Capital Expenditure, 자본적 지출)**: 유·무형자산 취득 지출.
- **CCC (Cash Conversion Cycle, 현금화 사이클)**: DSO + DIO - DPO.
- **CFO (Cash Flow from Operations, 영업현금흐름)**: 본업 현금 유입.
- **CFI (Cash Flow from Investing, 투자현금흐름)**: 자산 매입·매각.
- **CFF (Cash Flow from Financing, 재무현금흐름)**: 차입·증자·배당.
- **COGS (Cost of Goods Sold, 매출원가)**: 매출 직접비.
- **CSM (Contractual Service Margin)**: IFRS 17 핵심 — 보험 미래 이익을 부채로.
- **DCF (Discounted Cash Flow)**: 미래 현금흐름 현재가치 합으로 평가.
- **DIO (Days Inventory Outstanding)**: 재고 회전일수.
- **DPO (Days Payable Outstanding)**: 매입채무 회전일수.
- **DPS (Dividend Per Share, 주당배당금)**.
- **DSO (Days Sales Outstanding)**: 매출채권 회전일수.
- **DuPont 분해**: ROE = OPM × 자산회전율 × 레버리지.
- **EBIT (Earnings Before Interest and Taxes)**: 이자·세금 전 이익 (≈ 영업이익).
- **EBITDA**: EBIT + 감가상각 + 무형상각.
- **EPS (Earnings Per Share, 주당순이익)**.
- **FCF (Free Cash Flow, 잉여현금흐름)**: 영업CF - CapEx.
- **FIFO (First-In, First-Out)**: 선입선출.
- **GAAP (Generally Accepted Accounting Principles)**: 미국 회계기준.
- **GP (Gross Profit, 매출총이익)**.
- **IFRS (International Financial Reporting Standards)**: 국제회계기준.
- **K-IFRS**: 한국 채택 IFRS.
- **MD&A (Management Discussion and Analysis)**: 경영진 시각 분석.
- **NCI (Non-controlling Interests, 비지배지분)**: 자회사 외부 주주 지분.
- **NOPAT (Net Operating Profit After Tax)**: 영업이익 × (1 - 세율).
- **NPM (Net Profit Margin, 순이익률)**.
- **OPM (Operating Profit Margin, 영업이익률)**.
- **PP&E (Property, Plant and Equipment, 유형자산)**.
- **R&D (Research & Development, 연구개발비)**.
- **ROA (Return on Assets, 총자산이익률)**.
- **ROE (Return on Equity, 자기자본이익률)**.
- **ROIC (Return on Invested Capital, 투하자본이익률)**.
- **SBC (Stock-Based Compensation, 주식보상비용)**.
- **SG&A (Selling, General and Administrative, 판관비)**.
- **WACC (Weighted Average Cost of Capital, 가중평균자본비용)**.
- **간접법 (Indirect Method)**: 영업CF 작성 방식 — 순이익 출발.
- **감가상각 (Depreciation)**: 유형자산 비용 안분.
- **공사진행률**: 장기공사 매출 인식 기준.
- **금융자산**: 보유 주식·채권 등.
- **기말잔액**: 결산일 잔액.
- **기초잔액**: 결산기 시작일 잔액.
- **당좌비율**: (유동자산-재고)/유동부채.
- **대손충당금**: 회수 불가 외상 적립.
- **레버리지**: 자본 대비 자산 (부채 활용도).
- **무형자산상각 (Amortization)**: 무형자산 비용 안분.
- **밸류에이션**: 가치 평가 (PER·PBR·EV/EBITDA·DCF 등).
- **별도재무제표**: 단일 회사 재무제표.
- **부채비율**: 총부채/자기자본.
- **분개**: 거래의 회계적 기록.
- **사채(社債)**: 회사가 발행한 채권.
- **사업보고서**: 한국 상장사 연차 보고서.
- **세전이익 (EBT)**: 법인세 차감 전.
- **손상차손**: 자산 가치 회복 불가 시 비용 처리.
- **순운전자본**: 매출채권+재고-매입채무.
- **연결재무제표**: 모회사+자회사 합산.
- **영업권 (Goodwill)**: M&A 인수가-순자산.
- **영업이익**: 매출-매출원가-판관비.
- **유동비율**: 유동자산/유동부채.
- **유동성**: 현금화 속도.
- **유보율 (Retention Ratio)**: 1 - 배당성향.
- **이연법인세**: 시점 차이로 미래 세금 인식.
- **이익잉여금**: 누적 순이익 - 누적 배당.
- **자본금**: 액면 × 발행주식수.
- **자본잉여금**: 발행가-액면 차이.
- **자사주 (Treasury Stock)**: 회사가 자기 주식 보유.
- **재고자산평가손실**: 재고 가치 하락 인식.
- **재무상태표 (BS)**: 일정 시점 자산·부채·자본.
- **재무활동 CF**: 차입·상환·증자·배당.
- **주주총회**: 연 1회 주주 의결.
- **지분법**: 관계회사 평가 방법.
- **직접법 (Direct Method)**: 영업CF 작성 방식 — 직접 현금 흐름.
- **충당부채**: 미래 손실 추정 적립.
- **투자활동 CF**: 자산 취득·처분.
- **판관비 (SG&A)**: 본업 운영 간접비.
- **포괄손익계산서**: 손익+기타포괄손익.
- **현금흐름표 (CF)**: 기간 현금 흐름.

- 실전 적발 체크리스트 30개

## 부록 B — 공식 모음

### 손익계산서 관련
- 매출총이익률 = 매출총이익 / 매출 × 100
- 영업이익률 (OPM) = 영업이익 / 매출 × 100
- EBITDA = 영업이익 + 감가상각비 + 무형자산상각비
- 순이익률 (NPM) = 순이익 / 매출 × 100
- EPS = 순이익 / 평균 발행주식수

### 재무상태표 관련
- 자산 = 부채 + 자본
- BPS = 자기자본 / 발행주식수
- 부채비율 = 총부채 / 자기자본 × 100
- 유동비율 = 유동자산 / 유동부채 × 100
- 당좌비율 = (유동자산 - 재고) / 유동부채 × 100

### 현금흐름표 관련
- FCF = 영업CF - CapEx
- FCF Yield = FCF / 시가총액

### 효율성 비율
- DSO = (매출채권 / 매출) × 365
- DIO = (재고 / 매출원가) × 365
- DPO = (매입채무 / 매출원가) × 365
- CCC = DSO + DIO - DPO
- 자산회전율 = 매출 / 평균 총자산
- 재고회전율 = 매출원가 / 평균 재고

### 수익성 비율
- ROA = 순이익 / 평균 총자산 × 100
- ROE = 순이익 / 평균 자기자본 × 100
- ROIC = NOPAT / (자본 + 유이자부채) × 100
- NOPAT = 영업이익 × (1 - 실효세율)

### DuPont
- ROE = (순이익/매출) × (매출/총자산) × (총자산/자본)
     = 순이익률 × 자산회전율 × 레버리지

### 안정성
- 이자보상배율 = 영업이익 / 이자비용

## 부록 C — 실전 사례 미니 모음

### C-1 매출은 늘어도 망한 회사 — 매출채권의 함정

가상 사례: A제조사 5년 매출
- 매출: 1,000 → 1,200 → 1,500 → 2,000 → 2,800 (CAGR 29%)
- 매출채권: 200 → 280 → 450 → 800 → 1,500
- DSO: 73일 → 85일 → 110일 → 146일 → 196일

**상황 해석**: 매출은 빠르게 늘어 보이지만 외상이 더 빨리 늘어남. 4년차쯤 영업CF -300, 5년차 영업CF -800. 5년차 말 부도.

**진실**: 회사가 거래처에 무리하게 외상 매출 밀어붙임. 거래처는 결제 못하고, 회사는 자금경색.

### C-2 순이익은 작아도 알찬 회사 — FCF 챔피언

가상 사례: B서비스사
- 순이익: 100 → 110 → 120 → 130 → 140
- 영업CF: 200 → 220 → 250 → 280 → 320
- CapEx: 50 → 50 → 60 → 60 → 70
- FCF: 150 → 170 → 190 → 220 → 250

**상황 해석**: 순이익보다 영업CF가 2배. 매출채권 회전 빠르고, 감가상각 큰 자본집약 사업.
**투자 가치**: FCF Yield 5%만 잡아도 시가총액 5,000 적정. PER로만 보면 저평가 안 보일 수도.

### C-3 영업이익 폭증의 거짓말 — 일회성 처분이익

가상 사례: C제조사
- 영업이익: 500 → 1,500 (3배 증가!)
- 그러나 그 중 800은 본사 부동산 매각이익(일회성)
- 본업 영업이익 = 500 → 700 (40% 증가)

**투자자 시사점**: 영업이익만 보면 슈퍼주식. 본업 분해하면 그저그런 성장. 1회성 자산처분익은 다음 해 사라짐.

## 마무리 — 이 가이드를 끝까지 읽은 당신에게

여기까지 80,000자. 회계의 핵심은 **3개의 표 + 등식 1개 + 비율 12개 + 사이클 사고**다.

- 3개의 표: 손익계산서·재무상태표·현금흐름표
- 등식 1개: 자산 = 부채 + 자본
- 비율 12개: ROE·ROA·ROIC / OPM·NPM·GP / 자산·재고·매출채권 회전율 / 부채비율·유동비율·이자보상배율
- 사이클 사고: 호황·불황을 평균 내서 진짜 실력을 보는 안목

**다음 보고서를 펼 때**:
1. 항상 5년 데이터로 펼치기 (한 해는 우연일 수 있음).
2. 영업CF와 순이익의 차이부터 확인.
3. DSO·DIO 추세 보기.
4. 부채비율·이자보상배율로 안정성 확인.
5. ROE를 DuPont 분해해서 어디서 오는지 분석.
6. 일회성·반복 비용 분리.
7. 동종 업계 평균과 비교.

이 7단계만 매번 거치면 80% 이상 회사는 1시간 안에 진단된다.

투자 공부의 시작은 회계, 끝도 회계. **숫자를 읽는 사람만이 거짓말을 잡고, 진짜 가치를 본다.**
