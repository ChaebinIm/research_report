### 1.2 회사 이름의 의미 — "PEPT + RON"

**PEPT** = Peptide (펩타이드)
**RON** = -ron (~체, ~소자 의미)

→ **"펩타이드 분자를 다루는 회사"** 라는 뜻.

### 1.3 펩타이드(Peptide)란 무엇인가?

> 🔍 **용어 박스 #1 — 펩타이드**
> 펩타이드 = **단백질의 작은 조각**. 아미노산이 2~50개 사슬로 연결된 형태.
> 단백질 = 50개 이상 / 펩타이드 = 50개 이하.
>
> 비유: **레고 블록 100개 = 단백질, 레고 블록 10개 = 펩타이드**. 더 작아서 다루기 쉽고, 약으로 만들기 좋음.
>
> 대표 펩타이드 약물: **위고비(세마글루타이드)**, **마운자로(티르제파타이드)** = 모두 펩타이드.

### 1.4 한 줄 정체성 3가지

#### 정체성 #1: 펩타이드 약물 "지속형 변환" 전문 회사

펩트론은 **신약을 직접 만들지 않습니다**. 대신 **이미 있는 펩타이드 약(예: 위고비)을 더 오래 가는 형태로 바꿔주는 기술**을 팝니다.

비유: **카페가 새 커피 메뉴를 만들지 않고, 일반 커피를 24시간 떨어지지 않는 우유 거품(라떼아트)으로 만들어주는 기술만 파는 회사**.

#### 정체성 #2: 일라이 릴리의 "원픽" K-바이오

2025-06 일라이 릴리(미국 제약 시총 1위, 비만약 1위)와 **본계약 1.2조원** 체결.
- 전 세계에서 일라이 릴리가 펩타이드 지속형 기술 파트너로 펩트론 선택
- 알테오젠(MSD-키트루다)에 이은 **K-바이오 2번째 빅 딜**

#### 정체성 #3: "넥스트 알테오젠" 후보 1순위

- 알테오젠 = 2020 MSD 계약 → 2025 매출 폭발 → 시총 19조
- **펩트론 = 2025 일라이 릴리 계약 → 2027~2030 매출 폭발 예상 → 시총 ?조**
- 같은 패턴을 따라가고 있어 "다음 알테오젠" 별명

## 2. 핵심 기술 — 스마트데포(SmartDepot)와 미립구 마법

### 2.1 SmartDepot이란?

**SmartDepot** = 펩트론의 핵심 기술. **펩타이드 약물을 PLGA 미립구 안에 가둬서 천천히 방출**하는 기술.

비유:
- 일반 약 = **각설탕** (커피에 빠르게 녹음 = 약효 빨리 나오고 빨리 사라짐)
- SmartDepot 약 = **사탕 (캔디)** (입 안에서 천천히 녹음 = 약효가 오래 지속)

### 2.2 PLGA란?

> 🔍 **용어 박스 #2 — PLGA (Polylactic-co-glycolic Acid)**
> "체내에서 자연 분해되는 인공 고분자". 봉합사·임플란트·약물 전달에 많이 사용.
> 비유: **시간이 지나면 저절로 사라지는 빵 봉지** 같은 재료. 약을 그 안에 넣으면 봉지가 천천히 녹으면서 약이 흘러나옴.

### 2.3 미립구(Microsphere)란?

**미립구** = **머리카락 굵기의 1/10 ~ 1/100 크기의 작은 공 모양 입자**.
- 크기: 약 10~50 마이크로미터 (μm)
- 머리카락 굵기 = 약 100 μm

비유: **밀가루 알갱이 같은 작은 공 안에 약을 넣은 것**. 주사하면 피부 아래에 알갱이가 들어가고, 며칠~몇 주에 걸쳐 천천히 녹으면서 약을 풀어준다.

### 2.4 SmartDepot 제조 과정

```
[펩타이드 약물 + PLGA] + 용매(빙초산)
   ↓ 녹임
[액상 혼합물]
   ↓ 초음파 분무건조
[균일한 미립구 (분말)]
   ↓ 주사용 액에 섞음
[주사제 (월 1회)]
```

**핵심 단계**: **초음파 분무건조** — 초음파로 액상을 떨면서 분말로 만들어 균일한 미립구 생성.

비유: **에어컨 가습기**처럼 액체를 미세한 안개로 분사 → 그 안개가 굳으면서 작은 알갱이로 변함.

### 2.5 SmartDepot의 핵심 장점 4가지

#### 장점 #1: 1주 → 1개월 지속 변환
- 위고비 (1주 1회) → SmartDepot 적용 → **1개월 1회**
- 환자 편의성 4배 향상
- 처방 유지율 ↑

#### 장점 #2: 얇은 주사침 사용 (27~30G)
- 일반 미립구 주사: 23~25G (굵음 → 통증·부담 큼)
- **SmartDepot: 27~30G (얇음 → 통증·부담 적음)**
- 어린이·노인도 부담 적게 자가 주사 가능

> 🔍 **용어 박스 #3 — 게이지(G)**
> 주사침의 굵기 단위. 숫자가 클수록 가늘다.
> - 일반 주사: 21G
> - 인슐린 주사: 30G
> - SmartDepot: 27~30G (인슐린급으로 얇음)

#### 장점 #3: 입자 크기 정밀 제어
- 초음파 노즐로 입자 크기 균일 조절
- 약물 방출 속도 정확 예측 가능

#### 장점 #4: 광범위 적용 가능
- 펩타이드 분자 대부분 적용 가능
- GLP-1 외에도 인성장호르몬·호르몬·암 치료제 등

### 2.6 미립구 vs 카무루스(Camurus) FluidCrystal

펩트론의 최대 경쟁자는 스웨덴 **카무루스(Camurus)** 의 **FluidCrystal** 플랫폼.

| 항목 | 펩트론 SmartDepot | 카무루스 FluidCrystal |
|---|---|---|
| 기술 방식 | **PLGA 미립구** (분말) | 지질 기반 액상 겔 |
| 약물 방출 | 미립구가 천천히 분해 | 체내 수분과 반응해 겔 형성 |
| 유기용매 사용 | 빙초산 (제조 시) | **약 20% (주사액 자체)** |
| 주사침 굵기 | **27~30G (얇음)** | 22~24G (굵음) |
| 환자 편의 | 우수 | 상대적으로 떨어짐 |
| 상용화 | 일부 (셀비온·기존 약물) | 일부 (Buvidal, Brixadi) |
| 일라이 릴리 평가 | **본계약 ✅ (2025-06)** | 별도 계약 (다른 약물) |

#### 카무루스와의 "양다리" 사건
- 2025-03: 일라이 릴리가 **카무루스와도 비만약 계약** 발표
- 펩트론 주가 단기 -20% 충격
- 그러나 **두 회사 계약 약물은 다름** — 펩트론과 카무루스는 **서로 다른 GLP-1**에 기술 적용

→ "양다리" 자체는 일라이 릴리의 비만약 라인업 다각화 전략. 펩트론 본계약은 유효.

### 2.7 미립구 기술의 글로벌 시장 가치

| 항목 | 값 |
|---|---|
| 글로벌 미립구 약물 시장 (2024) | 약 $7B |
| 2030 전망 | 약 $20B |

**핵심 응용:**

## 3. 일라이 릴리 본계약 — 1.2조원 딜의 전모

### 3.1 일라이 릴리(Eli Lilly) 소개

**일라이 릴리** = 미국 인디애나주 본사의 글로벌 제약회사.
- **시가총액**: 약 **$700B (한국 1,000조원)** — **글로벌 제약 시총 1위**
- **2024 매출**: $45B (한국 63조원)
- **간판 약품**: **마운자로 / 젭바운드 (Mounjaro / Zepbound, 티르제파타이드)** — 비만·당뇨약 1위
- **경쟁자**: 노보 노디스크 (Novo Nordisk, 덴마크) — 위고비 보유

### 3.2 마운자로·젭바운드란?

**티르제파타이드 (Tirzepatide)** = 일라이 릴리의 펩타이드 비만·당뇨약.
- **마운자로 (Mounjaro)**: 당뇨병 용도 (2022 FDA 승인)
- **젭바운드 (Zepbound)**: 비만 용도 (2023 FDA 승인)
- 2024 매출 합산 **$11.5B** (위고비 $8B 앞섬)
- 2030 전망 **$50B**

**투약 방식**: 현재 **주 1회 피하주사**.

### 3.3 일라이 릴리가 펩트론을 찾은 이유

#### 일라이 릴리의 고민
- 마운자로/젭바운드 시장이 폭발적 성장
- 그러나 **주 1회 주사 = 환자 편의성 한계**
- 매주 주사 부담 → 처방 중단율 30%+

#### 해결책
- **월 1회 주사로 바꾸면 처방 유지율 ↑ + 매출 추가 확대**
- 그러나 1개월 지속 펩타이드 기술은 **세계에서 5개 회사 미만**만 보유:
  - 펩트론 (한국, SmartDepot)
  - 카무루스 (스웨덴, FluidCrystal)
  - 미국 신생 2~3개사

→ 일라이 릴리가 펩트론과 **2024-10 평가 계약** 체결.

### 3.6 4개 후보 물질의 가치

본계약에 **최대 4개 후보 물질** 포함:
1. 마운자로 SmartDepot 버전 (1순위)
2. 차세대 GLP-1 (2030 출시 예정)
3. GLP-1 + GIP 이중 작용제
4. 후속 비만·당뇨 펩타이드

→ **만약 4개 모두 양산되면 누적 로열티 수조원**.

### 3.7 평가 계약 연장의 의미

#### 시장 우려
- 평가 계약이 1년 → 2년으로 연장되면서 "본계약 지연" 우려
- 펩트론 주가 단기 -20% 충격

#### 회사 측 설명
- "추가 in vivo 실험을 위한 연장 — 본계약 자체는 유효"
- 일라이 릴리가 **신중한 검증** 진행 중

#### 시장 해석
- **연장 = 부정적 (지연 우려)** vs **연장 = 긍정적 (꼼꼼한 검증으로 결과 신뢰도 ↑)**
- 결국 **2026-10 완료 시 추가 본계약 또는 본 양산 단계 진입** 예상

## 4. GLP-1 비만약 시장 — 왜 이게 핫한가

### 4.1 GLP-1이란 무엇인가?

**GLP-1 (Glucagon-Like Peptide-1)** = 우리 몸이 자연 분비하는 **포만감 호르몬**.

> 🔍 **용어 박스 #4 — GLP-1**
> 식사 후 장에서 분비되는 호르몬.
> 작용:
> 1. **인슐린 분비 촉진** → 혈당 ↓
> 2. **위 배출 지연** → 포만감 오래 지속
> 3. **뇌 식욕 중추 자극** → 식욕 ↓
>
> **GLP-1 작용제 약** = 이 호르몬을 모방한 약물. 인공적으로 배고픔을 줄임.

### 4.2 위고비 vs 마운자로

| 항목 | 위고비 (Wegovy) | 마운자로 (Mounjaro / Zepbound) |
|---|---|---|
| 제조사 | 노보 노디스크 🇩🇰 | 일라이 릴리 🇺🇸 |
| 성분 | 세마글루타이드 | 티르제파타이드 |
| 작용 | GLP-1 단독 | **GLP-1 + GIP 이중** |
| 체중 감량 효과 | 평균 -15% | **평균 -22%** |
| 2024 매출 | $8B | $11.5B |
| 투약 | 주 1회 | 주 1회 |
| 가격 (월) | $1,300 (미국) | $1,000 (미국) |

→ **마운자로(릴리)가 효과 + 매출 모두 위고비 추월**.

### 4.3 GLP-1 시장 규모 — 메가 트렌드

| 연도 | 시장 규모 (USD) | 한국 환산 |
|---|---|---|
| 2024 | $50B | 70조원 |
| 2026E | $75B | 105조원 |
| 2028E | $120B | 168조원 |
| **2030E** | **$150B+** | **210조원+** |
| 2035E | $200B+ | 280조원+ |

→ **2030년 한국 반도체 전체 매출 규모의 시장**. 단일 약품 카테고리 사상 최대 성장.

비유: **"21세기 페니실린"** 또는 **"21세기 비아그라"** 라고 부르는 사람도 있을 정도. 일상 약물이 될 가능성.

### 4.4 GLP-1 시장의 핵심 트렌드

#### 트렌드 #1: 적응증 확장
- 비만·당뇨 외에도 **심혈관 질환·신장병·알츠하이머·파킨슨병·중독** 효과 입증 진행
- 시장 5~10배 확장 가능성

#### 트렌드 #2: 경구용 제형
- 현재 주사제 → **경구용 알약 개발 중** (노보 노디스크 리벨서스, 릴리 OG)
- 환자 편의성 ↑

#### 트렌드 #3: 지속형 (월 1회)
- 주 1회 → **월 1회 → 분기 1회** 진화
- **펩트론 SmartDepot이 정확히 이 영역**

#### 트렌드 #4: 다중 작용제
- GLP-1 단독 → GLP-1 + GIP (마운자로) → **GLP-1 + GIP + GCG (Triple agonist)**
- 효과 +30~50% 추가

### 4.5 펩트론 + 일라이 릴리 = 최고 시너지

- 일라이 릴리 = **GLP-1 시장 1위** (마운자로)
- 펩트론 = **지속형 변환 1위** 후보 (SmartDepot)
- 결합 시 → **"마운자로 월 1회 버전" = 차세대 표준약 될 가능성**

비유:
- 일라이 릴리 = **최고 셰프 (만드는 약 자체 최고)**
- 펩트론 = **최고 보존 기술자 (오래 가는 변환)**
- 결합 = **24시간 신선한 최고 요리** = 압도적 시장 우위

## 12. 밸류에이션 — 어떻게 평가하나 + 현재 위치

> **작성 시점:** 2026-05-14 기준. 매매 전 IR/공시 재확인 필수.

### 12.1 이 회사(섹터)는 무슨 잣대로 보나?

**쉽게 말하면**: 펩트론은 **적자 회사**라 P/E로 평가 불가. 대신 **"미래 로열티의 현재가치"** 인 rNPV(위험조정 순현재가치) + **PBR + EV/Sales** 혼합 평가.

비유: **신생 카페가 글로벌 카페 체인과 가맹 계약 체결 → 카페 자체 매출보단 "가맹 로열티 미래가치"가 평가 핵심**.

**조금 더 들어가면**: 펩트론은 알테오젠과 비슷한 단계 — **승인 직후~로열티 시작 전 단계**. 시장은 **본계약 완료 + 임상 성공 확률 + 매출 시나리오**를 모두 가격에 반영.

**전문가 관점**: 펩트론의 PBR 41배는 **글로벌 동종 (카무루스 10배·Alkermes 5배)** 대비 4~8배 프리미엄. 정당화 요인 = **일라이 릴리 본계약의 거대 미래 매출**. 본계약 완료 안 되면 멀티플 정상화(50% 하락).

> **용어 박스 #5 — rNPV / EV/Sales 적용**
> - **rNPV**: 임상 성공 확률 가중 미래 매출의 현재가치. 신약·바이오 회사 표준.
> - **EV/Sales**: 기업가치 ÷ 매출. 적자 성장주에 사용.
> - **펩트론 rNPV** (Bull 가정): 약 8~12조원
> - **펩트론 EV/Sales (2026E 매출 $1,000억 가정)**: 약 60x — **카무루스 13x의 4.6배**
