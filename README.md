## 🛠 작성 방식

각 리포트는 **Claude Code 기반 Agent Team 협업**으로 작성됩니다. 일반적으로 여러 서브에이전트를 병렬 투입해 ① 회사·제품 ② 재무·주가 ③ 시장·경쟁 ④ 뉴스·리스크 영역을 동시에 리서치하고, 결과를 단일 리포트로 통합·교차검증하는 방식입니다.

### 공통 구성
1. Executive Summary
2. 회사·주제 개요
3. 제품·핵심 기술 / 핵심 동인
4. 재무·실적·주가
5. 시장·경쟁 환경
6. 최신 뉴스·파트너십
7. 리스크·투자 전망 (Bull/Bear + 리스크 매트릭스 + Balanced 결론)
8. Sources (카테고리별 분류, 모든 인용 출처를 마크다운 링크로 명시)
